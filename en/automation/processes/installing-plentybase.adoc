plentyBase is a tool that allows your own computer to communicate with plentymarkets.

You need to install and run plentyBase to perform the following tasks in plentymarkets:

* Processes
** Create processes in plentymarkets
** Run processes in plentymarkets
* Printers
** Print documents from your plentymarkets system
** Access the print history from your plentymarkets system
** Print receipts for plentymarkets POS
* Save
** Save documents (originating in plentymarkets) on your computer's hard drive

If plentyBase is installed on the computer, these tasks can be performed in the plentymarkets back end.
