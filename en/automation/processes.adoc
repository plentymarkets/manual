**System » System settings » Processes**

Processes allow you to automate many complex tasks in plentymarkets. You can configure and run processes using the Terra version of plentymarkets 7. You need to install and run the plentyBase tool to manage and run processes. Go to the link:https://marketplace.plentymarkets.com/plugins/externe-tools/plentyBase_5053/[plentymarkets Marketplace^] to download the installation files.
