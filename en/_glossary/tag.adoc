[#tag]
Tags make it possible to emphasise certain data records. After tags have been assigned, data records can be filtered according to tags, allowing for better overview. Tags can be applied to individual <<item/managing-items#268, variations>>, <<item/settings/categories#700, categories>>, <<crm/managing-contacts#150, contacts>>, <<crm/messenger#create-message, messages in the Messenger>>, <<crm/using-the-ticket-system#1600, tickets>> and <<welcome/customise-system#380, tasks>>.
