==  Info: item revenue

This statistic shows the **revenue** of your **items** within a selected time span. The data is displayed for the **warehouse(s)**, **sales order referrers**, **payment methods** and **shipping profiles** that were chosen. Default setting for the time span: **The last 30 days**.

==  Details about the statistic

A variety of information about the **item revenue** will be displayed for the selected **time span** and **settings**. If no sales orders are available then you will see a message telling you that no data is available. In this statistic you can see, e.g. detailed information about your revenue data.

[TIP]
.Tip
====
This statistic is available **starting with plentymarkets version 4.501**.
====

[TIP]

Explanations of the options

[cols="a,a"]
|====
| Entry/Setting | Explanation

|
| Diagram view

|**Type of diagram**
| In this drop-down menu, you can choose how you want the statistics to be displayed. +
**Data table** = An overview of the data in table format

|**Export**
| The data for this diagram is exported in a **CSV file**. The file contains the same parameters as are displayed with the **Data table** diagram.

|**Raw data export**
| The raw data export also generates a **CSV file**. However, the file contains a more detailed set of data. For example, instead of simply indicating the number of sales orders that were generated on a specific day, the CSV file includes relevant data fields for each individual sales order, such as the **sales order ID**, **price** etc.

|
| Settings

|**Name**
| Shows the **name** of the statistic that was entered when the statistic was created. You can change the name here.

|**Time span**
| Each type of statistic has a default setting for **time span** that you can see in table 1 on the <<start/statistics#, **Statistics**  >> page of the manual. You can choose from a number of predefined time spans in a drop-down menu. In addition for some types of statistics you have the possibility to enter a time span by entering a **start and end date**.

|
|**Change** the **time span** if no or only very few data is shown and then click on the save icon to see the result. Choosing a **larger time span** will give you more information. If there is a lot of data, then a **shorter time span** may make more sense in order to be able to actually interpret the graphic.

|**Starting date ... Ending date**
| You can enter a specific **starting and ending date** here instead of selecting a **time span** (optional).

|**Date based on**
| Select which event should be used for analyzing the data: **Sales order received**, **Incoming payment** or **Outgoing items**.

|**Status options**
| Depending on the type of statistic there are different options available to narrow down the **status**. There are **standard values** set by default.

|**Multishop**
| Select the **webshops** or **multishops** that you would like to include into the statistic.

|**Warehouse**
| Select the **warehouse** that you would like to include into the statistic.You can use the option **All** to select all of the available ones at once.

|**Sales order referrer**
| Select one or more **sales order referrers**. Use the option **All** to select all of the available sales order referrers at once.

|**Payment method**
|**Select** which **payment method** should be included in the statistic or use the option **All** to include all of the payment methods.

|**Shipping profile**
|**Mark** which **shipping profiles** should be included in the statistic or use the option **All** to include all of the shipping profiles.

|**Net**
| Activate this option to display **net amounts** in the statistic.

|**Paid sales orders only**
| Select this option if you only want to include **paid sales orders** in the statistic.

|**Sales orders with outgoing items booked only**
| Select this option if you only want to include **sales orders with outgoing items booked** in the statistic.
|====
