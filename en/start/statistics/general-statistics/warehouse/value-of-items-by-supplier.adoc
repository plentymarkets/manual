==  Info: value of items

This statistic shows the **value of items by supplier** within a selected time span. The data is displayed for the **warehouse(s)** and **supplier(s)** that were chosen.

[TIP]
.Tip
====
This statistic is available **starting with plentymarkets version 4.501**.
====

==  Details about the statistic

The **number of items** and the **net value of items in Euro** will be displayed for each **supplier**. The data is displayed for the **warehouse(s)** that were chosen. The exact **net value of items** will be displayed as **information in a mouseover** for the corresponding supplier. Furthermore, a second color will display **how many items** are included in this total value (see image 1). In this statistic you can see, e.g. how many items are in each of your warehouses and what the value of these items is.

**Show details (mouseover)**

Point your **cursor** at one of the elements in the statistic in order to see **details** about that particular element.

[TIP]
.Tip
====
In image 1, you can see that **approximately 270,000 items** are in the **inventory** and that these items have a total **value** of **677,655.93 Euros**.The exact **values** are displayed as information in a **mouseover** for the blue and red columns.
====

[IMPORTANT]
.
====
The **quantity** and **value of items** will only be analyzed for **positive physical stock**.
====
