==  Setting up Mercateo in plentymarkets

Before you can set up Mercateo in plentymarkets, you need data for the FTP server that is to be used for the upload to Mercateo, as well as an email account for the order import.

[.instruction]
Setting up Mercateo in plentymarkets:

. Go to *System » Markets » Mercateo » Settings*.
. Carry out the settings. Pay attention to the explanations given in table 1.
. *Save* the settings.

.Basic settings for the market Mercateo
[cols="1,3"]
|====
| Setting | Explanation

|*Activate the automatic data exchange*
| Activate in order to automatically exchange data with Mercateo.

2+| General

|*Item ID*
| Select whether the *Variation ID* or the *Variation No.* should be used for the item data.

|*Transfer item number*
|Only select *Yes* if you are the manufacturer of the item. The selection that you make in the *Item ID* drop-down list does not affect the option here. For example, if you are the manufacturer of the item and you have already selected variation number above, then you will still have to select the option *Yes* here.

|*Item name*
|Select *Name 1*, *Name 2*, *Name 3* or *Variation name*. This name is displayed on Mercateo.

|*Add attribute value to item name*
|Select *Yes* if the attribute value should be displayed behind the item name on Mercateo.

|*Item description*
| Select which item description should be transferred to Mercateo.

2+| Input

|*FTP server*
| Enter the URL of the FTP server for the data upload to Mercateo.

|*FTP user*
| Enter the user name of the FTP server.

|*FTP password*
| Enter the password of the FTP server.

2+| Retrieval

|*User name*
| Enter the user name for the order retrieval.

|*Password*
| Enter the password for the order retrieval.

|*IMAP server*
| Enter the server address for the order retrieval.

|*Port*
|Enter port.

|*Encryption*
|Enter encryption.

2+|Other catalogues

|*Suffix synchronisation No. 2 - 4*
|Enter the *suffixes* for the additional catalogue of items.

|*Order referrer*
|Select the *order referrer* for the import of orders.

|*Automatic retrieval*
| Activate in order to automatically retrieve orders.

2+| Updates

|*Automatic price update*
| Activate in order to update the item prices hourly.

|*Automatic delivery status update*
| Activate in order to automatically update the delivery status.
|====

Connecting to this market allows for the following automatic functions to take place:

* Data exchange
* Order import
* Hourly price synchronisation
* Delivery status synchronisation

== Activating an order referrer

In order to link items, properties etc. with Mercateo, you have to activate the order referrer Mercateo in the menu *System » Orders » Order referrer*.

[.instruction]
Activating the order referrer for Mercateo:

. Go to *System » Orders » Order referrer*.
. Place a check mark next to *Mercateo*.
. *Save* the settings.

== Setting the item availability

Items have to be available for Mercateo. This is done in the *Availability* tab of an item within the <<en/item/managing-items#170, Item » Edit item » Open item » Tab: Variation ID>> menu.

[.instruction]
Setting the item availability for Mercateo:

. Go to *Item » Edit item » Open item » Tab: Variation ID » Tab: Settings*.
. Activate the main variation in the *Availability* section.
. Click on the *Availability* tab.
. Click in the selection field in the *Markets* section.  +
→ A list with all available markets is displayed.
. Place a check mark next to the option *Mercateo*.
. Place a check mark next to the option *Web API*.
. Click on *Add*. +
→ The market is added.
. *Save* the settings. +
→ The item is available on Mercateo.

The availability for variations can be individually edited in the *Item » Edit item » Open item » Tab: Variations » Open variation » Tab: Variation ID » Tab: Availability* menu.

==  Exporting items and item images

In the *System » Markets » Mercateo » Data export* menu, you can export your items and item images to Mercateo.

[.instruction]
Exporting items:

.  Go to *System » Markets » Mercateo » Data export*.
.  Click on *Export item data*. +
→ The item data is exported to Mercateo.

You can also export item images to Mercateo. A maximum of 500 images can be uploaded at once in JPEG format. No image is uploaded twice.

[.instruction]
Exporting item images:

.  Go to *System » Markets » Mercateo » Data export*.
.  Click on *Export 500 item images*. +
→ The item images are exported to Mercateo.

[#500]
== Defining a sales price

Proceed as described below to define a sales price for the order referrer Mercateo. This price is displayed on Mercateo.

[.instruction]
Defining a sales price for Mercateo:

. Open the menu *System » Item » Sales prices » Open sales price » Tab: Settings*.
. Place a check mark next to the referrer *Mercateo*.
. *Save* the settings.

[#600]
==  Viewing the export history

An overview of exported images is displayed in the *System » Markets » Mercateo » Export history* menu. The images are displayed on different pages. 25 images can be displayed per page.

[.instruction]
Selecting a page:

.  Go to *System » Markets » Mercateo » Export history*.
.  Select a *page* from the drop-down list.
. Click on the *gear-wheel icon* to display the page.

The images are displayed along with the corresponding item ID and time of export. It is not possible to export an image more than once. However, if you want to update an image, e.g. after changes have been made to it, then click on *Delete* in the *Release item image* column. Then export this image again in the *Data export » Item images export* menu.
