== Registering with grosshandel.eu

You have to link:http://www.grosshandel.eu/registrierung-verkaeufer.php[register^]{nbsp}icon:external-link[] in the grosshandel.eu seller portal before you can set up grosshandel.eu in your plentymarkets system.

[#200]
== Creating a user account

Your plentymarkets system is connected to grosshandel.eu via the SOAP API. In order to gain access, the market requires its own user account with access rights to the API in your plentymarkets system. Go to *Setup » Settings » User » Accounts* and create a new user account for grosshandel.eu with the user class *API*. Furthermore, activate the call *GetCategoryMappingForMarket*. This call ensures that the market will later be able to retrieve your category links.

[.instruction]
Creating a user account for grosshandel.eu:

. Go to *Setup » Settings » User » Accounts*.
. Click on *New*. +
→ The *Create new user* window opens.
. Enter the data.
. Select the user class *API*.
. *Save* (icon:save[role="green"]) the settings. +
→ The user is created.
. Select the user from the list. +
→ An editing window opens.
. Click on the tab *Right » API*.
. Expand the *Item* area.
. Activate the *GetCategoryMappingForMarket* call.
. *Save* (icon:save[role="green"]) the settings.

For further information refer to the <<business-decisions/user-accounts-access#, Managing user accounts>> page of the manual.

[#300]
== Procuring data

grosshandel.eu needs to have the following data from you in order to connect to your plentymarkets system:

* The URL for the SOAP API WSDL. +
→ This URL can be found in the *Setup » Settings » API » Data* menu under *SOAP WSDL*.
* The user name and password for the user account that you created for grosshandel.eu
* Your login details for the grosshandel.eu seller portal.

[#350]
== Activating an order referrer

In order to link items, properties etc. with grosshandel.eu, you have to activate the order referrer grosshandel.eu in the menu *Setup » Orders » Order referrer*.

[.instruction]
Activating the order referrer for grosshandel.eu:

. Go to *Setup » Orders » Order referrer*.
. Activate the referrer *grosshandel.eu*.
. *Save* (icon:save[role="green"]) the settings.

[#400]
== Setting the variation availability

You need to activate all variations that you want to sell on grosshandel.eu. This is done in the *Availability* tab of a variation within the *Item » Edit item » [Open item] » [Open variation]* menu.

[TIP]
.Selling all variations of an item on grosshandel.eu
====
If you want to sell all variations of an item on grosshandel.eu, carry out the settings below for the item's main variation. All other variations of the item then inherit the settings of the main variation.
====

[.instruction]
Activating the variation availability for grosshandel.eu:

. Go to *Item » Edit item » [Open item] » [Open variation] » Tab: Settings*.
. Open the variation that you want to sell on grosshandel.eu.
. In the *Availability* area, activate the option *Active*.
. Click on the *Availability* tab.
. In the *Markets* area, click in the selection field. +
→ A list of all available markets is displayed.
. Activate the option *grosshandel.eu*.
. Activate the option *Web API*.
. Click on *Add* (icon:plus-square[role="green"]). +
→ The market is added.
. *Save* (icon:save[role="green"]) the settings.

== Linking categories

Link your store categories to the grosshandel.eu categories in order to display your items in these categories. Further items from the linked categories are assigned automatically.

[.instruction]
Linking categories:

. Go to *Setup » Markets » grosshandel.eu*.
. Expand the category which you want to link to a grosshandel.eu category.
. Select a main category for grosshandel.eu from the drop-down list. +
→ A window with subcategories opens.
. Select the appropriate grosshandel.eu category from the subcategories that are shown. +
→ Once the lowest category level has been reached, the category path will be displayed in the field *Category*.
. Click on *Accept*. +
→ The category window is closed.
. *Save* (icon:save[role="green"]) the settings.

[IMPORTANT]
.Link to the lowest category level
====
Always select a path to the lowest level. Otherwise, the item will not be linked.
====

[#700]
== Linking multiple subcategories

Use the option *Apply the same selection to the sub-categories* to apply the same grosshandel.eu category path for all of the subcategories that are listed within a category. This is useful if you want to use the same link for the subcategories. If needed, you can use the drop-down list to manually change individual subcategories.

[.instruction]
Linking multiple subcategories:

. Go to *Setup » Markets » grosshandel.eu*.
. Expand the category which you want to link.
. Select a main category for grosshandel.eu from the drop-down list in the line *Apply the same selection to the sub-categories*. +
→ A window with subcategories opens.
. Select the appropriate category path from the subcategories that are shown. +
→ Once the lowest category level has been reached, the category path is displayed in the field *Category*.
. Click on *Accept*. +
→ The category window is closed.
. Click on *Select all* in the line *Apply the same selection to the subcategories*.
. Click on *Reload*. +
→ The category path is applied to all of the subcategories.
. *Save* (icon:save[role="green"]) the settings.
