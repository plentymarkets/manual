// tag::availability[]
You need to activate all variations that you want to sell on {market-availability}. This is done in the *Availability* tab of a variation within the *Item » Edit item »* _Open item_ *» Tab: Variation ID* menu.

[TIP]
.Use group function or batch processing
====
You can edit the availability of several variations at once by using either the <<item/managing-items#330, variation group function>> or <<item/managing-items#330, batch processing>>.
====

[.instruction]
Setting the variation availability for {market-availability}:

. Go to *Item » Edit item »* _Open item_ *» Tab: Variation ID » Tab: Settings*.
. Open the variation that you want to sell on {market-availability}. +
*_Tip:_* To sell all variations of an item on {market-availability}, carry out the settings for the item’s main variation. This activates the availability of all variations of the item.
. In the *Availability* area, activate the option *Active*.
. Click on the *Availability* tab.
. In the *Markets* area, click in the selection field. +
→ A list of all available markets is displayed.
. Activate the option *{option-availability}*.
// tag::web-api[]
. Activate the option *Web API*.
// end::web-api[]
. Click on *Add* (icon:plus-square[role="green"]). +
→ The market is added.
. *Save* (icon:save[role="green"]) the settings. +
→ The variation is available for {market-availability}.
// tag::neckermann-provision[]
. Click on the *Multi-Channel* tab.
. Select a *commission model*.
. Select an *Item type*.
. *Save* (icon:save[role="green"]) the settings.
// end::neckermann-provision[]
// tag::idealo-direkt[]
. If you want to use idealo Checkout, also activate the option *idealo Checkout* in the *Markets* area.
. *Save* (icon:save[role="green"]) the settings.
// end::idealo-direkt[]
// end::verfuegbarkeit[]
