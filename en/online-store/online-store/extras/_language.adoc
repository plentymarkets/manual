== General information about the language menu

In this menu, you can configure the language variety that you use in your online store. For example, you can select varieties of German for Austria and Switzerland. This setting is important if you want to use the translation services offered by <<omni-channel/online-store/setting-up-clients/extras/lionbridge#, Lionbridge>>. The language varieties saved in this menu are the target languages for Lionbridge.

== Configuration

Save a language variety for each language that you use in your store.

.selecting a language
image::omni-channel/online-store/setting-up-clients/extras/assets/EN-Settings-Basic-settings-Language-01.png[]

[.instruction]
Selecting a language:

. Go to *System » System settings » Settings » Language*.
. Save a *language code* for each language that you use in your store.
. *Save* the settings.
