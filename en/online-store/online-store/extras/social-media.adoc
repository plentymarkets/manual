== Social Media

In this menu, you can save *URLs* for *social media* platforms like Facebook and twitter. Furthermore, you can activate or deactivate the link to social media.

== Configuration

Pay attention to the explanations given in the following table and carry out the configuration as desired.

.settings in the *Social Media* menu
image::omni-channel/online-store/setting-up-clients/extras/assets/EN-Settings-Client-Standard-Services-Social-Media-01-SI.png[]

The following table explains the settings in the *System » Client » Select client » Services » Social Media* menu:

.settings for activating and linking *social media*
[cols="1,3"]
|====
|Setting |Explanation

|*Social Media*
|Select whether you would like the link to all social media platforms to be *active* or *inactive*.

|*Facebook URL*
|Enter the *URL* that you want to use for the link to *Facebook*.

|*Facebook login*
|Select *active* if you want to use the <<omni-channel/online-store/setting-up-clients/extras/facebook#, Facebook login function>>. The default setting is *inactive*, i.e. the login function will not be used.

|*Twitter URL*
|Enter the *URL* that you want to use for the link to *twitter*.

|*Google+ URL*
|Enter the *URL* that you want to use for the link to *Google+*.
|====
