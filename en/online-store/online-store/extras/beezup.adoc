== BeezUP

*BeezUP* helps you to manage and optimize the presentation of your items in price comparison portals, on markets and with affiliate services.

== Registering with BeezUP

Click on the following link in order to register: link:https://go.beezup.com/Register?cultureName=en-GB[www.beezup.co.uk^]. Afterwards, you will receive a *store ID* which you need for setting up the tool in plentymarkets. For further information about BeezUP, refer to the link:http://www.beezup.co.uk/home/faq[FAQ^].

[WARNING]
.Store ID
====
The *store ID* is only relevant for tariffs that cost a monthly fee. If you use a free tariff or if you only set up the item export, then you do not have to carry out any settings in the *tracker* tab.
====

== Setting up BeezUP in plentymarkets

Read the explanations below on how to set up BeezUP in your plentymarkets system.

=== Tab: Tracker

This tab is *only relevant for tariffs that cost a monthly fee*.

[.instruction]
Saving tracker settings for BeezUP:

. Go to *Setup » Client » Select client » Services » BeezUP*.
. Pay attention to the explanations given in the following table before configuring the settings in the *tracker* tab.
. *Save* the settings.

.*BeezUP*; settings in the *tracker* tab

The following table explains the settings in the *Setup » Client » Select client » Services » BeezUP* menu:

.*BeezUP*; settings in the *tracker* tab
[cols="1,3"]
|====
|Option |Explanation

|*Store ID*
|Enter the *store ID* that you received from BeezUP.

|*Tracker activated*
|Activate the tool by placing a *check mark*.

|*Item export format*
|Select the *export format* that you created in the *Price comparison portals* menu (see section 3.2).
|====


=== Creating an export format

Carry out the following settings to create an *export format* for *BeezUP*:

[.instruction]
Creating an export format for BeezUP:

. Go to *Setup » Markets » Price analysis » Exports » Tab: New*.
. Select the format *BeezUP* as the *data format*.
. Pay attention to the explanations given on the <<data/export-import/exporting-data#, Exporting data>> page and carry out the settings as desired.
. Click on the *save icon* to save the new export format. +
→ The export format will be saved as a *subfolder* within the *Exports* tab.
. Copy the *URL* from the corresponding *URL* field and save it in your *BeezUP* account.

==== Conditions for the item export

The following *item* parameters have to be configured for BeezUP:

* *Item name*
* *Item description (description or short description)*
* *Price*
* *Image*
* *Delivery costs*
* *Delivery period (availability = average delivery time in days)*
* *Category 1*
