==  affilinet

**link:http://www.affili.net/en/Homepage.aspx[affilinet]** is Germany's leading affiliate platform. It has an attractive offer as well as a comprehensive virtual sales network.

==  Registering with affilinet

The first step towards registering is filling out an **link:https://www.affili.net/en/Advertiser/Apply_now.aspx[application]** with affilinet. You can only register with affilinet once your application has been accepted. Before you can actively use this platform, affilinet has to accept you as an **advertiser**. You will then receive an **advertiser ID**, which you will need for the set-up in plentymarkets.

[WARNING]
.
====
**URL parameter**
====

During the registration process, the **tracking code** has to be included in the URL. This code is displayed as **ref=XYZ**.

**Example**: **www.YOUR-STORE.com/?****ref****=XYZ (XYZ = affilinet parameters)**

Furthermore, you have to activate the **Parameter-Forwarding** function in your affilinet account settings. This automatically attaches the **ref** parameter to your store's URL.

==  Setting up affilinet in plentymarkets

With the following configuration, we provide you with the compensation method **Pay-Per-Sale**, i.e. you only pay a **commission** when a sale occurs. The commission is calculated as a **percentage**. The **rate ID** is the parameter for calculating the commission in plentymarkets. You can find this in your affilinet commission model.

Once you have registered with affilinet and received both the **advertiser ID** and **rate ID**, then you can set up the interface in plentymarkets.

[IMPORTANT]
.
====
**Commission**
====

Has to be defined directly in your affiliate provider account. This interface is configured for the **Pay-Per-Sale** affiliate program by affilinet.
