== Technical specifications

include::./_textblocks/spezifikationen-en_gb.adoc[]

[[table-specifications-feedbacks]]
.Technical specifications of the *feedbacks* format
[cols="4"]
|====
|Property |Meaning |Value |Explanation

|dataName
|Name
|feedbacks
|Name of the creator process and prefix of the created file. +
File name: *feedbacks-xxx.csv.gz*

|fileFormat
|File format
|CSV, GZIP
|The data is provided as a GZIP-compressed CSV file (RFC 1952).

|defaultActive
|Activated by default
|no
|The data is only exported automatically in the background after you have activated the creator in the *Setup » Settings » Report » Raw data* menu.

|limitPage
|Lines per request
|up to 5,000
|The maximum number of data rows that is written into a single file.

|limitDay
|Lines per day
|up to 50,000
|The maximum number of data rows that is written on one day in total.

|earliestPossibleDataDate
|Earliest point in time
|2006-01-01
|Point in time as of which the raw data can be collected. The data is filtered by the date saved in the *dataUpdatedAtColumnName* column.

|dataUpdatedAtColumnName
|Column header
|f_updated_at
|Name of the column that contains the time stamp of the last data update.

|deleteRawDataAfterDaysMinimum
|Minimum number of days until the data is deleted
|20
|The minimum number of days until the generated raw data file is deleted from the cloud storage. Therefore, the data must be processed before this date. The exact duration can be defined in the *Setup » Settings » Report » Raw data* menu.

|deleteRawDataAfterDaysMaximum
|Maximum number of days until the data is deleted
|80
|The maximum number of days until the generated raw data file is deleted from the cloud storage. Therefore, the data must be processed before this date. The exact duration can be defined in the *Setup » Settings » Report » Raw data* menu.

|filterType
|Filter type
|lastUpdate
|Data are filtered based on the last update.

|====
