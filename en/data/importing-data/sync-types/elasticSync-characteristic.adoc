= Characteristic
:lang: en
include::{includedir}/_header.adoc[]
:keywords: ElasticSync, import, characteristics, importing characteristics
:position: 40

A sync of the type *Characteristics* allows you to import characteristics. Characteristics are old properties that are linked at the item level. Imported data is saved in the *Settings » Item » Properties* menu.
