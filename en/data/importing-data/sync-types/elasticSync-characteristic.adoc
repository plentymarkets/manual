= Characteristics
:lang: en
include::{includedir}/_header.adoc[]
:keywords: ElasticSync, import, characteristics, importing characteristics
:position: 110

A sync of the type *Characteristic* allows you to import characteristics. Characteristics are old properties that are linked at the item level. Imported data is saved in the *Settings » Item » Properties* menu.

<<table-matching-fields-characteristics>> explains the fields that are available for matching characteristics data. During the import, plentymarkets checks if the imported data record already exists based on the matching field.
