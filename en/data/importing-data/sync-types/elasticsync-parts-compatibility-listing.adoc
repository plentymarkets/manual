== Fields for mapping parts compatibility listings

This chapter explains the fields that are available for mapping parts compatibility listings.

[#300]
=== Parts compatibility listing

<<mapping-parts-compatibility-listing>> lists the fields that are available for importing parts compatibility listings. During the import, plentymarkets checks if the imported data record already exists based on the matching field.

[[mapping-parts-compatibility-listing]]
.Mapping fields for importing parts compatibility listings
[cols="1,1,2"]
|====
|Mapping field |Permitted import values |Explanation

|Parts compatibility listing ID
|Numerical
|The ID of the parts compatibility listing.

|Referrer ID
|10003 / 2.03 = United Kingdom +
1008 / 2.08 = Germany +
1009 / 2.09 = Motors
|The referrer ID.

|Name
|Alphanumerical
|The name of the parts compatibility listing.

|Category ID
|61455 = Cars Germany + United Kingdom, +
179753 = Motorcycles Germany + United Kingdom, +
65368 = Scooter United Kingdom, +
25640 = Scooter Germany + Motors, +
33565 = Cars Motors, +
10064 = Motorcycles Motors
|The category's ID.

|Delete
|
|Delete parts compatibility listing.
|====
