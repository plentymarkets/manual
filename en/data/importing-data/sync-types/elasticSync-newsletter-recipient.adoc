With syncs of the type *Newsletter recipient*, you import new newsletter data or update existing data records. This data contains for example the email address as well as the birthday and name of the newsletter recipient.

[#100]
== Fields for matching newsletter recipients

The following field is available for matching newsletter recipients:

[[table-matching-field-newsletter-recipient]]
.Fields for matching newsletter recipients
[cols="1,3"]
|====
|Matching field |Explanation

|ID
|ID of the email address to which the newsletter is sent

|====

During the import, plentymarkets checks if the imported data record already exists based on the matching field.

[#200]
== Fields for mapping newsletter recipients

<<table-mapping-fields-newsletter-recipients>> lists the fields available for importing serial number data. During the import, plentymarkets checks if the imported data record already exists based on the matching field.

[[table-mapping-fields-newsletter-recipients]]
.Fields for importing newsletter recipients
[cols="1,1,2"]
|====
|Mapping field |Permissible import value |Explanation

|ID
|Numeric (internal ID)
|ID of the email address to which the newsletter is sent

|ID of the folder
|Numeric (internal ID)
|ID of the folder in which the email address of the newsletter recipient is saved

|Contact ID
|Numeric (internal ID)
|ID of the contact

|First name
|Text
|First name of the newsletter recipient

|Last name
|Text
|Last name of the newsletter recipient

|Email
|Alpha-numeric
|Email address of the newsletter recipient

|Gender
|
|Gender of the newsletter recipient

|Birthday
|Date
|Date of birth of the newsletter recipient

|Template language
|
|Language that is saved for the email template. +
*_Note:_* The language is set in the *Email message* tab of the email template.

|Confirmed at
|Date
|Date when the newsletter recipient confirmed the newsletter receipt.

|Confirmation link
|URL
|Confirmation link that the contact has to click in the email template to register for the newsletter receipt. +
*_Note:_* The confirmation link can be inserted in the email template via the template variable *$NewsletterConfirmURL*.

|IP address
|Numeric
|IP address that the contact used to confirm the newsletter registration.

|====
