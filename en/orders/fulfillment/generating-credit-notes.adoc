== Generating credit notes

In this menu, credit notes are generated with the help of a **group function**. For example, credit notes could be generated for all orders that currently have the status **credit note** or for all orders where a credit note was already **disbursed**. These credit notes could be generated with a search request that has **order status [11] Credit note**.

== Options

You can carry out settings related to the layout of the credit notes by going to **Settings &gt; Orders &gt; Documents**.

Go to **Settings &gt; Orders &gt; Event procedures** in order to define which additional **procedure(s)** should be carried out when the credit note is generated.

== Generating credit notes

[NOTE]
.Generating credit notes:
====
Go to **Generate credit notes** (see path above).

Pay attention to the explanations given in table 1 and carry out the **settings** for the **search filter** as desired.

Under **procedure**, select the order status that you would like to have assigned to the selected orders after the credit notes were successfully created. Additionally, enter a **comment** if desired.

Click on the **gear-wheel icon** after having carried out all of the settings. +
→ You will receive a **PDF file**, which contains the credit notes of all the orders that match the filter options you selected.
====

|====
|Setting |Explanation

|**Order status**
|Select the current **status** of the orders that you would like to generate credit notes for.

|Owner
|Select an **owner** for the orders that you are searching for. Alternatively, select the option **ALL**.

|Client (store)
|Select one of the **stores** or select the option **ALL**.

|**Warehouse**
|Limit the search to a particular **warehouse** or carry out the search for **ALL** warehouses.

|**Payment method**
|Select one of the **payment methods** or select the option **ALL**.

|**Sorting**
|Determine which option you would like to use for sorting the credit notes that are generated. Options: **Credit note number**, **Order ID**, **Customer ID**, **Item ID**. If you choose the **Item ID** option, then the credit notes will automatically be sorted according to their **Order ID** as well.

|**Limit**
|You have to **limit** the number of credit notes that can be generated to one of the values in the drop-down menu. The maximum is **200** credit notes.

|**Change order status**
|Select a new **order status** that you would like to have assigned to the selected orders after the credit notes were successfully generated. Alternatively, select the option **no change**.

|**Comment**
|You can compose a **text** here, which you would like to have appear on all of the credit notes **underneath the stock units**. This entry is optional.

|**Change credit note layout**
|Click on this link to open the **Settings &gt; Orders &gt; Documents** area in a new window. There you can carry out detailed settings for the credit note template.

|**Event procedure**
|Click on this link to open the **Settings &gt; Orders &gt; Event procedures** area in a new window. There you can create new event procedures and edit existing ones.
|====

__Table 1: **settings** in the **Generate credit notes** menu__
