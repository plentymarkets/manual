== Generating an entry certificate (Gelangensbestätigung)

In this menu, entry certificates are generated with the help of a **group function**. For example, entry certificates could be generated for all orders that are currently ready for shipping. The entry certificates are generated with a search request that has **order status [5] cleared for shipping**.

== Options

You can carry out settings related to the layout of the entry certificates by going to **Settings &gt; Orders &gt; Documents**.

Go to **Settings &gt; Orders &gt; Event procedures** in order to define which additional **procedure(s)** should be carried out when the entry certificate is generated.

== Generating an entry certificate (Gelangensbestätigung)

[NOTE]
.Generating an entry certificate (Gelangensbestätigung):
====
Go to **Generate entry certificate (Gelangensbestätigung)** (see path above).

Pay attention to the explanations given in table 1 and carry out the **settings** for the **search filter** as desired.

Under **procedure**, select the **order status** that you would like to have assigned to the selected orders after the entry certificates were successfully created. Additionally, enter a **comment** if desired.

Click on the **gear-wheel icon** after having carried out all of the settings. +
→ You will receive a **PDF file**, which contains the entry certificates of all the orders that match the filter options you selected.
====

[cols="a,a"]
|====
|Setting |Explanation

|**Order status**
|Select the current **status** of the orders that you would like to generate entry certificates for.

|Owner
|Select an **owner** for the orders that you are searching for. Alternatively, select the option **ALL**.

|Client (store)
|Select one of the **stores** or select the option **ALL**.

|**Warehouse**
|Limit the search to a particular **warehouse** or carry out the search for **ALL** warehouses.

|**Payment method**
|Select one of the **payment methods** or select the option **ALL**.

|**Sorting**
|Determine which option you would like to use for sorting the entry certificates that are generated. Options: **Order ID**, **Customer ID**, **Item ID**. If you choose the **Item ID** option, then the entry certificates will automatically be sorted according to their **Order ID** as well.

|**Limit**
|You have to **limit** the number of entry certificates that can be generated to one of the values in the drop-down menu. The maximum is **200** entry certificates.

|**Change order status**
|Select a new **order status** that you would like to have assigned to the selected orders after the entry certificates were successfully generated. Alternatively, select the option **no change**.

|**Comment**
|You can compose a **text** here, which you would like to have appear on all of the entry certificates **underneath the stock units**. This entry is optional.

|**Change the layout of the entry certificate (Gelangensbestätigung)**
|Click on this link to open the **Settings &gt; Orders &gt; Documents** area in a new window. There you can carry out detailed settings for the entry certificate template.

|**Event procedure**
|Click on this link to open the **Settings &gt; Orders &gt; Event procedures** area in a new window. There you can create new event procedures and edit existing ones.
|====

__Table 1: **settings** in the **Generate entry certificate (Gelangensbestätigung)** menu__
