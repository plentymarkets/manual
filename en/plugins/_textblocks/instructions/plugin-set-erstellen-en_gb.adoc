[.instruction]
Creating new plugin sets:

. Go to *Plugins » Plugin overview*.
. Click on *Add* (icon:plus-square[role=green]). +
→ The *New plugin set* window opens.
. Enter a name for the plugin set.
. *Save* (icon:save[role=green]) the settings.
