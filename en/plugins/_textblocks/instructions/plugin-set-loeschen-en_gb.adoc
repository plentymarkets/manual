[.instruction]
Deleting plugin sets:

. Go to *Plugins » Plugin overview*.
. On a plugin set tile, click *Delete* (icon:minus-square[role=red]). +
→ The *Delete plugin set* window opens.
. Confirm the deletion. +
→ The plugin set is deleted.
