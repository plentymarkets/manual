Use the menu **Data exchange » Export » Elastic export** to create export formats, e.g. for price search engines, price comparison portals, markets and affiliate service providers. This page of the manual lists the export formats that are available in plentymarkets and teaches you how to create, edit and filter them.

== Registering with the price search engine

Before you can transfer your export format to a price search engine, you will have to register on the price search engine's website and follow any instructions given there. Refer to table 1 for links to the price search engines that have an interface in plentymarkets.

== Creating export formats

This chapter teaches you how to create new export formats and which settings you can carry out.

[.instruction]
Creating a new export format:

. Go to **Data exchange » Export » Elastic export**.
. Click on **New export**.
. Carry out the settings as desired. Pay attention to the information given in table 2.
. **Save** the settings. +
→ The export format will be given an ID and it will appear in the overview within the **Exports** tab.

[cols="a,a"]
|====
|Setting |Explanation

2+|Settings

|**Name**
|Enter a name. The export format will be listed under this name in the overview within the **Exports** tab.

|**Type**
|Select a type from the drop-down menu. Currently, **Item** is the only available option. Further options will be available soon.

|**Format**
|Select a format for the price portal from the drop-down menu.

|**Limit**
|Enter a number. If you want to transfer more than 10,000 data records to the price search engine, then the output file will not be generated again for another 24 hours. This is to save resources.

|**Provisioning**
|Select **URL**, **FTP** or **Admin area**. +
Different fields will appear depending on which option is selected here.

|**Token, +
URL**
|If you selected the option **URL** under **Provisioning**, then click on **Generate token**. The token will be entered automatically. The URL will be entered automatically if the token was generated under **Token**.

|**Server, +
Port, +
User name, +
Password, +
Path**
|If you selected the option **FTP** under **Provisioning**, then enter the parameters. The standard FTP port is **21**.

|**Transfer**
|Decide how often the data should be transferred.

2+|Item filters

|**Add item filters**
|Select an item filter from the drop-down menu and click on **Add**. It is possible to add multiple item filters from the drop-down menu one after the other. +
**Variations** = Select **Transfer all** or **Only transfer main variations**. +
**Markets** = Select one, several or **ALL** of the markets. +
**Currency** = Select the currency. +
**Category** = Activate to transfer the item with its category link. +
**Image** = Activate to transfer the item with its image. +
**Referrer** = Select the referrer. The referrer is used for calculating the sales price. +
**Stock** = Select which stock level should be exported. +
**Flag 1 - 2** = Select the flag. +
**Manufacturer** = Select one, several or **ALL** of the manufacturers.

2+|Format settings

|**Product URL**
|Activate if you want the product URL to be transferred to the price portal.

|**Order referrer**
|Select the order referrer from the drop-down menu. The selected referrer will be added to the product URL so that sales can be analysed later.

|**URL parameter**
|Enter a suffix for the product URL if this is required for the export. If you activated the **transfer** option for the product URL further up, then this character string will be added to the product URL.

|**Item name**
|Select **Name 1**, **Name 2** or **Name 3**. The names are saved in an item's **Texts** tab. +
Enter a number into the **Maximum number of characters (def. Text)** field if desired. This will specify how many characters should be exported for the item name.

|**Preview text**
|Decide whether you want to transfer preview text. If so, then select the text. +
Enter a number into the **Maximum number of characters (def. Text)** field if desired. This will specify how many characters should be exported for the preview text. +
Activate the option **Remove HTML tags** if you want HTML tags to be removed during the export. +
If you only want to allow specific HTML tags to be exported, then enter these tags into the field **Permitted HTML tags, separated by comma (def. Text)**. Use commas to separate multiple tags.

|**Description**
|Decide whether you want to transfer a description. If so, then select the text. +
Enter a number into the **Maximum number of characters (def. Text)** field if desired. This will specify how many characters should be exported for the description. +
Activate the option **Remove HTML tags** if you want HTML tags to be removed during the export. +
If you only want to allow specific HTML tags to be exported, then enter these tags into the field **Permitted HTML tags, separated by comma (def. Text)**. Use commas to separate multiple tags.

|**Barcode**
|Select the ASIN, ISBN or an EAN from the drop-down menu.

|**Image**
|Choose between **Position 0** and **First image**. +
**Position 0** = An image with position 0 will be transferred. +
**First image** = The first image will be transferred.

|**Offer price**
|Activate to transfer the offer price.

|**RRP**
|Activate to transfer the RRP.

|**Shipping costs**
|Activate this option if you want to use the shipping costs that are saved in a configuration. If this option is activated, then you will be able to select the configuration and the payment method from the drop-down menus. +
Activate the option **Transfer flat rate shipping charge** if you want to use a fixed shipping charge. If this option is activated, then you will be able to enter a monetary amount into the field below.

|**VAT note**
|Enter the amount. This amount includes the VAT but does not include shipping.

|**Item availability**
|Activate the **overwrite** option and enter item availabilities into the fields **1** through **10**. This will overwrite the item availabilities that are saved in the menu **Settings » Item » Item availability**.
|====

== Editing export formats

All of the export formats that you created are displayed in the overview. Use the **Customise the layout** function to select the columns to be displayed. Point your cursor over the title bar and click on **Customise the layout**. A pop-up window will open and you can activate or deactivate the columns. The following columns are available:

* ID
* Name
* Download
* Type
* Format
* Limit
* Last export change
* Create export
* Delete

Open an export format to check or edit its settings.

[.instruction]
Editing an export format:

. Go to **Data exchange » Export » Elastic export**.
. Open the export format.
. Change the settings. Pay attention to the information given in tables 2 and 3.
. **Save** the settings.

== Filtering export formats

You can use specific criteria to filter your export formats. Only these export formats will then be displayed in the overview within the **Exports** tab.

[.instruction]
Filtering export formats:

. Go to **Data exchange » Export » Elastic export**.
. Carry out the settings in the **Filter** area. Pay attention to the information given in table 4.
. Click on **Search**.

[cols="a,a"]
|====
|Filter |Explanation

|**ID**
|Enter a number.

|**Name**
|Enter a name or part of a name.

|**Type**
|Select the type **Item** or **All** from the drop-down menu.

|**Format**
|Select a format from the drop-down menu. You will see a list of all the export formats that match the selected format.
|====

__Table 4: filter settings in the menu **Data exchange » Export » Elastic export**__

[NOTE]
.Deleting search criteria
====
Click on **Reset** to delete the search criteria. Then click on **Search** again. All of your export formats will be displayed in the overview and you can select new filters.
====
