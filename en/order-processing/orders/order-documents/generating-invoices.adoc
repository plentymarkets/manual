== Generating invoices

In this menu, invoices are generated with the help of a **group function**. For example, all invoices that have currently been cleared for shipping could be generated with a search request that has **order status [5] cleared for shipping**.

== Options

You can carry out settings related to the **layout** of the invoices by going to **System » Client » Select client » Locations » Select location » Documents**.

Go to **Settings &gt; Orders &gt; Event procedures** in order to define which additional **procedure(s)** should be carried out when the invoice is generated. For example, you can configure an **event procedure** for automatically sending the invoice with an email template.

[TIP]
.Tip
====
This menu also contains a direct link to the **Event procedures** menu. It can be accessed by clicking on the **Event procedure** link.
====

== Generating invoices

[NOTE]
.Generating invoices:
====
Go to **generate invoices** (see path above).

Pay attention to the explanations given in table 1 and carry out the **settings** for the **search filter** as desired.

Under **procedure**, select an **invoice date** and the **order status** that you would like to have assigned to the selected orders after the invoices were successfully created.

Click on the **gear-wheel icon** after having selected all of the options. +
→ You will receive a **PDF file**, which contains the invoices of all the orders that match the filter options you selected.
====

|====
|Setting |Explanation

|**Order status**
|Select the current **status** of the orders that you would like to generate invoices for.

|Responsible
|Select a person **responsible** for the orders that you are searching for. Alternatively, select the option **ALL**.

|Client (store)
|Select one of the **stores** or select the option **ALL**.

|**Warehouse**
|Limit the search to a particular **warehouse** or carry out the search for **ALL** warehouses.

|**Payment method**
|Select one of the **payment methods** or select the option **ALL**.

|**Sorting**
|Determine the identifying feature that you would like to use for sorting the invoices that are generated. Alternatively, select the option **NONE** if you don't want to have the invoices be sorted in any particular way. Options: **Order ID**,**Customer ID**, **Item ID**.

|**Limit**
|You have to limit the number of invoices that can be generated to one of the values in the drop-down list. The maximum is **200** invoices.

|**Invoice date**
|Select a **date from the calendar**, which you want to have displayed on the invoice as its date.

|**Change order status**
|Select a new **order status** that you would like to have assigned to the selected orders after the invoices were successfully retrieved. Alternatively, select the option **no change**.

|**Change invoice layout**
|Click on this link to open the **Settings &gt; Orders &gt; Documents** area in a new window. There you can carry out detailed settings for the invoice template.

|**Event procedure**
|Click on this link to open the **Settings &gt; Orders &gt; Event procedures** area in a new window. There you can create new event procedures and edit existing ones.
|====

__Table 1: **settings** in the **generate invoices** menu__
