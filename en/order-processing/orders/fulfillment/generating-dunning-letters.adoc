== Generating dunning letters

In this menu, dunning letters are generated with the help of a **group function**. By selecting the corresponding search filter settings, you could e.g. generate dunning letters for **all of the orders from a specific time period** that are still currently in **status [3] waiting for payment**.

== Options

You can carry out settings related to the layout of the dunning letters by going to **Settings &gt; Orders &gt; Documents**.

Go to **Settings &gt; Orders &gt; Event procedures** in order to define which additional **procedure(s)** should be carried out when the dunning letter is generated.

== Generating dunning letters

[NOTE]
.Generating dunning letters:
====
Go to **Generate dunning letters** (see path above).

Pay attention to the explanations given in table 1 and carry out the **settings** for the **search filter** as desired.

Under **procedure**, select the order status that you would like to have assigned to the selected orders after the dunning letters were successfully created. Additionally, enter a **comment** if desired.

Click on the **gear-wheel icon** after having selected all of the options. +
→ You will receive a **PDF file**, which contains the dunning letters of all the orders that match the filter options you selected.
====

|====
|Setting |Explanation

|**Order date**
|Limit the orders that should receive dunning letters by **selecting starting and ending dates** from the calendars. In doing so, you will find those orders that have a order date within the selected time period.

|**Order status**
|Select the current **status** of the orders that you would like to generate dunning letters for.

|**Country**
|Select a **country** that the search should be limited to. Alternatively, select the option **ALL**.

|Owner
|Select an **owner** for the orders that you are searching for. Alternatively, select the option **ALL**.

|Client (store)
|Select one of the **stores** or select the option **ALL**.

|**Warehouse**
|Limit the search to a particular **warehouse** or carry out the search for **ALL** warehouses.

|**Payment method**
|Select one of the **payment methods** or select the option **ALL**.

|**Sorting**
|Determine the identifying feature that you would like to use for sorting the dunning letters that are generated. Alternatively, select the option **NONE** if you don't want to have the dunning letters sorted in any particular way.

|**Change order status**
|Select a new **order status** that you would like to have assigned to the selected orders after the dunning letters were successfully retrieved. Alternatively, select the option **no change**.

|**Item ID**
|You have the possibility of creating a **dunning charge** as a separate item. Once you have done this, you can enter the corresponding **item ID of the dunning charge** here. This will then automatically be added to all of the dunning letters that are generated.

|**Comment**
|You can compose a **text** here, which you would like to have appear on all of the dunning letters **underneath the stock units**.
|====
