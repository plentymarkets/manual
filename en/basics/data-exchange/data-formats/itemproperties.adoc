== General information about the data format ItemProperties

The data format **ItemProperties** is used for editing item property links via data exchange. You can use it to import, edit and/or export item property links.

== Important notes

[IMPORTANT]
.Mandatory fields
====
**ItemID** or **ItemNo** +
If the import contains texts, then the fields **Lang** and **Value** are mandatory fields.

Mandatory fields must be included and need to have an accepted value saved for them.
====

[WARNING]
.Spell the field names correctly!
====
Pay close attention to the exact spelling of the data fields (upper/lower case letters). The data can not be processed if the data field is spelled incorrectly.
====
