The data format **VariationBarcode** is used to export, edit and import barcodes. Barcodes are created in the **System » Item » Barcode** menu. Create and export the data format in the **Data » Dynamic export** menu. Import the data format in the **Data » Dynamic import** menu. The <<basics/data-exchange/exporting-data#, Exporting data>> and <<basics/data-exchange/importing-data#, Importing data>> pages of the manual describe how you export and import the data format.

Pay close attention to the exact spelling of the data fields. The data cannot be processed if the data fields are spelled incorrectly.

Table 1 explains the data fields for the **VariationBarcode** data format in alphabetical order as well as filter options that you set in the **Filter** tab of the data format.

== Overview of the synchronisation fields

The data fields that are listed in table 2 are available for <<basics/data-exchange/importing-data#25, data synchronisation>>. These data fields are mandatory synchronisation fields. For mandatory synchronisation fields, select the option **Synchronisation** as **Import procedure**. At least one of the data fields from the following table must be selected for synchronisation.
