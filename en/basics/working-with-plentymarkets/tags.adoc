Tags are keywords that can be used to mark data records in plentymarkets. Go to *Setup » Settings » Tags* to create and manage tags. These tags can be linked to data records in the following areas of plentymarkets:

* Board
* Category
* Ticket
* Contact
* Variation
* Order

[#50]
== Assigning rights for back end users

Users of the type *Back end* can only access limited areas and menus in plentymarkets. Thus, their access to the system is limited. If you want users of this type to work with tags, an *Admin* user has to assign them rights as listed in the following.

[.instruction]
Assigning rights for back end users:

. Go to *Setup » Settings » User » Rights » User*.
. Use the search function (icon:search[role="blue"]) and open the user account that should be edited.
. In the *Authorisations* area, activate the rights as listed in <<table-tags-required-rights-back-end-user>>.
. *Save* (icon:save[role="green"]) the settings.

[[table-tags-required-rights-back-end-user]]
.Creating a tag
[cols="1,3"]
|====
|Setting |Explanation

|*Setup > Tag > Update tag*
|Allows the back end user to update tags.

|*Setup > Tag > Show tag*
|Allows the back end user to see tags.

|*Setup > Tag > Create tag*
|Allows the back end user to create tags.

|*Setup > Tag > Delete tag*
|Allows the back end user to delete tags.

|====

== Creating a tag

Proceed as described below to create a tag.

[.instruction]
Creating a tag:

. Go to *Setup » Settings » Tags*.
. Click on *Create new tag* (icon:plus-square[role="green"]). +
→ A window opens on the right-hand side.
. Carry out the settings. Pay attention to the information given in <<table-tag-settings>>.
. *Save* (icon:save[role="green"]) the settings. +
→ The tag is created and added to the list.

[[table-tag-settings]]
.Creating a tag
[cols="1,3"]
|====
|Setting |Explanation

|*Name EN*
|Enter a name for the tag. +
*_Tip:_* Tag names can contain spaces. This allows you to create tag names with two or more words.

|*Colour*
|*_Optional:_* Enter a hexadecimal value of a colour for the tag. For the 16 primary colours, you can enter the name of the colour (black, red, silver, white etc.) instead of the hexadecimal value. +
*_Tip:_* Click on the square to the right to open the colour picker.

|*Area*
|Select the areas of the system where the tag is available. +
*Board* = Tag can be linked to task cards on the boards of the *plentymarkets Logo (Start) » Boards* menu. +
*Category* = Tag can be linked to categories. +
*Ticket* = Tag can be linked to tickets. +
*Contact* = Tag can be linked to contacts. +
*Variation* = Tag can be linked to variations. +
*Order* = Tag can be linked to orders.

|Name in additional languages
|Enter tag name in additional languages. For further information, refer to the <<basics/working-with-plentymarkets/tags#200, Adding additional languages to tags>> chapter.
|====

[#200]
== Adding tag names in additional languages

Proceed as described below to add names in additional languages to an existing tag.

[.instruction]
Adding tag names in additional languages:

. Go to *Setup » Settings » Tags*.
. Click on *Search* (icon:search[role="blue"]). +
→ All tags are shown.
. Click on the tag. +
→ A window opens on the right-hand side.
. Enter translations for the languages you need.
. *Save* (icon:save[role="green"]) the settings.

[#300]
== Filtering by tags

You can filter tags by *Name*, *Availability* and/or *Language*. Proceed as described below to filter tags based on these criteria.

[.instruction]
Filtering by tags:

. Go to *Setup » Settings » Tags*.
. Select one or multiple filters. Pay attention to the information given in <<table-tag-filter>>.
. Click on *Search* (icon:search[role="blue"]). +
→ The tags that correspond to the search criteria entered are displayed.

[[table-tag-filter]]
.Tag filters
[cols="1,3"]
|====
|Setting |Explanation

|*Name*
|Enter the name of the tag. +
*_Tip:_* If all languages were selected, all languages are included here as well. However, only the value that was entered under *Name EN* is shown.

|*Area*
|Select one or more areas to only search for tags that are activated for these areas.

|*Language*
|Select one or more languages to show tags with names in these languages.
|====
