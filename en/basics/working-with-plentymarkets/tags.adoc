Tags are keywords that can be used to mark data records in plentymarkets. Go to *System » System settings » Settings » Tags* to create and manage tags. These tags can be linked to data records in the following areas of plentymarkets:

* Board
* Category
* Ticket
* Contact
* Variation
* Order


[#50]
== Setting the visibility for back end users

In order that the *System » System settings » Settings » Tags* menu is visible for users with *Back end* access, the menu visibility as well as further REST rights must be activated in the user account.

[.instruction]
Setting the visibility for back end users:

. Go to *System » System settings » Settings » User » Rights » User*. +
→ The overview opens.
. *_Optional_*: In the filter settings on the left-hand side, enter an ID or a name to limit the search results.
. Click on *Search*. +
→ The accounts that correspond to the search criteria entered are shown. +
*_Tip:_* Accounts that are shown greyed out have admin rights.
. Click on the account for that you want to activate the visibility and the rights. +
→ The view *Rights* opens.
. In the area *Menu visibility*, expand the option *System > System settings > Settings*.
. Activate the option *Tag*. +
→ *_Note:_* In the *Rights* area, the option for *Tag > Show tag* is automatically activated.
. *Save* the settings.

== Creating a tag

Proceed as described below to create a tag.

[.instruction]
Creating a tag:

. Go to *System » System settings » Settings » Tags*.
. Click on *Create a new tag*. +
→ A window opens on the right-hand side.
. Carry out the settings. Pay attention to the information given in <<table-tag-settings>>.
. *Save* the settings. +
→ The tag is created and added to the list.

[[table-tag-settings]]
.Creating a tag
[cols="1,3"]
|====
|Setting |Explanation

|*Name EN*
|Enter a name for the tag. +
*_Tip:_* Tag names can contain spaces. This allows you to create tag names with two or more words.

|*Colour*
|*_Optional:_* Enter a hexadecimal value of a colour for the tag. For the 16 primary colours, you can enter the name of the colour (black, red, silver, white etc.) instead of the hexadecimal value. +
*_Tip:_* Click on the square to the right to open the colour picker.

|*Area*
|Select the areas of the system where the tag is available. +
*Board* = Tag can be linked to task cards on the boards of the *Start » Boards* menu. +
*Category* = Tag can be linked to categories. +
*Ticket* = Tag can be linked to tickets. +
*Contact* = Tag can be linked to contacts. +
*Variation* = Tag can be linked to variations. +


|Name in additional languages
|Enter tag name in additional languages. For further information, refer to the <<basics/working-with-plentymarkets/tags#200, Adding additional languages to tags>> chapter.
|====

[#200]
== Adding tag names in additional languages

Proceed as described below to add names in additional languages to an existing tag.

[.instruction]
Adding tag names in additional languages:

. Go to *System » System settings » Settings » Tags*.
. Click on *Search*. +
→ All tags are shown.
. Click on the tag. +
→ A window opens on the right-hand side.
. Enter translations for the languages you need.
. *Save* the settings.

[#300]
== Filtering by tags

You can filter tags by *Name*, *Availability* and/or *Language*. Proceed as described below to filter tags based on these criteria.

[.instruction]
Filtering by tags:

. Go to *System » System settings » Settings » Tags*.
. Select one or multiple filters. Pay attention to the information given in <<table-tag-filter>>.
. Click on *Search*. +
→ The tags that correspond to the search criteria are shown.

[[table-tag-filter]]
.Tag filters
[cols="1,3"]
|====
|Setting |Explanation

|*Name*
|Enter the name of the tag. +
*_Tip:_* If all languages were selected, all languages are included here as well. However, only the value that was entered under *Name EN* is shown.

|*Area*
|Select one or more areas to only search for tags that are activated for these areas.

|*Language*
|Select one or more languages to show tags with names in these languages.
|====
