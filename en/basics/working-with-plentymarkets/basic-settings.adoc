= Basic settings
:lang: en
include::{includedir}/_header.adoc[]
:position: 8

In the **plentymarkets » Settings » Basic settings** menu, carry out basic settings for working with plentymarkets. You will save your company's **<<basics/working-with-plentymarkets/basic-settings/master-data#, master data  >>** and <<order-processing/payment/managing-bank-details#, **bank details**  >>. You can also create and manage <<basics/working-with-plentymarkets/manage-users#, **users**  >>. Furthermore, you can save **<<basics/working-with-plentymarkets/basic-settings/tags#, tags  >>**. If you work with the **<<basics/plentymarkets-client#, plentymarkets Client  >>** then this is where you configure your **<<basics/working-with-plentymarkets/basic-settings/printer#, printer  >>**.

== Links to other menus

Syntax variables display master data as well as bank details e.g. in **<<crm/sending-emails#1200, E-mail templates >>**.

Tags are keywords that can be used to mark texts and pages. In plentymarkets, tags are created and managed centrally in one menu. The tags can then be used in different areas, such as in **<<item/categories#, categories  >>** and the **<<crm/using-the-ticket-system#, ticket system  >>**.
