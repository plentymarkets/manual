==  Setting up Mercateo in plentymarkets

Before you can set up Mercateo in plentymarkets, you need to have data for the FTP server that is to be used for the upload to Mercateo, as well as an email account for the order import.

[.instruction]
Setting up Mercateo in plentymarkets:

. Go to **Settings » Markets » Mercateo » Basic settings**.
. Carry out the settings as desired. Pay attention to the information given in table 1.
. **Save** the settings.

.basic settings for the market Mercateo
[cols="a,a"]
|====
| Setting | Explanation

|**Activate the automatic data exchange**
| Activate in order to automatically exchange data with Mercateo.

2+| General

|**Item ID**
| Select whether the **Variation ID** or the **Variation No.** should be used for the item data.

|**Transfer item number**
| Only select **Yes** if you are the manufacturer of the item. The selection that you make in the **Item ID** drop-down menu does not affect the option here. For example, if you are the manufacturer of the item and you already selected variation number above, then you will still have to select the option **Yes** here.

|**Item name**
| Select **Name 1**, **Name 2**, **Name 3** or **Variation name**. This name will be displayed on Mercateo.

|**Add attribute value to item name**
| Select **YES** if the attribute value should be displayed behind the item name on Mercateo.

|**Item description**
| Select, which item description should be transferred to Mercateo.

2+| Input

|**FTP server**
| Enter the URL of the FTP server for the data upload to Mercateo.

|**FTP user**
| Enter the user name of the FTP server.

|**FTP password**
| Enter the password of the FTP server.

2+| Retrieval

|**User name**
| Enter the user name for the order retrieval.

|**Password**
| Enter the password for the order retrieval.

|**IMAP server**
| Enter the server address for the order retrieval.

|**Port**
|Enter port.

|**Encryption**
|Enter encryption .

2+|Other catalogues

|**Suffix synchronisation No. 2 - 4**
| Enter the **suffix** for the additional catalogue of items.

|**Order referrer**
| Select the **order referrer** for the import of orders.

|**Automatic order retrieval**
| Activate in order to automatically retrieve orders.

2+| Updates

|**Automatic price update**
| Activate in order to hourly update the item prices.

|**Automatic delivery status update**
| Activate in order to automatically update the delivery status.
|====

Connecting to this market allows for the following automatic functions to take place:

* Data exchange
* Order import
* Hourly price synchronisation
* Delivery status synchronisation

== Setting the item availability

Items have to be available for Mercateo. This is done in the **Availability** tab of an item within the <<item/managing-items#170, **Item » Edit item » Tab: Variation ID**>> menu.

[.instruction]
Setting the item availability for Mercateo:

. Open the **Item » Edit item » Open item » Tab: Variation ID » Tab: Settings** menu.
. Activate the main variation in the **Availability** section.
. Click on the **Availability** tab.
. Click in the **Markets** section in the selection field.  +
→ A list with all available markets will be displayed.
. Activate the option **Mercateo**.
. Activate the option **Web API**.
. Click on **Add**. +
→ The Market will be added.
. **Save** the settings.  +
→ The item is available on Mercateo.

The availability for variations can be individually edit in the **Item » Edit item » Open item » Tab: Variation ID » Open variation » Tab: Availability** menu.

==  Exporting items and item images

In this menu, you can export your items and item images to Mercateo.

[.instruction]
Exporting items:

.  Go to **Settings » Markets » Mercateo » Data export**.
.  Click on **Export item data**. +
→ The item data will be exported to Mercateo.

You can also export item images to Mercateo. A maximum of 100 images can be uploaded at once in JPEG format. No image will be uploaded twice.

[.instruction]
Exporting item images:

.  Go to **Settings » Markets » Mercateo » Data export**.
.  Click on **Export 100 item images:**. +
→ The item images will be exported to Mercateo.

== Defining a sales price

Proceed as described below to define a sales price for the order referrer Mercateo. This price will be displayed on Mercateo.

[.instruction]
Defining a sales price for Mercateo:

. Open the menu **Settings » Item » Sales price » Open sales price » Tab: Settings**.
. Place a checkmark next to the referrer **Mercateo**.
. **Save** the settings.

[#600]
==  Viewing the export history

An overview of exported images is displayed in this menu. The images will be displayed on different pages. 25 images can be displayed per page.

[.instruction]
Selecting a page:

.  Go to **Settings » Markets » Mercateo » Export history**.
.  Select a **Page** from the drop-down menu.
. Click on the **gear-wheel icon** to display the page.

The images will be displayed along with the corresponding item ID and time of export. It is not possible to export an image more than once. However, if you want to update an image, e.g. after changes have been made to it, then click on **Delete** in the **Release item image** column. Then export this image again in the **Item export** menu.
