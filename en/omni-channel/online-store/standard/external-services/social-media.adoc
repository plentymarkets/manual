== Social Media

In this menu, you can save **URLs** for **social media** platforms like Facebook and twitter. Furthermore, you can activate or deactivate the link to social media.

== Configuration

Pay attention to the explanations given in the following table and carry out the configuration as desired.

image::omni-channel/online-store/standard/external-services/assets/EN-Settings-Client-Standard-Services-Social-Media-01-SI.png[]

__Image 1: settings in the **Social Media** menu__

The following table explains the settings in the **Settings » Client (store) » Standard » Services » Social Media** menu:

[cols="a,a"]
|====
|Setting |Explanation

|**Social Media**
|Select whether you would like the link to all social media platforms to be **active** or **inactive**.

|**Facebook URL**
|Enter the **URL** that you want to use for the link to **Facebook**.

|**Facebook login**
|Select **active** if you want to use the **<<omni-channel/online-store/global-settings/external-services/facebook#, Facebook login function  >>**. The default setting is **inactive**, i.e. the login function will not be used.

|**Twitter URL**
|Enter the **URL** that you want to use for the link to **twitter**.

|**Google+ URL**
|Enter the **URL** that you want to use for the link to **Google+**.
|====
