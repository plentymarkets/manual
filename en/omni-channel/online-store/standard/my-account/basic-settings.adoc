== General information about the basic settings

This menu contains basic settings for the individual customer area of your online store. Here you can select whether specific features should be activated or deactivated and which information should be displayed in the **<<omni-channel/online-store/cms/web-design/editing-the-web-design/my-account#, My Account  >>** area, for example weights and options for displaying the prices.

== Configuration

The settings of this menu will be explained below in table 1. There you will also find references and links to further pages of the manual that provide detailed information about those settings and functions.

__Image 1: options in the **Basic settings** menu__

[.instruction]
Configuring basic settings for the My Account area:

. Go to **Settings » Client (store) » Standard » My Account » Basic settings**.
. Pay attention to the explanations given in the following table and carry out the settings as desired.
. Click on the **save icon** to save the settings.

[cols="a,a"]
|====
|Setting |Explanation

|**Customer login**
|Select which data your customers should use to log into the online store. The **e-mail address and password** or the **customer ID and password**.

|**Validity of checkout URL**
|Select how long a checkout URL should be valid after it was generated. You can select different options between **7** and **360 days** or you can select **Never**. If you select the option **Never**, then customers will have to log into the My Account area in order to access order information with the checkout URL.

|**Watchlist**
|**Activate** or **deactivate** the watchlist in the online store. +
__**Info**__: The submenu under <<omni-channel/online-store/cms/web-design/editing-the-web-design/my-account#, **CMS » Web design » My account**  >> becomes visible once this function has been activated.

|**Wish list**
|**Activate** or **deactivate** the **wish list**. +
__**Info**__: The submenu under **<<omni-channel/online-store/cms/web-design/editing-the-web-design/my-account#, CMS » Web design » My account  >>** becomes visible once this function has been activated.

|**Documents**
|**Activate** or **deactivate** the function for downloading documents in the online store.

|**Data export**
|**Activate** or **deactivate** the function for downloading data formats in the online store.

|**My coupons**
|**Activate** or **deactivate** the function for displaying the customer's coupons.[#10]

|**Return**
|Specifies whether customers should be allowed to create returns within their own accounts. +
**Activated** = Customers can create returns. However, labels will not be provided. You will need to decide how you want to proceed when customers return items. For example, you can send customers the label via e-mail. +
<<order-processing/fulfilment/preparing-the-shipment#3200, **DHL Retoure Online** >> = Customers can create returns, then download and print the DHL return label. +
**Deactivated** = Customers can not create returns. If you select this option, then you will need to decide how your company deals with returns. For example, you may require customers to call or send an e-mail if they want to return an item. +
If you allow customers to create returns themselves, then they will be able to choose the appropriate <<order-processing/orders/settings/sales-order-types/return#, **reason for return**  >> in their "My account" area.

|**Price display for wish list and watchlist**
|There are the following options for displaying prices in the online store: **Item price only**, **Item price with asterisk (*)**, **Price including VAT and shipping** +
__**Tip**__: The option **Item price with asterisk (*)** can be used to give additional price information below the asterisk, e.g. to give VAT information. +
__**Info**__: The option **price including VAT and shipping** is required if you want your online store to be certified by Trusted Shops.

|**Shipping weight in wish list and watchlist**
|**Activate** or **deactivate** the function for displaying the shipping weight in the wish list and watchlist. +
__**Important**__: If you decide to display this information, then go to <<_includes/number-formats#, **Settings » Basic settings » Number formats**  >> and check the settings for number formats. For example, if **in kilograms: no decimal place** is set here, then weights that are less than 1000 grams will be displayed as 0.

|**Subscriptions**
|**Activate** or **deactivate** the function for displaying the **Subscription** area. +
__**Info**__: The submenu under **<<omni-channel/online-store/cms/web-design/editing-the-web-design/my-account#, CMS » Web design » My account  >>** becomes visible once this function has been activated.

|**Payment method: Subscriptions**
|Select whether customers should be able to **change the payment method** for a subscription.

|**Download link for SEPA direct debit mandate**
|**Activate** or **deactivate** the function for displaying the download link.

|**Change e-mail address**
|**Activate** or **deactivate** the function for changing the e-mail address. +
In order to change the address, the account's password has to be entered.

|**Change password**
|**Activate** or **deactivate** the function for changing the <<basics/working-with-plentymarkets/manage-users#, **password**  >>.

|**E-mail confirmation when changing the password**
|**Activate** or **deactivate** the automatic e-mail confirmation when the password is changed.

|**Hide logout**
|**Activate** or **deactivate** the function for displaying the logout.
|====

__Table 1: options in the **Basic settings** menu__
