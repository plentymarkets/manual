==  General information about etracker

link:https://www.etracker.com/en.html[etracker^] offers a wide range of tools for web analytics. The prices vary, depending on the scope of services provided. You can try out the service packages before deciding which one you would like to use.

==  Setting up etracker in plentymarkets

Once you have acquired a license from etracker, set up the analysis tool in plentymarkets as described below.

[.instruction]
Copying the basic code from etracker:

. Log into the admin area of *etracker* and open the *Configuration* folder. Then open the menu *Setup/HTML code*.
. Click on the menu item *Basic code* in the top right corner.
. Copy the complete source text below *Basic code*, starting with the line *-- etracker PARAMETER 2.4 --*.
. Insert the code into plentymarkets as described below.

.inserting the basic code from etracker
image::omni-channel/online-store/setting-up-clients/extras/assets/EN-Settings-Client-Conversion-Tracking-etracker-01.png[]

[.instruction]
Setting up etracker in plentymarkets:

. Go to *CMS » Web design » Folder: Layout*.
. Open the folder *PageDesign* and then the template *PageDesignContent*.
. Click on the *PageDesignContentMainFrame* tab.
. Insert the *basic code* at the end of the text field.
. *Save* the settings. +
*→* Now the code is active and etracker starts collecting data.
