**System » Settings » Language**

== General information about the language menu

In this menu, you can configure the **__language variety__** that you use in your online store. For example, you can select varieties of German for Austria and Switzerland. This setting is important if you want to use the translation services offered by **<<omni-channel/online-store/extras/lionbridge#, Lionbridge>>**. The language varieties saved in this menu are the target languages for Lionbridge.

== Configuration

Save a **__language variety__** for each language that you use in your store.

image::omni-channel/online-store/extras/assets/EN-Settings-Basic-settings-Language-01.png[]

__Image 1: **selecting a language**__

[.instruction]
Selecting a language:

. Go to **System » Settings » Language**.
. Save a **language code** for each language that you use in your store.
. Click on the **save icon** to save the settings.
