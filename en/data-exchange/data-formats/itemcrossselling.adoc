== General information about the data format ItemCrossSelling

The data format **ItemCrossSelling** is used for the import/export as well as for editing <<item/managing-items#950., **cross selling items** >>.

== Important notes

[IMPORTANT]
.Mandatory fields
====
**ItemID** or **ItemNo** and **CrossItemID** or **CrossItemNo**

Mandatory fields must be included and need to have an accepted value saved for them.
====

[WARNING]
.Spell the field names correctly!
====
Pay close attention to the exact spelling of the data fields (upper/lower case letters). The data can not be processed if the data field is spelled incorrectly.
====

== Overview of the data fields

The following table explains the data fields for the data format **ItemCrossSelling**. The data fields are listed in alphabetical order:

[cols="a,a,a"]
|====
|Field |Explanation |Filter options (tab: <</#, Filter >>)

|**CrossItemID**
|Cross-selling item ID
|

|**CrossItemNo**
|Cross-selling item number
|

|**CrossSellingDelete**
|Delete cross-selling item +
**0** = No +
**1** = Yes
|

|**ItemID**
|Item ID
|

|**ItemNo**
|Item number
|

|**Relationship**
|Relationship between item and its corresponding cross-selling item +
**Accessory** = accessories +
**ReplacementPart** = replacement part +
**Similar** = similar item +
**Bundle** = item bundle
|Select the operator and value for the data field **Relationship**.
|====

__Table 1: data fields of the data format **ItemCrossSelling**__
