= ItemListingMarketText
:lang: en
include::{includedir}/_header.adoc[]
:keywords: Data format ItemListingMarketText
:position: 160

Use the data format **ItemListingMarketText** to  exportieren the Listing's description, subtitle and title in different languages. Create and export the data format in the **Data exchange » Export » Dynamic export** menu. Import the data format in the **Data exchange » Import » Dynamic import** menu. The **<<data-exchange/exporting-data#, Exporting data  >>** and **<<data-exchange/importing-data#, Importing data  >>** pages of the manual describe how you export and import the data format.

== Important notes

[WARNING]
.Spell the field names correctly!
====
Pay close attention to the exact spelling of the data fields (upper/lower case letters). The data can not be processed if the data field is spelled incorrectly.
====

== Overview of the data fields

The following table explains the data fields for the data format **ItemListingMarketText**. The data fields are listed in alphabetical order:

.Data fields of the data format **ItemListingMarketText**
[cols="a,a,a"]
|====
|Field |Description |Filter

|**Description**
|Description text
|Select the operator and enter the value.

|**Lang**
|Language
|Select the operator and enter the value.

|**ListingMarketID**
|Market-ID
|Select the operator and enter the value.

|**Subtitle**
|The subtitle of the Listing
|Select the operator and enter the value.

|**Title**
|The title of the Listing
|Filter aktivieren, Operator wählen und Wert eingeben.
|====
