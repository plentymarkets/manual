= OrderComplete
:lang: en
include::{includedir}/_header.adoc[]
:keywords: OrderComplete, Data format, Data export, Data exchange
:position: 200

The data format **OrderComplete** is used to export order. This data format can only be used for the export. Create and export the data format in the **Data exchange » Export » Dynamic export** menu. The **<<data-exchange/exporting-data#, Exporting data  >>** page of the manual describes how you export the data format.

Pay close attention to the exact spelling of the data fields. The data cannot be processed if the data fields are spelled incorrectly.

== Overview of the procedures

In the **Procedures** tab, select which procedures you want to be carried out during the data export. Activate the **OrderStatus** procedure and select an order status from the drop-down menu. The orders will then be set to this order status during the import.
