= VariationBundle

:description: Data format VariationBundle
:position: 250

The data format **VariationBundle** is used to export, edit and import variation bundles. Create and export the data format in the **Data exchange » Export » Dynamic export** menu. Import the data format in the **Data exchange » Import » Dynamic import** menu. The **<<data-exchange/exporting-data#, Exporting data  >>** and **<<data-exchange/importing-data#, Importing data  >>** pages of the manual describe how you export and import the data format.

Pay close attention to the exact spelling of the data fields. The data cannot be processed if the data fields are spelled incorrectly.

toc::[]

== Overview of the mandatory fields

The following mandatory fields must be available for the data format **VariationBundle** and need to have a valid value saved for them:

* **Quantity**
* **VariationID**
* **VariationSourceID**
