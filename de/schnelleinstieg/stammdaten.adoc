Im Menü *System » Einstellungen » Stammdaten* hinterlegen Sie die Stammdaten Ihres Unternehmens. Sie speichern hier zentral die Adresse und Kontaktdaten und die Umsatzsteuer-ID des Unternehmens. Eine Änderung dieser Daten wirken sich auf das ganze System aus.

[WARNING]
.Bei Menüänderungen werden Benutzer ausgeloggt
====
Wenn Sie in diesem Menü Änderungen vornehmen, werden andere Benutzer automatisch ausgeloggt und müssen sich neu einloggen. Kündigen Sie Änderungen daher rechtzeitig an.
====

[.instruction]
Stammdaten speichern:

. Öffnen Sie das Menü *System » Einstellungen » Stammdaten*.
. Nehmen Sie die Einstellungen vor. Beachten Sie <<tabelle-stammdaten-speichern>>.
. Klicken Sie auf *Speichern*. +
→ Die Stammdaten werden gespeichert.

[IMPORTANT]
.Vertragsdaten ändern
====
Eine Änderung dieser Stammdaten ändert nicht die Kundenstammdaten Ihres plentymarkets Vertrages z.B. für die Rechnungsstellung. Vertragsdaten passen Sie im Menü *Support » Service-Center* an.
====

[[tabelle-stammdaten-speichern]]
.Stammdaten speichern
[cols="1,3"]
|====
|Einstellung |Erläuterung

|*Firmenname*
|Namen des Unternehmens eingeben.

|*Geschäftsführer*
|Vor- und Zunamen der Person eingeben, die das Unternehmen führt.

|*Straße, Hausnummer*; +
*PLZ*; +
*Ort*
|Adressdaten des Unternehmens eingeben.

|*Region*
|Region des Unternehmens aus der Dropdown-Liste wählen.

|*Zeitzone*
|Zeitzone des Unternehmens aus der Dropdown-Liste wählen, damit Datum und Uhrzeit, z.B. in Aufträgen oder Tickets, korrekt ausgegeben werden.

|*Land*
|Land des Unternehmens aus der Dropdown-Liste wählen.

|*E-Mail*; +
*Telefon*; +
*Telefax*
|Kontaktdaten des Unternehmens eingeben.

|*Hotline*
|Hotline-Telefonnummer des Unternehmens eingeben. +
**__Hinweis:__** Wenn das Unternehmen keine Hotline-Nummer hat, die im Feld *Telefonnummer* eingegebene Telefonnummer auch in dieses Feld eingeben. Die Ausgabe der Hotline-Nummer ist über die Template-Variable *$CompanyHotline* an verschiedenen Stellen im System vorkonfiguriert.

|*Umsatzsteuer-ID*
|Umsatzsteuer-ID des Unternehmens eingeben. Diese Umsatzsteuer-ID wird auch für Mandanten verwendet, bei denen im Menü <<auftragsabwicklung/auftraege/buchhaltung#, Buchhaltung>> keine Umsatzsteuer-ID hinterlegt wurde.
|====

[TIP]
.Daten in Webshop-Templates enthalten
====
Die Felder, die im Menü *System » Einstellungen » Stammdaten* konfiguriert werden, können über Template-Variablen z.B. in E-Mail-Vorlagen und im Webshop angezeigt werden und sind dort zum Teil bereits enthalten. +
Die Stammdaten zeigen Sie über die folgenden Template-Variablen an: +
*$CompanyCEO* +
*$CompanyCity* +
*$CompanyCountry* +
*$CompanyEmail* +
*$CompanyFax* +
*$CompanyFon* +
*$CompanyHotline* +
*$CompanyIsSmallBusiness* +
*$CompanyName* +
*$CompanyStreet* +
*$CompanyVATNumber* +
*$CompanyZIP* +
====
