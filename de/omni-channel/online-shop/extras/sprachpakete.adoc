== Sprachpakete

Die Sprachpakete enthalten die vom System verwendeten Begriffe und Texte, die vorgegeben sind und automatisch im jeweiligen Bereich eingesetzt werden, z. B. **Kategorie wählen** oder **in den Warenkorb**.

== Texte in Sprachpaketen bearbeiten

In diesem Menü haben Sie die Möglichkeit, die vom System verwendeten Texte des Bestellvorgangs sowie des Mein-Konto-Bereichs anzupassen. Sie können an den Standardübersetzungen Änderungen vornehmen, wenn Sie andere Begriffe oder Formulierungen verwenden möchten.

image::omni-channel/online-shop/extras/assets/DE-Mandant-GlobEinstell-Sprachpakete-01-SI.png[]

__Bild 1: **Sprachpakete**; vorgegebene und eigene Texte__

[IMPORTANT]
.
====
Es gibt vier Textbereiche, an denen Sie Änderungen vornehmen können:
====

* Allgemeine Texte
* Hinweise
* Fehlermeldungen
* Warnungen

Sie können diese Änderungen für jedes zur Verfügung stehende Sprachpaket durchführen.

[NOTE]
.Texte eines Sprachpaketes bearbeiten:
====
Unter **Sprache** das gewünschte Sprachpaket auswählen.

Unter **Bereich** einen der vier Textbereiche wählen.

Klicken Sie auf die **Lupe**. +
→ Die Liste der **Texte** wird angezeigt.

Sie können nun in der Spalte **Eigene Übersetzung** Ihre Änderungen vornehmen (siehe Bild 1, Pfeil).

Klicken Sie auf **Speichern**, um die Änderungen zu sichern.
====
