== eKomi

Vor dem Einkauf möchten potentielle Käufer wissen, wie lange die Lieferzeit im Durchschnitt dauert und ob nach der Überweisung des Geldes die Ware überhaupt ausgeliefert wird. Unternehmen können dieses Vertrauen zwar mit Gütesiegeln und Auszeichnungen aufbauen, aber nichts ist so effizient wie das Vorhandensein glaubwürdiger Kundenmeinungen. link:http://www.ekomi.de[**eKomi**^] bringt die **5-Sterne-Bewertungen** direkt auf Webseiten und Suchmaschinen und trägt so maßgeblich zur Steigerung der Konversionsraten bei (Info: eKomi Webseite; www.ekomi.de).

[IMPORTANT]
.Bewertungen
====
Die **Bewertungen** werden **einmal täglich** abgerufen.
====

[WARNING]
.AGB der Marktplätze beachten
====
Beachten Sie die AGB des jeweiligen Marktplatzes, bevor Sie diese Funktion der Aufforderung zu einer Bewertung Ihres Webshops einrichten. Manche Marktplätze erlauben dies nur unter gewissen Voraussetzungen oder gar nicht. Sie sollten den jeweiligen Marktplatz dann unter der Option **Herkunft** im **Aktionsmanager** von der automatischen Erinnerungsfunktion ausschließen.
====

== Bei eKomi registrieren

Die Registrierung nehmen Sie über den Link **link:http://www.ekomi.de/de/register-enterprise-de[http://www.ekomi.de/de/register-enterprise-de^]** auf der Website von eKomi vor. Sie erhalten die **Zugangsdaten**, die Sie für die Einrichtung des Tools in plentymarkets benötigen.

== eKomi in plentymarkets einrichten

Mit den nachfolgend beschriebenen Einstellungen richten Sie **eKomi** in Ihrem plentymarkets-System ein.

=== Benutzer einrichten

Ihr plentymarkets-System wird über die SOAP-Schnittstelle mit **eKomi** verbunden. Für den Zugriff benötigt dieses Bewertungstool ein eigenes **<<basics/arbeiten-mit-plentymarkets/benutzer-verwalten#, Benutzerkonto>>** mit Zugriffsrechten auf die API in Ihrem System.

[.instruction]
Benutzer für eKomi einrichten:

. Öffnen Sie das Menü **System » Einstellungen » Benutzer**.
. Richten Sie ein neues Benutzerkonto der Benutzerklasse **Variabel** für eKomi ein.
. Aktivieren Sie für diesen Benutzer im Tab **Berechtigungen » API » Customer** den Call **GetCustomers** und im Tab **Order** den Call **SearchOrders**.
. Klicken Sie auf das **Speichern-Symbol**, um die Einstellungen zu sichern.

[WARNING]
.Calls
====
Aktivieren Sie **ausschließlich die genannten Calls** für diesen Benutzer.
====

=== Daten an eKomi senden

Schicken Sie eKomi anschließend eine E-Mail, die folgende Daten enthält:

**API URL** +
z.B.: http://www.example.de/plenty/api/soap/version114/?xml

**Benutzername des Benutzerkontos** +
z.B.: eKomi

**Benutzerpasswort des Benutzerkontos** +
z.B.: 123abc!:_ (Bitte unbedingt ein eigenes erstellen!)

an support@ekomi.de mit Ihrer eKomi- bzw. Kunden-ID in der Betreffzeile.

== Artikelbewertung konfigurieren

Um eKomi zu nutzen, ist es erforderlich, dass Sie das Menü **<<omni-channel/online-shop/extras/kunden-feedback#, Kunden-Feedback>>** konfigurieren. Fügen Sie Template-Variablen und Code in das Template **ItemViewSingleItem** ein. Wählen Sie für die Option **Maximale Bewertungspunkte** die Einstellung **5** (Sterne).
