== Allgemeines zum Gewinnspiel

Gewinnspiele in Form von Verlosungen, Quizspielen und Rätseln sind ein beliebtes Mittel zur Absatzsteigerung und werden beispielsweise verwendet, um Aufmerksamkeit auf ein bestimmtes Produkt zu lenken oder Kontaktdaten von Interessenten zu gewinnen. plentymarkets verfügt über ein Gewinnspiel-Modul, das individuell angepasst werden kann.

== Gewinnspiel planen

Überlegen Sie zuerst, was Sie mit Ihrem Gewinnspiel erreichen möchten. Ein solides Konzept spart Ihnen hinterher viel Zeit und Aufwand. Ziehen Sie z.B. folgende Fragen in Erwägung:

* Welche Zielgruppe möchten Sie ansprechen (z.B. Neukunden, Stammkunden)?
* Was möchten Sie mit Ihrem Gewinnspiel erreichen (z.B. für Top-Angebote werben, Kontaktdaten sammeln)?
* Wie oft sollen neue Gewinnspiele veröffentlicht werden (z.B. wöchentlich, monatlich)?
* Welche Sorte von Gewinnspiel möchten Sie nutzen (z.B. Verlosung ohne Fragen, Quiz, Rätselspiel)?
* Welche Fragen und Antworten sind am besten für Ihre Gewinnspiele geeignet?
* Welche rechtlichen Informationen sind für Ihr Gewinnspiel relevant? Beachten Sie auch den Eintrag "Rechtlicher Hinweis" in Tabelle 1 und fragen Sie ggf. bei Ihrem Rechtsberater nach.

== Gewinnspiel anlegen

Nachdem Sie ein gut durchdachtes Konzept für Ihr Gewinnspiel gefunden haben, nehmen Sie die entsprechenden Einstellungen in plentymarkets vor. Aus diesen Einstellungen wird das Anmeldeformular für das Gewinnspiel generiert.

image::omni-channel/online-shop/extras/assets/Mandant-GlobEinstell-Gewinnspiel-01.png[]

[.instruction]
Gewinnspiel anlegen:

. Öffnen Sie das Menü *System » Mandant » Global » Gewinnspiel*.
. Nehmen Sie die *Einstellungen* gemäß Tabelle 1 vor.
. Klicken Sie auf *Speichern*, um die Einstellungen zu sichern.

*Verlosung ohne Fragen*: Es muss mindestens eine Antwort gespeichert werden. Sonst erhalten Ihre Kunden eine Fehlermeldung, wenn sie sich für das Gewinnspiel im Webshop anmelden. Für eine reine Verlosung nur eine Antwortmöglichkeit hinterlegen, z.B. mit dem Text "Ich möchte teilnehmen."

=== Kategorie des Typs Content anlegen

Nachdem Sie die Gewinnspieleinstellungen vorgenommen haben, müssen Sie das __*Eingabeformular*__ in eine Kategorie einfügen. Hierzu müssen Sie eine neue Kategorie anlegen.

[.instruction]
Kategorie anlegen:

. Öffnen Sie das Menü <<artikel/kategorien-verwalten#, Artikel » Kategorien>>.
. Klicken Sie links unten auf das *grüne Plus*, um eine neue Kategorie anzulegen.
. Hinterlegen Sie den entsprechenden Text für diese Kategorie im Tab *Beschreibung 1*, z.B. Ihre Gewinnspielfrage.
. Fügen Sie den Platzhalter*{% Form(-2) %}*, z.B. direkt unterhalb Ihrer Gewinnspielfrage, ein.  +
→ Anstelle dieses Platzhalters wird im Webshop das Anmeldeformular zum Gewinnspiel angezeigt.
. Klicken Sie auf *Speichern*, um die Einstellungen zu sichern.

[WARNING]
.Formular-ID
====
Die ID für das Formular eines Gewinnspiels ist immer die *__-2__*.  +
Daher kann immer nur *__ein Gewinnspiel aktiv__* sein.
====

=== Kategorie ins Webshop-Layout einbinden

Nachdem Sie die Kategorie des Typ Contents angelegt haben, müssen Sie diese noch in Ihren Webshop einbinden, damit das Gewinnspiel sichtbar wird.

[.instruction]
Kategorie ins Layout einbinden:

. Öffnen Sie das Menü *CMS » Webdesign » Layout*.
. Wählen Sie das Layout, in das die Kategorie eingebunden werden soll.  +
→ Für das Beispiel wird *PageDesignContent* gewählt.
. Fügen Sie den Platzhalter *{% Link(x) %}* an der gewünschten Stelle im Layout ein (siehe nachfolgendes Codebeispiel sowie Bild 2).
. Klicken Sie auf *Speichern*, um die Einstellungen zu sichern.

[IMPORTANT]
.Kategorie-ID einfügen
====
Ersetzen Sie den Textplatzhalter *x* im Codebeispiel durch die *ID* der Kategorie.
====

== Bestätigung der Anmeldung

Nach der Anmeldung zum Gewinnspiel kann der Teilnehmer eine *__E-Mail als Bestätigung__* erhalten. Wenn dies gewünscht ist, müssen Sie eine entsprechende *E-Mail-Vorlage* anlegen und diese E-Mail-Vorlage bei *Automatischer E-Mail-Versand* auswählen.

[.instruction]
E-Mail-Vorlage zur Bestätigung anlegen:

. Öffnen Sie das Menü <<crm/e-mails-versenden#1200, System » Mandant » Mandant wählen » E-Mail » Vorlagen>>.
. Legen Sie eine entsprechende *E-Mail-Vorlage* an.
. Öffnen Sie das Menü <<crm/e-mails-versenden#2300, System » Mandant » Mandant wählen » E-Mail » Automatischer Versand>>.
. Wählen Sie die zuvor von Ihnen angelegte *E-Mail-Vorlage* aus dem Dropdown-Menü *Gewinnspielanmeldung*.
. Klicken Sie auf *Speichern*, um die Einstellungen zu sichern.

[NOTE]
.E-Mail-Zugangsdaten hinterlegen
====
Damit plentymarkets über Ihr E-Mail-Konto E-Mails für Sie versenden kann, müssen Sie Ihre <<crm/e-mails-versenden#100, E-Mail-Zugangsdaten>> vorher im System eintragen.
====

== Eintragung im Newsletter-E-Mail-Ordner

Bei einer Anmeldung können die Daten der Teilnehmer direkt in einen Newsletter-E-Mail-Ordner eingetragen werden. Der Vorteil ist, dass die E-Mail-Adressen und Namen der Teilnehmer im System gespeichert bleiben, auch nachdem Sie die Teilnehmer gelöscht haben (siehe weiter unten). Außerdem können Sie so die gewonnenen E-Mail-Adressen bequem in diesem Menü verwalten, z.B. Daten suchen oder exportieren, Newsletter schicken, usw. Die Daten der Teilnehmer können Sie in einen schon bestehenden oder in einen extra dafür angelegten E-Mail-Ordner speichern. Natürlich können Sie auch jederzeit die gewonnenen Teilnehmerdaten von einem Ordner in einen anderen verschieben. Gehen Sie wie nachfolgend beschrieben vor, um einen neuen E-Mail-Ordner für Ihre Gewinnspielteilnehmer anzulegen:

[.instruction]
E-Mail-Ordner anlegen:

. Öffnen Sie das Menü <<crm/newsletter-versenden#300, System » CMS » Newsletter » plentymarkets » E-Mail-Ordner>>.
. Tragen Sieunter *Neuen E-Mail-Ordner anlegen* eine *Bezeichnung* für den Ordner ein.
. Klicken Sie auf *Speichern*, um die Einstellungen zu sichern. +
→ Der E-Mail-Ordner wird angelegt und in der Übersicht angezeigt.
. Wählen Sie ggf. weitere Einstellungen für diesen E-Mail-Ordner aus.

Damit bei der Teilnahme am Gewinnspiel die Daten in den korrekten E-Mail-Ordner eingetragen werden, muss in der Kategorie, die die Gewinnspielanmeldung enthält, oberhalb von *[Form_x]* (*x* entspricht der ID des Formulars) ein spezieller Platzhalter eingefügt werden. Dadurch erhält das Anmeldeformular die Information, in welchen E-Mail-Ordner die Daten eingetragen werden sollen.

[.instruction]
Platzhalter einfügen:

. Öffnen Sie das Menü *Artikel » Kategorien*.
. Wählen Sie die Kategorie Ihres Gewinnspiels aus.
. Fügen Sie den *Platzhalter {EMailDirID(x)}* in der Kategorie direkt oberhalb von*{% Form(x) %}*(*x* entspricht der ID des Formulars) ein und ersetzen Sie *x* durch die ID des gewünschten E-Mail-Ordners (Beispiel: {% EMailDirID(6) %} für den E-Mail-Ordner mit der ID 6).
. Klicken Sie auf *Speichern*, um die Einstellungen zu sichern.

== Gewinner ermitteln

Im Menü *System » Mandant » Global » Gewinnspiel* können Sie im Bereich *Übersicht* jederzeit sehen, wie viele Teilnehmer an dem Gewinnspiel teilgenommen haben. Nachdem das Gewinnspiel beendet wurde, können Sie dort auch den Gewinner ermitteln.

[.instruction]
Gewinner ermitteln:

. Öffnen Sie das Menü *System » Mandant » Global » Gewinnspiel*.
. Klicken Sie auf *Gewinner ermitteln*. +
→ Ein neues Fenster wird geöffnet. Hier werden alle Teilnehmer des Gewinnspiels aufgelistet. Die Reihenfolge wird per Zufall bestimmt.

image::omni-channel/online-shop/extras/assets/Mandant-GlobEinstell-Gewinnspiel04-SI.png[]

__Bild 4: *Gewinner ermitteln*__

Eine E-Mail-Adresse kann nicht mehrmals für das Gewinnspiel verwendet werden. Das heißt, dass ein Kunde nur einmal mit seiner E-Mail-Adresse teilnehmen kann. Trägt ein Kunde eine schon verwendete E-Mail-Adresse ein, wird das Formular im Webshop nicht abgeschickt. Falls ein Kunde mehrere E-Mail-Adressen besitzt, wäre es allerdings möglich, mit jeder dieser E-Mail-Adressen teilzunehmen.

== Teilnehmer löschen

Wenn Sie z.B. alle Teilnehmer eines abgelaufenen Gewinnspiels löschen möchten, damit diese erneut an Gewinnspielen teilnehmen können oder weil Sie die Daten nicht mehr benötigen, gehen Sie wie beschreiben vor.

[.instruction]
Teilnehmer löschen:

. Klicken Sie auf *Löschen*, um die Teilnehmer eines Gewinnspiels zu löschen.
. Bestätigen Sie die Sicherheitsabfrage mit *OK*.  +
→ Die Werte in der Übersicht werden auf null zurückgesetzt. Danach können Sie ein komplett neues Gewinnspiel erstellen oder das gleiche Gewinnspiel erneut laufen lassen.

Nachdem Sie die Teilnehmer gelöscht haben, können Kunden schon eingetragene E-Mail-Adressen noch einmal verwenden, um das Formular abzuschicken.
