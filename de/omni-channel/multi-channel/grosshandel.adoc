link:http://www.grosshandel.eu/[grosshandel.eu^] ist ein reines B2B-Portal, d.h. sowohl Anbieter als auch Nachfrager müssen im Besitz eines gültigen Gewerbenachweises sein. Auf diesem Marktplatz können Sie reguläre Ware und Markenartikel kaufen und verkaufen (Quelle: grosshandel.eu).

[#100]
== Bei grosshandel.eu registrieren

Um Ihr plentymarkets System für grosshandel.eu einzustellen, link:http://www.grosshandel.eu/registrierung-verkaeufer.php[registrieren^] Sie sich zunächst im Händlerportal von grosshandel.eu.

[#200]
== Benutzerkonto erstellen

Ihr plentymarkets System wird über die SOAP-API mit grosshandel.eu verbunden. Für den Zugriff benötigt der Marktplatz ein eigenes Benutzerkonto mit Zugriffsrechten auf die API in Ihrem System. Erstellen Sie im Menü <<basics/arbeiten-mit-plentymarkets/benutzer-verwalten#10, System » Einstellungen » Benutzer » Konten>> ein neues Benutzerkonto der Benutzerklasse *API* für grosshandel.eu. Aktivieren Sie außerdem den Call *GetCategoryMappingForMarket*. Dieser Call sorgt dafür, dass der Marktplatz später Ihre Kategorieverknüpfungen auslesen kann.

[.instruction]
Benutzerkonto für grosshandel.eu erstellen:

. Öffnen Sie das Menü *System » Einstellungen » Benutzer » Konten*.
. Klicken Sie auf *Neu*. +
→ Das Fenster *Neuen Benutzer erstellen* wird geöffnet.
. Geben Sie die Daten ein.
. Wählen Sie als Benutzerklasse *API*.
. *Speichern* Sie die Einstellungen. +
→ Der Benutzer wird erstellt.
. Wählen Sie den Benutzer aus der Liste. +
→ Das Bearbeitungsfenster wird geöffnet.
. Öffnen Sie das Tab *Berechtigung » API*.
. Klappen Sie den Bereich *Item* auf.
. Aktivieren Sie den Call *GetCategoryMappingForMarket*.
. *Speichern* Sie die Einstellungen.

Weitere Informationen finden Sie auf der Handbuchseite <<basics/arbeiten-mit-plentymarkets/benutzer-verwalten#, Benutzer verwalten>>.

[#300]
== Daten bereitstellen

Für die Anbindung an Ihr plentymarkets System benötigt grosshandel.eu die folgenden Daten von Ihnen:

* URL der SOAP-API WSDL.  +
→ Sie finden die URL im Menü <<basics/datenaustausch/api-daten#, System » Einstellungen » API » Daten>> unter *SOAP WSDL*.
* Benutzername und Passwort für das von Ihnen erstellte Benutzerkonto für grosshandel.eu
* Ihre Login-Daten für das Händlerportal von grosshandel.eu

[#350]
== Auftragsherkunft aktivieren

Damit Sie Artikel, Merkmale etc. mit grosshandel.eu verknüpfen können, muss im Menü *System » Aufträge » Auftragsherkunft* die Auftragsherkunft grosshandel.eu aktiviert werden.

[.instruction]
Auftragsherkunft für grosshandel.eu aktivieren:

. Öffnen Sie das Menü *System » Aufträge » Auftragsherkunft*.
. Setzen Sie bei *grosshandel.eu* ein Häkchen.
. *Speichern* Sie die Einstellungen.

[#400]
== Artikelverfügbarkeit einstellen

Artikel, die Sie auf grosshandel.eu verkaufen möchten, müssen im Menü <<artikel/artikel-verwalten#170, Artikel » Artikel bearbeiten » Artikel öffnen » Tab: Varianten-ID>> im Tab *Verfügbarkeit* aktiviert werden.

[.instruction]
Artikelverfügbarkeit für grosshandel.eu einstellen:

. Öffnen Sie das Menü *Artikel » Artikel bearbeiten » Artikel öffnen » Tab: Varianten-ID » Tab: Einstellungen*.
. Aktivieren Sie die Hauptvariante im Bereich *Verfügbarkeit*.
. Wechseln Sie in das Tab *Verfügbarkeit*.
. Klicken Sie im Bereich *Märkte* in das Auswahlfeld.  +
→ Eine Liste mit allen verfügbaren Märkten wird angezeigt.
. Setzen Sie ein Häkchen bei *grosshandel.eu*.
. Setzen Sie ein Häkchen bei *Web-API*.
. Klicken Sie auf *Hinzufügen*. +
→ Der Marktplatz wird hinzugefügt.
. *Speichern* Sie die Einstellungen. +
→ Der Artikel ist auf grosshandel.eu verfügbar.

Die Verfügbarkeit für Varianten kann im Menü *Artikel » Artikel bearbeiten » Artikel öffnen » Tab: Varianten » Variante öffnen » Tab: Varianten-ID » Tab: Verfügbarkeit* individuell angepasst werden.

[#450]
==  SKU manuell festlegen

Im Menü <<artikel/artikel-verwalten#170, Artikel » Artikel bearbeiten » Artikel öffnen » Tab: Varianten-ID>> im Tab *Verfügbarkeit* fügen Sie, wenn nötig, marktplatzspezifische SKUs hinzu.

[.instruction]
SKU hinzufügen

. Öffnen Sie das Menü *Artikel » Artikel bearbeiten » Artikel öffnen » Tab: Varianten-ID » Tab: Einstellungen*.
.  Wechseln Sie in das Tab *Verfügbarkeit*.
.  Klicken Sie im Bereich *SKU* auf *Hinzufügen*. +
→ Das Bearbeitungsfenster *Neue SKU* wird angezeigt.
.  Wählen Sie die Herkunft *grosshandel.eu*.
.  Geben Sie die SKU ein.
.  Klicken Sie auf *Hinzufügen*. +
→ Die SKU wird gespeichert und angezeigt.

[#500]
== Verkaufspreis festlegen

Gehen Sie wie im Folgenden beschrieben vor, um für die Auftragsherkunft grosshandel.eu einen Verkaufspreis festzulegen. Dieser Preis wird auf grosshandel.eu angezeigt.

[.instruction]
Verkaufspreis für grosshandel.eu festlegen:

. Öffnen Sie das Menü *System » Artikel » Verkaufspreise » Verkaufspreis öffnen » Tab: Einstellungen*.
. Setzen Sie ein Häkchen bei der Herkunft *grosshandel.eu*.
. *Speichern* Sie die Einstellungen.

[#600]
== Kategorien verknüpfen

Verknüpfen Sie Ihre Webshop-Kategorien mit den Kategorien von grosshandel.eu, damit Ihre Artikel in diesen Kategorien angezeigt werden. Weitere Artikel der verknüpften Kategorien werden dann automatisch zugewiesen.

[.instruction]
Kategorien verknüpfen:

. Öffnen Sie das Menü *System » Märkte » grosshandel.eu*.
. Klappen Sie die Kategorie auf, die Sie mit einer grosshandel.eu-Kategorie verknüpfen möchten.
. Wählen Sie eine grosshandel.eu-Hauptkategorie aus der Dropdown-Liste. +
→ Ein Fenster mit Unterkategorien wird geöffnet.
. Wählen Sie aus den weiteren Unterkategorien die passende grosshandel.eu-Kategorie aus. +
→ Wenn die tiefste Kategorieebene erreicht ist, erscheint der Kategoriepfad im Textfeld *Kategorie*.
. Klicken Sie auf *Übernehmen*. +
→ Das Kategoriefenster wird geschlossen.
. *Speichern* Sie die Einstellungen.

[IMPORTANT]
.Verknüpfung bis in die unterste Kategorieebene
====
Verknüpfen Sie Ihre Kategorien immer bis in die unterste Ebene. Andernfalls ist eine Verknüpfung nicht möglich.
====

[#700]
== Unterkategorien mehrfach verknüpfen

Mit der Funktion *Auswahl für Unterkategorien übernehmen* haben Sie die Möglichkeit, für alle Unterkategorien einer Kategorie denselben grosshandel.eu-Kategoriepfad zu übernehmen. Denselben grosshandel.eu-Kategoriepfad für alle Unterkategorien einer Kategorie zu übernehmen ist sinnvoll, wenn die Unterkategorien die gleiche Verknüpfung erhalten sollen. Die einzelnen Unterkategorien können Sie bei Bedarf manuell über die Dropdown-Liste ändern.

[.instruction]
Unterkategorien mehrfach verknüpfen:

. Öffnen Sie das Menü *System » Märkte » grosshandel.eu*.
. Klappen Sie die Kategorie auf, für die Sie eine Kategorieverknüpfung erstellen möchten.
. Wählen Sie eine grosshandel.eu-Hauptkategorie aus der Dropdown-Liste in der Zeile *Auswahl für Unterkategorien übernehmen*. +
→ Ein Fenster mit Unterkategorien wird geöffnet.
. Wählen Sie aus den weiteren Unterkategorien den passenden Kategoriepfad aus. +
→ Wenn die tiefste Kategorieebene erreicht ist, erscheint der Kategoriepfad im Textfeld *Kategorie*.
. Klicken Sie auf *Übernehmen*. +
→ Das Kategoriefenster wird geschlossen.
. Klicken Sie auf *Alle auswählen* in der Zeile *Auswahl für Unterkategorien übernehmen*.
. Klicken Sie auf *Reload*. +
→ Der Kategoriepfad wird für alle Unterkategorien übernommen.
. *Speichern* Sie die Einstellungen.
