Mit dem Datenformat *Campaign* bearbeiten und exportieren Sie die im Menü <<auftraege/gutscheine#, Aufträge » Gutscheine>> erstellten Kampagnen. Sie können auch eine neue Kampagne erstellen.

Sie erstellen und exportieren das Datenformat im Menü *Daten » Dynamischer Export*. Wie Sie das Datenformat exportieren, ist auf der Handbuchseite <<daten/export-import/daten-exportieren#, Daten exportieren>> beschrieben.

Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.
