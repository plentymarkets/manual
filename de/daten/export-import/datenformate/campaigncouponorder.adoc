= CampaignCouponOrder
:lang: de
include::{includedir}/_header.adoc[]
:description: CampaignCouponOrder
:position: 50

Das Datenformat *CampaignCouponOrder* liefert Informationen über die Einlösung von Codes über die Aufträge. Ein Import ist für dieses Format nicht möglich, nur der Export.

Sie exportieren das Datenformat im Menü **Daten » Dynamischer Export**. Wie Sie das Datenformat exportieren, ist auf der Handbuchseite <<basics/datenaustausch/export-import/daten-exportieren#, Daten exportieren>> beschrieben.

Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.
