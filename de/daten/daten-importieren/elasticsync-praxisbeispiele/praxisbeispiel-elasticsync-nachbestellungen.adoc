In diesem Beitrag wird beschrieben, wie du Nachbestellungen importierst.

NOTE: Für dieses Best Practice solltest du mit der generellen Funktionsweise des Import-Tools vertraut sein. Allgemeine Infos dazu findest du auf der Handbuchseite <<daten/daten-importieren/ElasticSync#, Import-Tool nutzen>>.

Ausgangssituation
Vorbereitungen / CSV-Datei einrichten

[#importeinstellungen]
== Importeinstellungen wählen

<<daten/daten-importieren/ElasticSync#1210, Erstelle einen neuen Import>>. Wähle als *Typ* im Bereich *Grunddaten* aus der Dropdown-Liste *Aufträge* aus. Die weiteren Einstellungen können frei gewählt werden. Richte dich danach, wie du deine CSV-Datei gestaltet hast.
