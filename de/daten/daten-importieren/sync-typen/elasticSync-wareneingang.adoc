|*Artikel-ID*
|ID des Artikels

|*Varianten-ID*
|ID der Variante

|*Variantennummer*
|Variantennummer

|*Barcode*
|Barcode +
*_Hinweis_*: Stelle sicher, dass im Menü *Artikel » Artikel bearbeiten* im Tab *Einstellungen* der Code für den Barcode eingetragen ist. Außerdem muss im Abgleich des Syncs im rechten Abgleichfeld *Barcode* der Name des Barcodes (z.B. GTIN 13) ausgewählt sein.
|====
