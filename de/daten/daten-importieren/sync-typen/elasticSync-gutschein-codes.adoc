Mit einem Sync des Typs *Gutschein-Code* importierst du Daten zu Gutschein-Codes.

[#100]
== Felder für den Abgleich von Gutschein-Codes

Die folgenden Felder stehen für den Abgleich von Gutschein-Codes zur Verfügung. Anhand des gewählten Abgleichfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

* Kampagnen-ID

[#200]
== Felder für die Zuordnung von Gutschein-Codes

In <<tabelle-zuordnungsfelder-gutschein-codes>> werden Felder erläutert, die für den Datenabgleich von Gutschein-Codes zur Verfügung stehen. Anhand des gewählten Abgleichfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

[[tabelle-zuordnungsfelder-gutschein-codes]]
.Zuordnungsfelder für den Import von Gutschein-Codes
[cols="1,1,2"]
|====
|Zuordnungsfeld |Zulässiger Importwert |Erläuterung

|Kampagnen-ID
|Numerisch (interne ID)
|ID der Kampagne

|Gutschein-Code
|Alphanumerisch
|Gutschein-Code

|Rabattwert
|Numerisch
|Rabattwert des Gutscheins

|Anzahl der Einlösungen
|Numerisch
|Anzahl, wie oft der Gutschein bereits eingelöst wurde.

|Eingelöster Betrag
|Numerisch
|Eingelöster Betrag des Gutscheins. +
*_Wichtig:_* Beim Import wird nicht der Restwert des Gutscheins importiert, sondern der bereits eingelöste Betrag. +
*_Beispiel:_* Der Gutschein hat einen Wert über 100,00 EUR. Der Kunde löst 80,00 EUR ein. Somit beträgt der Restwert 20,00 EUR. Der Wert, der mit diesem Zuordnungsfeld importiert wird, beträgt 80,00 EUR.

|====
