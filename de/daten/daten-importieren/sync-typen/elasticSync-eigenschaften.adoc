Mit einem Sync des Typs *Eigenschaften* importierst du Eigenschaftsdaten.

[#10]
== Felder für den Abgleich von Eigenschaften

Die folgenden Felder stehen für den Abgleich von Eigenschaftendaten zur Verfügung. Anhand des gewählten Abgleichfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

* Eigenschafts-ID
* Eigenschaftsname
* ID der Eigenschaftsgruppe
* Name der Eigenschaftsgruppe
