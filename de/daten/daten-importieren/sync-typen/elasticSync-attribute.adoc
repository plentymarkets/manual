Mit einem Sync des Typs *Attribute* importieren Sie Attribute und Attributwerte. Die importieren Daten werden im Menü *System » Systemeinstellungen » Artikel » Attribute* gespeichert.

[#2490]
== Felder für den Attributabgleich

Die folgenden Felder stehen für den Abgleich von Attributen zur Verfügung. Anhand des gewählten Abgleichfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

* Attribut-ID
* ID des Attributwerts
* Attributname
* Name des Attributwertes
