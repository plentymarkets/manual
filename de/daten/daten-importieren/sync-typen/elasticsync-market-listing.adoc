Mit Syncs des Typs *Market-Listing* importieren Sie Listing-Daten in plentymarkets. Einen neuen Sync erstellen Sie im Menü *Daten » ElasticSync*.

[#100]
== Felder für den Market-Listing-Abgleich

Für den Market-Listing-Abgleich steht Ihnen das Abgleichfeld *Market-Listing-ID* zur Verfügung.

[#200]
== Felder für die Market-Listing-Zuordnung

In diesem Kapitel werden Felder erläutert, die für die Zuordnung von Market-Listings zur Verfügung stehen.

[#300]
=== Listing

Für den Import von Listings stehen die in <<listing-zuordnung>> aufgelisteten Felder zur Verfügung. Anhand des gewählten Abgleichsfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

[[listing-zuordnung]]
.Zuordnungsfelder für den Listingimport
[cols="1,1,2"]
|====
|Zuordnungsfeld |Zulässige Importwerte |Erläuterung

|Listing-Typ-ID
|*IDS EINGEBEN*
|Die ID des Listing-Typs.

|Bestandsabhängigkeits-ID
|*IDS EINGEBEN*
|Die ID der Bestandsabhängigkeit.

|Inhalts-ID
|*IDS EINGEBEN*
|Die ID des Listing-Inhalts.
|====

[#400]
=== Listing-Eigenschaft

Für den Import von Listing-Eigenschaften stehen die in <<listing-eigenschaft-zuordnung>> aufgelisteten Felder zur Verfügung. Wählen Sie einen Wert für die Listing-Eigenschaft aus der Dropdown-Liste, die erscheint, wenn Sie *Wert* zuordnen. +
