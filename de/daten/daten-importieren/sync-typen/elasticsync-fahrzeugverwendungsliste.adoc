Mit Syncs des Typs *Fahrzeugverwendungsliste* importieren oder aktualisieren Sie Fahrzeugverwendungslisten in plentymarkets. Einen neuen Sync erstellen Sie im Menü *Daten » ElasticSync*.

[#100]
== Felder für den Abgleich von Fahrzeugverwendungslisten

Für den Abgleich von Fahrzeugverwendunglisten steht Ihnen das Abgleichfeld *Fahrzeugverwendungslisten-ID* zur Verfügung.

[#200]
== Felder für die Zuordnung von Fahrzeugverwendungslisten

In diesem Kapitel werden Felder erläutert, die für die Zuordnung von Fahrzeugverwendungslisten zur Verfügung stehen.

[#300]
=== Fahrzeugverwendungsliste

Für den Import von Fahrzeugverwendungslisten stehen die in <<fahrzeugverwendungsliste-zuordnung>> aufgelisteten Felder zur Verfügung. Anhand des gewählten Abgleichsfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.
