== Felder für den Abgleich von Warenbestand

In <<tabelle-Abgleichfelder-Warenbestand>> werden Felder erläutert, die für den Abgleich von Warenbeständen zur Verfügung stehen. Anhand des gewählten Abgleichfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

[[tabelle-Abgleichfelder-Warenbestand]]
.Abgleichfelder für Warenbestand
[cols="1,3"]
|====
|Abgleichfeld |Erläuterung

|*VariantenID*
|ID der Variante

|*Barcode*
|Barcode +
*_Hinweis_*: Stelle sicher, dass im Menü *Artikel » Artikel bearbeiten* im Tab *Einstellungen* der Code für den Barcode eingetragen ist. Außerdem muss im Abgleich des Syncs im rechten Abgleichfeld *Barcode* der Name des Barcodes (z.B. GTIN 13) ausgewählt sein.

|*Variantennummer*
|Variantennummer

|*ArtikelID*
|ID des Artikels
|====
