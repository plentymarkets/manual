Mit einem Sync des Typs *Hersteller* importierst du Herstellerdaten.

[#3100]
== Felder für den Herstellerabgleich

Die folgenden Felder stehen für den Abgleich von Herstellerdaten zur Verfügung. Anhand des gewählten Abgleichfelds wird geprüft, ob der importierte Datensatz in plentymarkets vorhanden ist.

* Hersteller-ID
* Herstellername
* Externer Herstellername
