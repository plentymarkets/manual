link:https://www.dropbox.com/de/[Dropbox^] ist ein Cloud-Dienst und ermöglicht Ihnen, Daten zu speichern, mit anderen zu teilen oder gemeinsam in der Cloud zu bearbeiten. Auf in der Dropbox gespeicherte Dateien können Sie von jedem beliebigen Computer oder von mobilen Endgeräten aus zugreifen. Das Menü **System » Mandant » Global » Dienste » Dropbox** schafft durch die Verknüpfung mit Ihrem Dropbox-Konto die technischen Voraussetzungen dafür, Dateien zwischen plentymarkets und Ihrer Dropbox auszutauschen.

==  Funktionen in plentymarkets

Im ersten Schritt bietet Ihnen die Verknüpfung mit der Dropbox die Möglichkeit, Designs im Menü <<omni-channel/online-shop/webshop-einrichten/cms#webdesign-webdesign-bearbeiten, CMS » Webdesign>> zu exportieren und zu importieren. Beim Export werden die Dateien strukturiert in der Dropbox abgelegt und können dort komfortabel bearbeitet werden. Da es sich um Dateien im Textformat handelt, ist auch eine Bearbeitung über mobile Endgeräte möglich. Nach der Bearbeitung werden die Änderungen jedoch erst nach einem Import des Designs in Ihr plentymarkets-System wirksam. Die Voraussetzung dazu ist die Verknüpfung mit dem Dropbox-Konto. Gehen Sie dazu wie im Folgenden beschrieben vor.

Damit haben Sie auch die Möglichkeit, einem Dienstleister über die Dropbox Zugriff auf die Dateien Ihres Designs zu geben, ohne dass dieser Zugriff auf Ihr System erhält. In den Einstellungen Ihres Dropbox-Accounts können Sie Zugriffsrechte vergeben und finden Sie eine Dateihistorie, um bei Bedarf Dateien auf einen früheren Zustand zurückzusetzen.

==  plentymarkets mit Dropbox-Konto verknüpfen

Sie benötigen ein Dropbox-Konto, mit dem Sie plentymarkets verknüpfen. Sie können ein bestehendes Dropbox-Konto nutzen oder bei der Verknüpfung ein neues Konto anlegen. Gehen Sie wie unten beschrieben vor, um plentymarkets mit einem Dropbox-Konto zu verknüpfen.

[NOTE]
.Popups in den Browser-Einstellungen zulassen
====
Bei der Verknüpfung von plentymarkets mit Ihrem Dropbox-Konto wird eine neue Browser-Seite als Popup geöffnet. Wenn Sie einen Popup-Blocker nutzen, wird das Öffnen dieser Seite möglicherweise verhindert. Beachten Sie daher die Browser-Einstellungen und lassen Sie Popups für Ihr plentymarkets-Backend temporär zu, um die Verknüpfung herzustellen.
====


[.instruction]
plentymarkets mit einem Dropbox-Konto verknüpfen:

.  Öffnen Sie das Menü **System » Mandant » Global » Dienste » Dropbox**.
.  Klicken Sie auf **Code abrufen**. +
→ Ein neues Browser-Tab wird geöffnet.
.  Geben Sie in diesem Browser-Tab die Zugangsdaten für Ihr bestehendes Dropbox-Konto ein oder klicken Sie auf **Neu bei Dropbox? Konto erstellen**, wenn Sie noch kein Konto bei Dropbox haben.
.  Klicken Sie auf **Zulassen**, um zu bestätigen, dass plentymarkets auf die Daten und Ordner in Ihrer Dropbox zugreifen darf. +
→ Ein Code wird angezeigt.
.  Kopieren Sie den Code in die Zwischenablage.
.  Wechseln Sie zurück in Ihr plentymarkets-Backend.
.  Fügen Sie den Code im Popup-Fenster **Code speichern** in das Feld **Code** ein.
.  *Speichern* Sie die Einstellungen. +
→ Das Popup-Fenster wird geschlossen.
.  Ändern Sie bei Bedarf den Namen des **Basis-Ordners**, der in Ihrer Dropbox angelegt wird. Dieser Ordner wird später für Exporte/Importe verwendet.
.  *Speichern* Sie die Einstellungen. +
→ plentymarkets wird mit Ihrem Dropbox-Konto verknüpft. Ihr Dropbox-Konto wird im Feld **Verknüpftes Konto** angezeigt, wenn Sie das Menü erneut öffnen.

[IMPORTANT]
.Verknüpftes Konto noch nicht sichtbar
====
Wenn Sie die Verknüpfung gespeichert haben, aber neben der Option **Verknüpftes Konto** die E-Mail-Adresse des Dropbox-Accounts noch nicht angezeigt wird, laden Sie das Menü neu.
====

[NOTE]
.Verknüpfung mit Dropbox aufheben
====
Die Verknüpfung von plentymarkets mit Ihrer Dropbox heben Sie auf, indem Sie erneut auf **Code abrufen** klicken und im folgenden Fenster das Feld neben **Code** leer lassen und speichern. Laden Sie dann das Menü neu. Die Einstellungen unter **Verknüpftes Konto** und **Basis-Ordner** sind nun leer, die Verknüpfung ist aufgehoben. Wenn gewünscht können Sie nun eine neue Verknüpfung konfigurieren.
====
