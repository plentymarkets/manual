= accountContacts
:lang: de
include::{includedir}/_header.adoc[]
:keywords: accountContacts, report, raw data, rohdaten
:position: 10


Der Datenformat-Creator accountContacts exportiert die im System vorhandenen Kontakte (Kundendaten) gemäß der im Folgenden definierten Regeln und speichert diese im Cloud-Speicher ab.

== Spezifikationen

.Technische Spezifikationen
[cols=4]
|====
|Eigenschaft |Bezeichnung |Wert |Erläuterung

|dataName
|Name
|accountContacts
|Name des Creator-Prozesses und Prefix der erstellten Dateien.

|fileFormat
|Dateiformat
|CSV, GZIP
|Die Daten werden als als per GZIP (RFC 1952) komprimierte CSV Datei bereitgestellt.

|defaultActive
|Standardmäßig aktiv
|nein
|Dieser Vorgang läuft erst dann automatisch im Hintergrund, wenn zuvor im Menü *Daten » Rohdaten* der gleichnamige Creator aktiviert wurde.

|intervall
|Intervall
|täglich
|Einmal pro Tag, gewöhnlich über Nacht, werden alle seit dem letzten Durchlauf geänderten Daten exportiert.

|limitPage
|Zeilen pro Anfrage
|bis zu 3.000
|Die Anzahl der Datenzeilen, die in eine einzelne Datei maximal geschrieben werden.

|limitDay
|Zeilen pro Tag
|bis zu 80.0000
|Die Anzahl der Datenzeilen, die in an einem Tag maximal geschrieben werden.

|earliestPossibleDataDate
|Frühester Zeitpunkt
|01.01.2017
|Zeitpunkt ab dem diese Rohdaten erhoben werden können, gefiltert nach der Spalte in dataUpdatedAtColumnName.

|dataUpdatedAtColumnName
|Spaltenbezeichnung
|ac_updated_at
|Name der Spalte, welche den Zeitstempel der letzten Aktualisierung enthält.

|deleteRawDataAfterDaysMinimum
|Minimale Anzahl an Tagen bis zur Löschung
|20
|Die minimale Anzahl an Tagen bis die generierte Rohdaten-Datei wieder vom Cloud-Speicher entfernt wird, eine Weiterverarbeitung muss somit vorher erfolgen. Die genaue Dauer kann im Menü *Daten » Rohdaten* definiert werden.

|deleteRawDataAfterDaysMaximum
|Maximale Anzahl an Tagen bis zur Löschung
|80
|Die maximal Anzahl an Tagen bis die generierte Rohdaten-Datei wieder vom Cloud-Speicher entfernt wird, eine Weiterverarbeitung muss somit vorher erfolgen.

|====
