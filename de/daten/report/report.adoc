== Was ist ein Report?

Ein Report ist eine Datenanalyse, welche als Ergebnis eine einzelne Kennzahl (KPI) oder eine Ergebnisliste ermittelt. Als Datenbasis für diese Analyse werden regelmäßig Rohdaten aus plentymarkets exportiert und auf einem sicheren Cloud-Speicher abgelegt.

[TIP]
.Aktueller Stand
====
Zum aktuellen Zeitpunkt kann die Analyse lediglich über externe Business-Intelligence (BI) Tools erfolgen. Für das laufende Jahr ist auch ein internes Modul zur Datenanalyse geplant.
====

=== Rohdaten

Rohdaten werden im Hintergrund und im Regelfall nachts ermittelt. Welche Rohdaten abgelegt werden, wird im Menü *Daten » Rohdaten* definiert. Die generierten Dateien können per REST-API abgerufen und extern verarbeitet werden.

=== Für Endanwender

Wir beobachten, dass Geschäftsentscheidungen oft nicht auf Basis von Kennzahlen, sondern eher aus dem Bauch heraus getroffen werden. Um dies zu verändern, empfehlen wir den Einsatz eines Business-Intelligence-Tools. Bestenfalls werden darin unterschiedliche Daten aus mehreren Quellen analysiert, um daraus konkrete Handlungsempfehlungen ableiten zu können.

Die Daten aus plentymarkets können einfach in eine bestehende BI-Lösung integriert werden. Sofern noch kein Standard-Connector existiert, leiten Sie bitte die URL zu dieser Seite an den Lösungsanbieter weiter.

== Verfügbare Datenformate
