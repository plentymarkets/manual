[#1600]
== VariationEigenschaften

Für den Export von Eigenschaften stehen die in <<tabelle-felder-eigenschaften>> aufgelisteten Felder zur Verfügung.

[TIP]
.Warum sind die Felder beim Export leer? Warum geben sie mir unerwartete Daten aus?
====
Das Feld wird nur dann befüllt, wenn die Eigenschaft verknüpft ist.
Ist die Eigenschaft nicht verknüpft, bleibt das Feld leer.

Denk außerdem dran, plentymarkets mitzuteilen, über welche Eigenschaft du Informationen haben möchtest. Mit anderen Worten, _ordne eine Eigenschaft zu_. Klicke dazu auf *assignment* (icon:sign-in[role="darkGrey"]) und wähle die gewünschte Eigenschaft aus der Dropdown-Liste.

Eine Ausnahme bildet das Feld *all*. Hier brauchst du keine bestimmte Eigenschaft zuzuweisen, da du Informationen über _alle Eigenschaften_ erhältst.
====

[[tabelle-felder-eigenschaften]]
.Felder für Export von Eigenschaften
[cols="1,3"]
|====
|Feld |Erläuterung

|all
|Gibt alle Eigenschaften aus. Die Informationen werden wie folgt formatiert: +
`name:wert:sprache;name:wert:sprache`

|cast
a|Gibt den Typen der zugeordneten Eigenschaft aus, wenn diese verknüpft ist. Ist die zugeordnete Eigenschaft nicht verknüpft, ist das Feld leer.
