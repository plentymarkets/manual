Diese Seite beschreibt die Datenfelder und Filter, die dir beim Export von Herstellerdaten über das Katalog-Tool zur Verfügung stehen.
Diese Seite dient lediglich als Referenz und setzt voraus, dass du bereits Grundkenntnisse über das <<Katalog-Tool>> hast.

[TIP]
.Standardformat? Kein Export an Marktplätze
====
Beachte, dass diese Seite ein sogenanntes Standardformat beschreibt.
Das heißt ein Format, mit dem du Daten exportieren, aber nicht an einen Marktplatz übertragen kannst.
====

== Filter

Um nur bestimmte Daten zu exportieren, kannst du Filter wählen und somit die Daten eingrenzen. Wie du Filter hinzufügst, erfährst du in <<daten/daten-exportieren/kataloge-verwalten#257, diesem Kapitel>>.
In der folgenden Tabelle findest du alle für das Export-Format *Hersteller* verfügbaren Filter sowie eine Erläuterung zur Funktion des jeweiligen Filters.
