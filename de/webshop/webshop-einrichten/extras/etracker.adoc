==  Allgemeines über etracker

link:http://www.etracker.de[etracker^] bietet Tools zur Web-Analyse in unterschiedlichen Leistungsumfängen zu verschiedenen Preisen. Die Pakete können Sie testen, bevor Sie sich für die Nutzung entscheiden.

==  etracker in plentymarkets einrichten

Nachdem Sie eine Lizenz von etracker erworben haben, richten Sie das Analyse-Tool in plentymarkets wie im Folgenden beschrieben ein.

[.instruction]
Basis-Code aus etracker kopieren:

. Öffnen Sie im Administrationsbereich von *etracker* im Ordner *Konfiguration* den Menüpunkt *Setup/HTML-Code*.
. Öffnen Sie oben rechts den Menüpunkt *Basic Code*.
. Kopieren Sie den kompletten Quelltext unterhalb von *Basic Code*, der mit der Zeile *-- etracker PARAMETER 2.4 --* beginnt.
. Fügen Sie den Code wie unten beschrieben in plentymarkets ein.

.Basic Code von etracker einfügen
image::omni-channel/online-shop/webshop-einrichten/extras/assets/DE-Webshop-Conversion-etracker-01.png[]

[.instruction]
etracker in plentymarkets einrichten:

. Öffnen Sie das Menü *CMS » Webdesign » Ordner: Layout*.
. Öffnen Sie den Ordner *PageDesign* und dort das Template *PageDesignContent*.
. Wechseln Sie in das Tab *PageDesignContentMainFrame*.
. Fügen Sie den *Basic Code* am Ende des Textfeldes ein.
. *Speichern* Sie die Einstellungen. +
*→* Der Code ist aktiv und etracker beginnt, Daten zu sammeln.
