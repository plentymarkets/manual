== FINDOLOGIC - Suche und Abverkaufssteuerung

Die intelligente Suchtechnologie von FINDOLOGIC sorgt mit dafür, dass deine Kunden die gesuchten Produkte finden, schafft dabei echte Einkaufserlebnisse und macht Shopbesucher zu zufriedenen Kunden!

=== Features

Erfahre mehr über die Features der FINDOLOGIC-Suche auf unserer link:https://www.plentymarkets.eu/produkt/online-shop/webshop-erweiterungen/findologic/a-1769/[Partnerseite^].

=== Suchqualität

Die Suchfunktion sorgt dafür, dass die Artikel gefunden werden, die der User wirklich sucht. Dabei sorgt eine intelligente Technologie dafür, dass du keine Gewichtungen oder anderes mehr ändern musst! D.h. ein Verschlimmbessern durch ständiges manuelles abändern ist ausgeschlossen – eines der größten Probleme herkömmlicher Suchmöglichkeiten. +
Auch lernt die FINDOLOGIC-Suche stetig durch das Nutzverhalten dazu und reiht automatisch die relevanten Artikel höher. Dabei "lernt" die FINDOLOGIC-Suche die Begriffe der Nutzerinnen und Nutzer zu verstehen und baut einen eigenen Wortschatz hierfür auf.

== Integration der Such-Software FINDOLOGIC in plentymarkets

Um die FINDOLOGIC-Suche auch in deinem Webshop nutzen zu können, bestellst du diese zunächst über das Online-Bestellformular bei FINDOLOGIC: +
link:http://www.findologic.com/pricing[FINDOLOGIC-Software bestellen^].

=== Benutzer für FINDOLOGIC einrichten

Dein plentymarkets-System wird über die *SOAP-Schnittstelle* mit FINDOLOGIC verbunden. Für den Zugriff wird ein eigenes Benutzerkonto mit Zugriffsrechten auf die API in deinem System benötigt.

[NOTE]
.API-Benutzerkonto einrichten
====
Richte unter *System » Systemeinstellungen » Einstellungen » Benutzer* ein neues Benutzerkonto der Benutzerklasse *API* für FINDOLOGIC ein.

Beachte dazu die Handbuchseite link:https://www.plentymarkets.eu/handbuch/einstellungen/grundeinstellungen/benutzer/benutzer-anlegen/[Benutzer anlegen].
====

=== Informationen für FINDOLOGIC

Folgende Informationen benötigt FINDOLOGIC von dir für die Anbindung an dein System:

* *URL der SOAP API WSDL* +
Diese URL ist im Menü *System » Systemeinstellungen » Einstellungen » API » Daten* unter *SOAP WSDL* zu finden. Die WSDL hat folgenden Aufbau: *http://WWW.SHOPURL.COM/plenty/api/soap/version1xx/?xml.*(*xx* = jeweils aktuelle SOAP-Version)

[NOTE]
.WSDL einfügen
====
*Kopiere* deine *SOAP WSDL* und füge diese im FINDOLOGIC-Kundenaccount im Feld *Export URL* ein.

Achte darauf, dass anstelle des obigen Eintrags *SHOPURL.COM* die *URL deines Webshops* in der WSDL steht.
====

* Die *Logindaten* für den Benutzer (siehe Kapitel 2.1) +
Den *Benutzernamen* und das *Passwort*. [NOTE]

.Benutzerdaten eintragen
====
Trage den *Benutzernamen* und das gewählte *Passwort* des API-Benutzers im FINDOLOGIC-Kundenaccount im Feld *Export-ID* bzw. *Export-Passwort* ein.
====

== Design des Suchergebnisses

Es stehen dir für das Design des Suchergebnisse die folgenden Optionen zur Wahl.

=== Option "FINDOLOGIC"

Bei dieser Einstellung wird das Design des Suchergebnisses im Template-Manager von FINDOLOGIC eingestellt. Das Template heißt *plenty.tpl*.

Beim FINDOLOGIC-Design fehlen bestimmte Eigenschaften, die demgegenüber vom plentymarkets-Design bereitgestellt werden, z. B. *Verfügbarkeit* (s. Kapitel 2.3).

=== Option "Plenty"

Mit der Einstellung *Plenty* wird das Suchergebnis vom plentymarkets-System generiert.

[TIP]
.Empfehlung
====
Stelle das Design des Suchergebnisses auf *Plenty*, anstatt auf FINDOLOGIC. Anderenfalls werden in der Suche lediglich die Artikel-IDs angezeigt, anstatt der korrekten Artikel-Darstellung. +
Besonders wichtig ist auch, dass du am Ende den *Slash "/"* nicht vergisst, da die Suche sonst nicht korrekt in deinen Shop eingebunden wird.
====

Die Einstellungen nimmst du im Menü *CMS » Webdesign » Layout » ItemView » ItemViewSearchResultsList* vor (<<bild-code-suchergebnis>>).

== FINDOLOGIC-Navigation

Die *FINDOLOGIC-Navigation* ermöglicht eine genauere und gezieltere Suche eines gewünschten Artikels. Die Integration der Navigation erfordert nur wenige Schritte, die nachfolgend beschrieben werden.

=== Navigation integrieren

Die Navigation wird im Template *PageDesignContent* über die Template-Funktion *{% FindologicFilterContainer() %}* eingebunden.

Die Funktion des Selectors steuert das Template *ItemViewThirdPartySearchResultsList*.

[.instruction]
Navigation einbinden:

. Öffne das Menü *CMS » Webdesign » Layout » ItemView » ItemViewThirdPartySearchResultsList*.
. Füge die *Template-Funktion {% FindologicFilterContainer() %}* an einer geeigneten Stelle in das Template ein, z.B. in der Seitenleiste oder unter der Kategorienavigation.
. *Speichere* die Einstellungen.
. Füge anschließend für die Darstellung der Ergebnisse der Navigation folgenden Code in das Template *ItemViewThirdPartySearchResultsList* ein.
