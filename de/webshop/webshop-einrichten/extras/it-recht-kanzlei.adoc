== Benutzerkonto einrichten

Dein plentymarkets System wird über die SOAP-Schnittstelle mit der Dienstleistung der IT-Recht Kanzlei verbunden, d.h., du benötigst das SOAP-Modul. Für den Zugriff benötigt dieser Dienst einen Benutzer mit Zugriffsrechten auf die API deines Systems.

[.instruction]
Neuen Benutzer anlegen:

. Öffne das Menü *Einrichtung » Einstellungen » Benutzer » Konten*.
. Klicke auf *Neu*. +
→ Das Bearbeitungsfenster *Neuen Benutzer anlegen* wird angezeigt.
. Trage den *Benutzernamen*, den *realen Namen* und das *Passwort* ein.
. Wiederhole das Passwort.
. Trage die E-Mail-Adresse des Benutzers ein, damit der Benutzer Benachrichtigungen des Systems empfängt.
. Wähle die Benutzerklasse *API* oder *Variabel*.
. *Speichere* (icon:save[role="green"]) die Einstellungen.

Folgende Calls müssen für den Benutzer freigegeben werden:

* GetServerTime
* GetPlentyMarketsVersion
* GetMultiShops
* GetTermsAndCancellation
* SetTermsAndCancellation
* GetLegalInformation
* SetLegalInformation

Beachte dazu die Handbuchseite <<omni-channel/online-shop/webshop-einrichten/extras/it-recht-kanzlei#4, Einführung SOAP-API>>.

== Rechtstexte automatisch übertragen

Damit Rechtstexte automatisch von der IT-Recht-Kanzlei übertragen werden, musst du in deinem Konto bei der IT-Recht Kanzlei eine Datenschnittstelle einrichten. Folgende Informationen benötigt die IT-Recht Kanzlei von dir für die Anbindung an dein System:

* Login-Daten, Benutzername und Passwort für den Benutzer
* URL der SOAP API WSDL +
Die URL findest du im Menü *Einrichtung » Einstellungen » API » Daten* unter <<basics/datenaustausch/api-daten#, SOAP WSDL>>.
