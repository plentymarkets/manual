== picalike

picalike ist ein Tool für den Vergleich von Bildern. Diese Technologie unterstützt Ihre Kunden bei der Suche nach passenden Artikeln. +
Produktbilder werden im E-Commerce immer wichtiger. Mit picalike erweitern Sie Ihre *Merkmal-Suchfunktion*, also die Filterung nach Preis, Geschlecht, Kategorien etc. um eine weitere Dimension, die *Ähnlichkeit*, und vermitteln Ihren Besuchern damit ein neues Shoppingerlebnis in Ihrem Webshop. +
picalike analysiert Ihre Artikelbilder und baut einen Index aus Ähnlichkeiten zwischen den Bildern auf. Wenn ein Besucher Ihres Webshops nach Ähnlichkeit filtert, wird picalike informiert, welches Bild als Referenz genutzt werden soll und zeigt dann ähnliche Bilder sortiert an.

== Bei picalike anmelden

Nehmen Sie Kontakt zu picalike auf und registrieren Sie sich für die Bildersuche. Beachten Sie dabei die folgenden Punkte:

* Sie finden auf der picalike-Website detaillierte Informationen zu *Funktion* und *Kosten*. Klären Sie diese Punkte, bevor Sie picalike in plentymarkets aktivieren.
* picalike benötigt zur Bildersuche die *Export-URL* Ihres plentymarkets-Systems. Sie finden die URL im Menü *System » Systemeinstellungen » Mandant » Global » Dienste » picalike* (<<bild-einstellungen-picalike>>).
* Sie benötigen für die Aktivierung den *API-KEY* von picalike. Sie finden diesen Key im Menü *Einstellungen* Ihres registrierten Bereichs bei picalike (<<bild-api-key-aktivierung>>).

== picalike in plentymarkets aktivieren

Gehen Sie wie nachfolgend beschrieben vor, um die picalike-Bildersuche in plentymarkets zu aktivieren. Es sind zwei Konfigurationsschritte erläutert:

[.instruction]
Einstellungen für picalike konfigurieren:

. Öffnen Sie das Menü *System » Systemeinstellungen » Mandant » Global » Dienste » picalike*.
. Nehmen Sie die *Einstellungen* gemäß der nachfolgenden Tabelle vor.
. *Speichern* Sie die Einstellungen.

[[bild-einstellungen-picalike]]
.*Einstellungen* für picalike
image::omni-channel/online-shop/webshop-einrichten/extras/assets/DE-Mandant-GlobEinstell-ExtDienste-picalike-02-SI.png[]

Erläuterungen zu den Optionen im Menü *picalike*:

.*Einstellungen* für picalike
[cols="1,3"]
|====
|Einstellung |Erläuterung

|*picalike aktivieren*
|Wählen Sie *Ja*, wenn Sie picalike aktivieren und *Nein*, wenn Sie die Bildersuche (wieder) deaktivieren möchten.

|*API-Key*
|Kopieren Sie den *API-Key* aus dem Menü *Einstellungen* Ihres picalike-Kontos (<<bild-api-key-aktivierung>>) und fügen Sie den Key in dieser Zeile ein.

|*Export-URL*
|Die *Export-URL* benötigt picalike, um die Bildersuche zu aktivieren. Diese URL wird vom System erzeugt und sollte bereits in dieser Zeile stehen, wenn Sie das Menü erstmalig öffnen. +
*Wichtig*: Stellen Sie sicher, dass Sie picalike die Export-URL übermittelt haben, andernfalls ist die Bildersuche nicht möglich.

|*Suchgenauigkeit*
|Sie können wählen, in welchen *Kategorie-Bereichen* gesucht werden soll.
|====

[.instruction]
picalike-Bildersuche in Template einbauen:

. Öffnen Sie das Menü *CMS » Webdesign » Layout » ItemView » ItemViewSingleItem*.
. Fügen Sie folgenden Code an einer geeigneten Stelle ein: +
*&lt;a id="link_picalike_$ID" href="{% Link_PicalikeSearch(1) %}" rel="nofollow"&gt;Ähnliche Bilder suchen&lt;/a&gt;* +
→ Der Text *Ähnliche Bilder suchen* wird im Webshop als Link angezeigt. Sie können den Text durch einen beliebigen eigenen Text austauschen.
. *Speichern* Sie die Einstellungen.

== Bildersuche mit mehreren Bildern einbauen

Für eine Auswahl aus mehreren Artikelbildern muss für jedes Bild ein „onClick“-Ereignis hinzugefügt werden.

[IMPORTANT]
.onClick-Befehl
====
Der onClick-Befehl ist wie folgt aufgebaut: +
*onClick="document.getElementById('link_picalike_$ID').href='[Link_PicalikeSearch_5]';" +
⇒ 5* = Artikelbild Nr. 5
====

onClick-Ereignisse werden jeweils durch ein Semikolon getrennt.

Jedes Bild in der Artikelansicht erfordert einen individuellen Link, wobei die Links eine fortlaufende Nummerierung erhalten. +
LinkTo_PicalikeSearch_2 +
LinkTo_PicalikeSearch_3 +
LinkTo_PicalikeSearch_4 +
etc.

Für das zweite Artikelbild lautet der Code somit:

*onClick="document.getElementById('link_picalike_$ID').href='{% Link_PicalikeSearch(2) %}';"*

[.instruction]
picalike-Bildersuche mit mehreren Bildern in Template einbauen:

. Öffnen Sie das Menü *CMS » Webdesign » Layout » ItemView » ItemViewSingleItem*.
. Fügen Sie den jeweiligen Code, wie in <<bild-code-weitere-artikelbilder>> gezeigt, für die weiteren Artikelbilder ein.
. *Speichern* Sie die Einstellungen.

Auf <<bild-code-weitere-artikelbilder>> sehen Sie, wo der Code für Artikelbild 2 eingefügt wird (Pfeil): vor der spitzen Klammer.

Verfahren Sie mit weiteren Bildern nach dem gleichen Muster.
