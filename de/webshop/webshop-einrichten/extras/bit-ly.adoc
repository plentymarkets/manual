== Allgemeines

Auf der Website des Online-Dienstes *bit.ly* hast du die Möglichkeit, lange URLs kürzen zu lassen. Hast du eine gültige Konfiguration für den Kurz-URL-Dienst bit.ly hinterlegt, werden z.B. die URLs, die plentymarkets automatisch für die Tweets von <<omni-channel/online-shop/webshop-einrichten/extras/twitter#, Twitter>> generiert (Kapitel 4), direkt gekürzt.

== Registrierung

link:https://bitly.com/a/sign_in[Registriere^] dich bei bit.ly, sofern du noch keinen bit.ly-Account besitzt.

== Konfiguration

Gehe wie nachfolgend beschrieben vor, um bit.ly in plentymarkets einzurichten.

[.instruction]
bit.ly konfigurieren:

. Öffne das Menü *Einrichtung » Mandant » Global » Dienste » bit.ly*.
. Hinterlege unter *Login* den Benutzernamen.
. Hinterlege unter *API-Key* den API Key, wie von bit.ly erhalten. +
→ Wenn du bei bit.ly eingeloggt bist, findest du den API-Key unter folgendem link:http://bit.ly/a/your_api_key[Link^].
. *Speichere* (icon:save[role="green"]) die Einstellungen.

.bit.ly-Zugangsdaten hinterlegen

== Anwendungsbeispiel

Der Kurz-URL-Dienst wird beispielsweise für automatische Tweets verwendet, die mit dem <<omni-channel/online-shop/webshop-einrichten/extras/live-shopping#100, Aktionsmanager>> in plentymarkets konfiguriert werden. Dabei werden die URLs der Kategorieseiten, z.B. für die Aktion *Live-Shopping twittern*, automatisch gekürzt. Damit stehen dann mehr Zeichen für die Nachricht zur Verfügung.
