== Allgemeines

Auf der Website des Online-Dienstes *bit.ly* haben Sie die Möglichkeit, lange URLs kürzen zu lassen. Haben Sie eine gültige Konfiguration für den Kurz-URL-Dienst bit.ly hinterlegt, werden z.B. die URLs, die plentymarkets automatisch für die Tweets von <<omni-channel/online-shop/webshop-einrichten/extras/twitter#, Twitter>> generiert (Kapitel 4), direkt gekürzt.

== Registrierung

link:https://bitly.com/a/sign_in[Registrieren^] Sie sich bei bit.ly, sofern Sie noch keinen bit.ly-Account besitzen.

== Konfiguration

Gehen Sie wie nachfolgend beschrieben vor, um bit.ly in plentymarkets einzurichten.

[.instruction]
bit.ly konfigurieren:

. Öffnen Sie das Menü *System » Systemeinstellungen » Mandant » Global » Dienste » bit.ly*.
. Hinterlegen Sie unter *Login* den Benutzernamen.
. Hinterlegen Sie unter *API-Key* den API Key, wie von bit.ly erhalten. +
→ Wenn Sie bei bit.ly eingeloggt sind, finden Sie den API-Key unter folgendem link:http://bit.ly/a/your_api_key[Link^].
. *Speichern* Sie die Einstellungen.

== Anwendungsbeispiel

Der Kurz-URL-Dienst wird beispielsweise für automatische Tweets verwendet, die mit dem <<omni-channel/online-shop/webshop-einrichten/extras/live-shopping#100, Aktionsmanager>> in plentymarkets konfiguriert werden. Dabei werden die URLs der Kategorieseiten, z.B. für die Aktion *Live-Shopping twittern*, automatisch gekürzt. Damit stehen dann mehr Zeichen für die Nachricht zur Verfügung.
