Mithilfe der Adressprüfung durch die Software AddressDoctor können Sie die Adressen Ihrer Kunden schon direkt bei der Eingabe prüfen und ggf. korrigieren lassen oder wenn der Kunde seine Daten speichert.

== Bei AddressDoctor registrieren

Um AddressDoctor in plentymarkets nutzen zu können, müssen Sie sich zunächst registrieren. Weitere Informationen zur Anmeldung finden Sie auf dieser **link:https://www.informatica.com/de/addressdoctor.html/[Webseite^]**.

== Zugangsdaten eingeben

Im Menü **System » Systemeinstellungen » Mandant » Mandant wählen » Dienste » Addressdoctor » Zugangsdaten** aktivieren Sie die Schnittstelle und geben die Zugangsdaten, die Sie von AddressDoctor erhalten haben, ein. Damit eine Adressprüfung durchgeführt wird, müssen Sie anschließend weitere Einstellungen im Menü **System » Systemeinstellungen » Mandant » Mandant wählen » Dienste » Addressdoctor » Adressprüfung** vornehmen.

[.instruction]
Zugangsdaten eingeben:

. Öffnen Sie das Menü **System » Systemeinstellungen » Mandant » Mandant wählen » Dienste » Addressdoctor » Zugangsdaten**.
. Nehmen Sie die Einstellungen vor. Beachten Sie die Erläuterungen in Tabelle 1.

== Adressprüfung einrichten

Mithilfe der Adressprüfung können Sie Adressen Ihrer Kunden schon direkt bei der Eingabe prüfen lassen oder wenn der Kunde seine Daten speichert. Damit die Prüfung durchgeführt werden kann, müssen Sie die Schnittstelle zu AddressDoctor einrichten und aktivieren. Nähere Informationen über AddressDoctor und zu den Preisen für Adressprüfungen erhalten Sie von **link:https://www.informatica.com/de/addressdoctor.html/[AddressDoctor^]**.

Im Menü **System » Systemeinstellungen » Mandant » Mandant wählen » Dienste » Adressdoctor » Adressprüfung** stellen Sie ein, welche Adresse geprüft wird und wann die Prüfung erfolgt. In diesem Menü stehen Ihnen vier verschiedenen Tabs zur Verfügung: **Kundenregistrierung**, **Rechnungsanschrift**, **Lieferanschrift** und **Kundenstammdaten**. Wechseln Sie in eines der Tabs. Nehmen Sie dort die Einstellungen, wie im Folgenden beschrieben, vor.

[.instruction]
Adressprüfung einrichten:

. Öffnen Sie das Menü **System » Systemeinstellungen » Mandant » Mandant wählen » Dienste » Addressdoctor » Adressprüfung**.
. Wechseln Sie in eines der oben genannten Tabs.
. Nehmen Sie die Einstellungen vor. Beachten Sie dazu die Erläuterungen in Tabelle 2.
. **Speichern** Sie die Einstellungen. +
→ Anschließend können Sie die Prüfung für weitere Adressen einrichten. Gehen Sie dabei gleichermaßen vor.

.Adressprüfung im Menü **System » Systemeinstellungen » Mandant » Mandant wählen » Dienste » Adressdoctor » Adressprüfung** einrichten
[cols="1,3"]
|====
|Einstellung |Erläuterung

|**Adressprüfung bei Speicherung**
|Dienstleister aus der Dropdown-Liste wählen, mit dem die Adressprüfung durchgeführt werden soll. Aktuell besteht nur eine Schnittstelle zu AddressDoctor. Mit dieser Einstellung wird die Adresse des Kunden geprüft, wenn er seine Daten speichert.

|**Adressprüfung bei Eingabe**
|Dienstleister aus der Dropdown-Liste wählen, mit dem die Adressprüfung durchgeführt werden soll. Aktuell besteht zunächst eine Schnittstelle zu AddressDoctor. Mit dieser Einstellung wird die Adresse des Kunden bereits beim Eintippen der Daten geprüft. +
**__Wichtig__**: Nur eine der beiden Optionen wählen: **Adressprüfung bei Eingabe** oder **Adressprüfung bei Speicherung**.

|**Adressprüfung erzwingen**
|Zeitraum in Tagen eingeben, nach dem die Adressprüfung wiederholt werden soll. Es erfolgt dann automatisch nach dem eingegebenen Zeitraum eine erneute Prüfung, auch wenn die Adresse jeweils nicht erneut gespeichert oder neu eingegeben wurde. +
**__Tipp__**: Zeitraum möglichst so wählen, dass eine Prüfung erfolgt, bevor der erste Versand erfolgt. So werden ggf. falsche Angaben oder Änderungen rechtzeitig erkannt und können noch korrigiert werden.
|====

== Code in CSSContent einfügen

Wenn Sie die Funktionen des AddressDoctor nutzen möchten, benötigen Sie ein entsprechendes Design für die im Webshop anzuzeigenden Funktionen. Sie können das folgende CSS-Beispiel verwenden, sollten aber entsprechend der Konfiguration Ihres Webshops ggf. noch Anpassungen vornehmen. Mit dem folgenden Beispiel erhalten Sie jedoch eine gute Basis.

[.instruction]
Code in CSSContent einfügen:

. Öffnen Sie das Menü **CMS » Webdesign**.
. Öffnen Sie den Ordner **Layout » CSS » CSSContent**. +
→ Wenn Sie keinen Inhalt im Ordner **CSSContent** gespeichert haben, befindet sich dieser Ordner unter **Layout » CSS » Unbenutzt**.
. Fügen Sie den Code im Anschluss an den vorhandenen Code ein.
. **Speichern** Sie die Einstellungen.

[cols=""]
|====
|
[source,xml]
----
/* ADDRESS SUGGESTION LIST */

/* container */
.PlentyWebAddressSelectContainer {
z-index: 10000;
background-color: #fff;
border: 1px solid #ccc;

-moz-border-radius: 5px;
-webkit-border-radius: 5px;
border-radius: 5px;

-moz-box-shadow: 0 1px 5px 0 rgba(0, 0, 0, 0.3);
-webkit-box-shadow: 0 1px 5px 0 rgba(0, 0, 0, 0.3);
box-shadow: 0 1px 5px 0 rgba(0, 0, 0, 0.3);
}

/* ul */
.PlentyWebAddressSelectList {
margin: 0;
padding: 0;
}
/* li */
#PlentyWebAddressSelectMainContainer > .PlentyWebAddressSelectContainer > ul.PlentyWebAddressSelectList > li.PlentyWebAddressSelectRow {
margin: 0;
padding: 0 5px;
list-style-type: none;
cursor: pointer;
border-top: 1px solid #f0f0f0;
}
/* li first */
.PlentyWebAddressSelectList > li:first-child {
border-top: none;

-moz-border-radius: 5px 5px 0 0;
-webkit-border-radius: 5px 5px 0 0;
border-radius: 5px 5px 0 0;
}
/* li last */
.PlentyWebAddressSelectList > li:last-child {
-moz-border-radius: 0 0 5px 5px;
-webkit-border-radius: 0 0 5px 5px;
border-radius: 0 0 5px 5px;
}
/* li hover */
.PlentyWebAddressSelectList > li:hover {
background-color: #eee;
}

/* address data */
.PlentyWebAddressSelectStreet {
}
.PlentyWebAddressSelectHouseNo {
padding-left: 5px;
}
.PlentyWebAddressSelectHouseNo:after {
content: ",";
}
.PlentyWebAddressSelectZip {
padding-left: 5px;
}
.PlentyWebAddressSelectCity {
padding-left: 5px;
}

----
