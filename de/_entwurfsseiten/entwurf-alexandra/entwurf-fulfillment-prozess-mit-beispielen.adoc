Fulfillment beschreibt die einzelnen Schritte der Auftragsabwicklung wie das Erzeugen von Rechnungen oder den Druck von Versandlabels. Diese Handbuchseite beschreibt eine mögliche Konfiguration der wichtigsten Untermenüs im Menü **Aufträge » Fulfillment**. Das Ziel ist, Ihnen zu zeigen, wie Sie Ihren Fulfillment-Prozess möglichst effektiv gestalten. Dadurch gewährleisten Sie einen schnellen und optimalen Service, der im Online-Handel enorm wichtig ist und u.a. dazu führt, dass aus Gelegenheitskäufern in Ihrem Webshop Stammkunden werden.

Diese Handbuchseite umfasst die Schritte, die ab der Freigabe zum Warenversand bis zum tatsächlichen Warenausgang nötig sind. Die folgende Beschreibung ist als Beispiel zu verstehen, wie Sie den kompletten Vorgang sinnvoll konfigurieren und einfach und schnell abarbeiten können.

Es wird auch beschrieben, wie Sie automatisierte Statuswechsel einstellen, die dann beim Durchlauf des Fulfillment-Prozesses automatisch ausgeführt werden. Erfasst werden Aufträge ab dem Status [5]. Bei diesen Aufträgen wurde bereits ein Zahlungseingang gebucht (Wechsel von Statusbereich [3] in [4]) und die Ware ist verfügbar (Wechsel von Statusbereich [4] in [5]).

Sie haben alternativ die Möglichkeit, diese Schritte weiter zu automatisieren, indem Sie für bestimmte Schritte **<<basics/automatisierung/ereignisaktionen#, Ereignisaktionen>>** einrichten. Diese Handbuchseite enthält Beispiele dazu. Weiterhin können Sie das Fulfillment über einen **<<auftragsabwicklung/fulfillment/versand-vorbereiten#250, Versanddienstleister>>** oder einen **<<basics/automatisierung/prozesse/prozesse-einrichten#, Prozess>>** automatisieren.

== Vorbereitende Einstellungen vornehmen

Damit Ihre Fulfillment-Prozesse reibungslos ablaufen, müssen Sie die nachfolgend beschriebenen Einstellungen vornehmen bzw. prüfen und anschließend ggf. anpassen.

=== Auftragsstatus anlegen

Legen Sie im Menü **System » Aufträge » Auftragsstatus** für jeden Schritt einen Unterstatus an, in den Aufträge dann geändert werden. In Ihrem plentymarkets System sind bereits einige Statuseinträge standardmäßig voreingetragen. Sie können die Bezeichnungen ändern und die Liste durch weitere Stautseinträge erweitern. Weitere Informationen zu den Statuseinträgen finden Sie auf der Handbuchseite <<auftragsabwicklung/auftraege/auftraege-verwalten#, **Auftragsstatus**>>.

[.instruction]
Auftragsstatus anlegen:

. Öffnen Sie das Menü **System » Aufträge » Auftragsstatus**.
. Wählen Sie im Dropdown-Menü oben links den Shop, in dem Sie den Auftragsstatus anlegen möchten.
. Wählen Sie im mittleren Dropdown-Menü die Sprache, für die Sie den Auftragsstatus anlegen möchten.
. Geben Sie den Namen in der Spalte **Bezeichnung** beim passenden **Status** ein.
. **Speichern** Sie die Einstellungen.

.Beispiel für Statuswechsel von **Freigabe Versand** in **Rechnung erzeugt**

Außerdem ist das die Voraussetzung dafür, dass Aufträge eindeutig von Ihrem plentymarkets System für den nächsten Schritt erfasst werden können. Die Pickliste enthält die Informationen der Artikel, die Sie für den jeweiligen Fulfillment-Prozess benötigen mit Angabe der Menge sowie des Lagerortes, damit die Artikel im Lager unmittelbar gefunden werden. Die nach der Pickliste im Lager entnommenen Artikel werden dann zentral, z.B. am Packtisch, auftragsweise gesammelt und verpackt.

== Lieferanschriftenliste

Wenn Sie Pakete über einen Versanddienstleister versenden, benötigen Sie eine Lieferanschriftenliste. Es handelt sich dabei um einen Export mit Sendungsinformationen, den Sie dem Versanddienstleister anschließend bereitstellen, so dass dieser Ihnen wiederum gültige Versandlabels für Ihre Pakete zur Verfügung stellen kann.

In diesem Menü exportieren Sie die Lieferanschriften Ihrer Aufträge in eine Datei, die Sie für die Software Ihres Versanddienstleisters benötigen.

Bereiten Sie die im Folgenden beschriebene Ereignisaktion vor, diese greift automatisch nach den in Kapitel 7 und 8 durchzuführenden Schritten.

. Klicken Sie auf **Ereignisaktion hinzufügen**. +
→ Das Fenster **Neue Ereignisaktion erstellen** wird geöffnet.
. Geben Sie einen Namen ein.

== Adressetiketten

Im Gegensatz zu Versandetiketten können Sie für **link::/[Adressetiketten]** in plentymarkets genau festlegen, welche Informationen an welcher Stelle enthalten sein sollen. Da Sie Adressetiketten nicht vorgefertigt von einem Versanddienstleister erhalten, ist hier weder das Porto bereits abgerechnet, noch eine Sendungsnummer enthalten. Die Verwendung von Adressetiketten eignet sich besonders für **Warensendungen**, z.B. den Versand von Büchern, die Sie durch den individuellen Aufdruck **Büchersendung** u.U. günstiger versenden können als über das Gewicht des Pakets. Sie können in plentymarkets die benötigten Adressetiketten konfigurieren, eine PDF-Datei erzeugen und mit dieser dann Etiketten ausdrucken.

[NOTE]

== Paketnummern importieren

Mit einem Paketnummernimport per CSV-Datei im Menü link::/[**Aufträge » Fulfillment » Tab: Import Paketnummern**] können Sie auch größere Mengen an Paketnummern komfortabel in Ihrem plentymarkets System bearbeiten. Außerdem ist über eine Ereignisaktion eine Information des Empfängers und über eine Tracking-URL eine Sendungsverfolgung im Internet möglich.
