==== plentymarkets App mit plentyBase verbinden

Das Plugin plentyBase bietet eine Schnittstelle, über die Sie auf die Drucker eines Computers zugreifen.

* Das Plugin plentyBase muss auf dem lokalen Rechner installiert sein.
* In der plentymarkets App müssen die Verbindungsdaten des Rechners gespeichert werden, auf dem plentyBase installiert ist.
* Der Rechner, auf dem plentyBase installiert ist, muss erreichbar sein.

===== plentyBase Daten in plentymarkets App speichern

Halten Sie die folgenden Daten bereit, um die plentyBase Verbindungsdaten in der plentymarkets App zu speichern:

* IP-Adresse des Rechners, auf dem plentyBase installiert ist
* Server-Port, der plentyBase auf dem Rechner zugewiesen wurde

Nachdem Sie die plentyBase Verbindungsdaten in der plentymarkets App gespeichert haben, können Sie:

* In der App Dokumente über alle Drucker ausdrucken, auf die über den Rechner zugegriffen werden kann, auf dem plentyBase installiert ist.
* Plugins in der App nutzen, die plentyBase benötigen, z.B. Pick-by-light oder POS-Kauf auf Rechnung

[TIP]
.Bon-Drucker
====
Der Bon-Drucker wird über die IP-Adresse an die plentymarkets App angeschlossen. Daher ist plentyBase für einen Bon-Drucker nicht erforderlich.
====


[.instruction]
plentyBase Verbindungsdaten in plentymarkets App speichern:

. Melden Sie sich bei der plentymarkets App an.
. Tippen Sie in der Menüleiste oben links auf das *Menüsymbol*.
. Tippen Sie auf *Einstellungen*.
. Tippen Sie auf *plentyBase*. +
→ Das Einstellungsmenü *plentyBase* wird angezeigt.
. Nehmen Sie die Einstellungen vor. Beachten Sie <<tabelle-App-plentyBase>>.
. *Speichern* Sie die Einstellungen.

|Den Port eingeben, der plentyBase auf dem Rechner zugeordnet wurde. Standardmäßig ist der Port 7331 angegeben. Den Port Auf das plentyBase Icon in der Taskleiste des Rechners und dann auf *Konfiguration öffnen* klicken, um den Server-Port anzuzeigen.

|*Verbindung testen*

|*Access-Token*
|Access-Token eingeben, der in plentyBase für den Rechner gespeichert ist.
*_Tipp:_* Der Access-Token wird im Menü *Start » plentyBase » Ansicht: Einstellungen* des plentymarkets Backend als QR-Code dargestellt. Um den QR-Code einzuscannen, oben rechts in der App auf das QR-Code-Symbol klicken und den QR-Code scannen.

|*Access-Token generieren*

*_Hinweis:_* Wenn ein Access-Token in der plentymarkets App generiert wird, muss dieses Token auch in den Einstellungen des plentyBase Tools gespeichert werden.
