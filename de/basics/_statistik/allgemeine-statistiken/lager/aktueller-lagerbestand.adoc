== Aktueller Lagerbestand

Zeigt den __**aktuellen Lagerbestand**__ nach __**Kategorien**__ gesplittet an. Es wird jeweils der __**Warenwert**__ und die __**Artikelanzahl**__ des __**verfügbaren**__ und des __**reservierten Bestandes**__ angezeigt.

== Statistik-Details

Entsprechend des gewählten __**Lagers**__ wird der __**Warenwert pro Kategorie**__ und der __**Warenbestand**__ angezeigt. Sind keine Warenbestände für eine Kategorie vorhanden, wird kein Balken angezeigt. Als Tooltip wird der __**Verfügbare Warenwert**__, der __**verfügbare Warenbestand**__, der __**reservierte Warenbestand**__ sowie der __**reservierte Warenwert**__ angezeigt, jeweils in einer anderen Farbe (Bild 1). Sie können mit dieser Statistik z.B. erkennen, welche verfügbaren und reservierten Warenwerte und Warenbestände im jeweiligen Lager oder insgesamt vorhanden sind.

In Bild 1 wird z.B. für die Kategorie __**Body &amp; Catsuit**__ ein __**verfügbarer Warenwert**__ von __**1.048,01 Euro**__ angezeigt sowie ein __**Verfügbarer Warenbestand**__ von __**41 Artikeln**__. Als __**reservierter Warenbestand**__ werden __**8 Artikel**__ angezeigt und als __**reservierter Warenwert**__ ein Betrag von __**204,49 Euro**__.

[NOTE]
.Details anzeigen (Tooltip)
====
Zeigen Sie mit dem __**Mauszeiger**__ auf ein Element der Statistik, um die jeweiligen __**Details**__ anzuzeigen.
====

== Einstellungen

Legen Sie zunächst wie auf der Handbuchseite **<<basics/statistik#, Statistik>>** beschrieben eine neue Statistik an. Für die Statistik **Aktueller Lagerbestand** stehen dann die in Tabelle 1 beschriebenen __**Einstellungen**__ zur Verfügung.

image::basics/statistik/allgemeine-statistiken/lager/assets/DE-Statistik-Lagerbestand-02-SI.png[]

__Bild 2: Statistik____**Aktueller Lagerbestand**____; **Einstellungen**__

Die folgende Tabelle erläutert die Einstellungen im Menü **Start » Statistik → Statistik-Typ: Lager / Aktueller Lagerbestand**:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|**Gestapeltes-Säulendiagramm mit Linien** = Darstellung der Werte als Säulen. Die Warenwerte sind als **Säulen** dargestellt, die Bestände als **Linien**. +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine __**CSV-Datei**__ exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Es wird ebenfalls eine __**CSV-Datei**__ erzeugt. Die Daten werden allerdings detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Die __**Bezeichnung**__ der Statistik, die beim Erstellen gewählt wurde. Der Name kann hier geändert werden.

|**Lager**
|Das __**Lager**__ wählen, das für die Statistik berücksichtigt werden soll. **Alle** wählen, um alle Lager einzubeziehen.

|**Preisauswahl**
|**Durchschnittlicher EK** = Der vom System berechnete __**durchschnittliche Einkaufspreis**__ wird berücksichtigt. +
**Aktueller EK** = Der jeweils __**aktuelle Einkaufspreis**__, der in den Artikeldaten hinterlegt ist, wird für die Statistik herangezogen. +
**Durchschnittlicher EK, wenn 0, dann aktuellen EK verwenden** = Wenn __**kein durchschnittlicher EK hinterlegt**__ ist, wird automatisch der __**aktuelle EK**__ berücksichtigt. +
**Verkaufspreis** = Der Preis, der bei der Option **Preis** des jeweiligen Artikels hinterlegt ist.
|====

__Tab. 1: **Statistik bearbeiten**; **Einstellungen**__

== Daten-Export

Wie in Tabelle 1 erläutert können Sie zwischen einem einfachen und einem detaillierten Exportergebnis wählen. Nachfolgend finden Sie entsprechende Beispiele.

=== Export der Diagrammdaten

[IMPORTANT]
.Datenfelder
====
**CategoryName** = Kategoriebezeichnung, **NetStock** = Verfügbarer Warenbestand, **PhysicalStock** = Warenbestand gesamt, **ReservedStock** = Reservierter Warenbestand, **ValueOfPhysicalStock** = Wert des Warenbestands, **ValueOfNetStock** = Wert des verfügbaren Warenbestands, **ValueOfReservedStock** = Wert des reservierten Warenbestands
====

=== Export der Rohdaten

[IMPORTANT]
.Datenfelder
====
**ItemID** = Artikel-ID, **PriceSetID** = ID des Preissets des Artikels, **AttributeValueSetID** = ID der Artikelvariante, **CategoryID** = Kategorie-ID, **WarehouseID** = Lager-ID, **PhysicalStock** = Warenbestand gesamt, **NetStock** = Verfügbarer Warenbestand, **ReservedStock** = Reservierter Warenbestand, **AveragePurchasePrice** = Durchschnittlicher EK, **ItemPurchasePrice** = EK, **VariationPurchasePrice** = EK der Artikelvariante, **ItemSellingPrice** = VK
