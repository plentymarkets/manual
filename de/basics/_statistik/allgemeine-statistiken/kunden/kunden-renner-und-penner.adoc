== Kunden Renner und Penner

Diese Statistik zeigt die Umsatzdaten sortiert nach den **Kunden-Rennern** - das sind die Kunden, die den höchsten Umsatz bringen - oder **Kunden-Pennern** - das sind die Kunden, die am wenigsten Umsatz generieren - innerhalb der gewählten **Zeitspanne**. Standardeinstellung der Zeitspanne: **Die letzten 12 Monate**.

== Statistik-Details

Entsprechend der gewählten **Zeitspanne** und der Auswahl für die **Auswertung** werden die Anzahl der Aufträge sowie der Umsatz des Kunden angezeigt. An oberster Stelle erscheinen die **Kunden-Renner** - das sind Kunden, mit denen Sie den **höchsten Umsatz** generiert haben bzw. **Kunden-Penner -**also Kunden, die den **geringsten Umsatz** generiert haben, angezeigt. Die Kunden werden jeweils absteigend angezeigt, d.h. beginnend mit dem Kunden mit den höchsten bzw. niedrigsten **Umsatz**. Zum Schluss der Renner-Statistik sehen Sie den Kunden, der unter den Rennern am wenigsten Umsatz generiert hat. Bei der Penner-Statistik ist es umgekehrt, d.h. Sie sehen zuerst den Kunden mit dem geringsten Umsatz, dann absteigend die Kunden mit immer mehr Umsatz. Sie können mit dieser Statistik erkennen, welche Kunden viel oder besonders wenig Umsatz generieren und so z.B. besondere Aktionen für bestimmte Kunden anbieten.

|**Diagrammtyp**

**Umsatz** = Beträge inkl. Umsatzsteuer (brutto) +

|Die Daten des Diagramms werden in eine **CSV-Datei** exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen der **Daten-Tabelle**.

|**Rohdaten-Export**
|Beim Rohdaten-Export wird ebenfalls eine **CSV-Datei** erzeugt. Dabei werden detailliertere Daten exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Umsatz** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Hier wird der **Name** bzw. die **Bezeichnung** der Statistik angezeigt, die beim Erstellen einer neuen Statistik gewählt wurde. Sie können diesen Namen hier ändern.

|**Zeitspanne**
|Die **Zeitspanne** ist für die Statistiktypen jeweils voreingestellt, die Standardwerte können Sie Tabelle 1 der Handbuchseite <<basics/statistik#, Statistik>> entnehmen. Es stehen feste Werte zur Verfügung, die per Dropdown-Menü auswählbar sind. Darüber hinaus bietet dieser Statistiktyp die Möglichkeit, eine Zeitspanne in Form eines **Datumbereichs** einzutragen. +
**Verändern** Sie die **Zeitspanne**, wenn keine Daten angezeigt werden oder nur wenige Daten und klicken Sie auf das Speichern-Symbol, um das Ergebnis zu sehen. Sie können durch Wahl eines **längeren Zeitraums** mehr Informationen anzeigen. Wenn viele Daten vorhanden sind, bewirkt die **Verringerung der Zeitspanne** ggf. eine aussagekräftigere Auswertung.

|**Startdatum ... Enddatum**
|Hier können Sie optional einen bestimmten **Datumsbereich** alternativ zur Option **Zeitspanne** eingrenzen.

|**Mandant (Shop)**
|Wählen Sie die **Webshops** bzw. **Mandanten**, deren Daten für die Statistik verwendet werden sollen.

|**Auftragsherkunft**
|Sie können hier eine oder mehrere **Auftragsherkünfte** oder mit der Option **Alle** sämtliche verfügbaren auswählen. +
Die **Auftragsherkünfte** beziehen sich auf den **Auftrag**, nicht auf die Auftragspositionen.

|**Kunden ohne Aufträge berücksichtigen**
|Setzen Sie ein Häkchen, wenn in der Tabelle auch Kunden ohne Aufträge in dem gewählten Zeitraum berücksichtigt werden sollen.

|**Auswertung**
|Wählen Sie die jeweilige Option, um entweder die **Renner** oder die **Penner** in der Statistik anzuzeigen.
