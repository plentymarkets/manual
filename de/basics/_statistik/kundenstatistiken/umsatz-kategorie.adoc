== Kategorieumsätze

Hier wird der **Umsatz (Brutto) pro Kategorie** gezeigt sowie die **Menge der Artikel**, die jeweils in einem definierten Zeitraum von dem betreffenden Kunden gekauft wurden. Standardeinstellung der **Zeitspanne**: die **letzten 30 Tage**.

== Statistik-Details

Entsprechend der gewählten **Zeitspanne** wird der **Umsatz (Brutto) pro Kategorie** gezeigt sowie die **Anzahl der Artikel** des Kunden datumsbezogen angezeigt. Sind keine Umsätze vorhanden, wird die jeweilige Kategorie nicht angezeigt. Als **Tooltip** wird die **Anzahl der verkauften Artikel** und der zugehörige gesamte **Umsatz** angezeigt, die der Kunde mit der jeweiligen Anzahl an verkauften Artikeln erzielt hat (siehe Bild 1). Sie können mit dieser Statistik z.B. erkennen, wieviel Umsatz der Kunde in den verschiedenen Kategorien erzielt hat oder auch durch Wahl bzw. Änderung der Zeitspanne, in welchen Zeiträumen der Kunde wieviel Umsatz mit welchen Kategorien generiert hat.

[WARNING]
.**Ausschlüsse**
====
**Artikel**, die [.underline]#nicht mit einer Kategorie verknüpft# sind und **Gutscheine** werden bei der Auswertung dieser Statistik nicht berücksichtigt.
====

[NOTE]
.**Details anzeigen (Tooltip)**
====
Zeigen Sie mit dem **Mauszeiger** auf ein Element der Statistik, um die jeweiligen **Details** anzuzeigen.
====

[TIP]
.Tipp
====
In Bild 1 wird z.B. für die Kategorie **Foto** ein **Umsatz** von **11244,42 Euro** angezeigt, der mit **46 Artikeln** generiert wurde.
====

== Einstellungen

Die nachfolgend beschriebenen **Einstellungen** stehen für diese Statistik zur Verfügung.

Beachten Sie die Erläuterungen in der folgenden Tabelle:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|In diesem Dropdown-Menü wählen Sie die Art der Darstellung aus. +
**Balken- und Liniendiagramm** = Darstellung der Werte als Balken bzw. Linie +
**3D_Kreisdiagramm** = Darstellung der Werte als dreidimensionale Kreissektoren +
**2D-Kreisdiagramm** = Darstellung der Werte als flächige Kreissektoren +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine **CSV-Datei** exportiert. Dabei werden die Parameter wie angezeigt verwendet, sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Beim Rohdaten-Export wird ebenfalls eine **CSV-Datei** erzeugt. Dabei werden die Daten aber detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|**Anzeige der Werte**
|Klicken Sie links unten auf die Werte, um die **Anzeige** jeweils im Diagramm zu **deaktivieren** bzw. zu **aktivieren**. So können Sie nur jeweils die Daten separat anzeigen, die Sie sehen möchten und andere ausblenden. Dies kann bei komplizierten Daten eine Hilfe sein, die einzelnen Werte besser zu erkennen.

|
|Einstellungen

|**Name**
|Hier wird der **Name** bzw. die **Bezeichnung** der Statistik angezeigt, die beim Erstellen einer neuen Statistik gewählt wurde. Sie können diesen Namen hier ändern.

|**Zeitspanne**
|Die **Zeitspanne** ist für die Statistiktypen jeweils voreingestellt, die Standardwerte können Sie Tabelle 1 der Handbuchseite <<basics/statistik#, **Statistik**>> entnehmen. Es stehen feste Werte zur Verfügung, die per Dropdown-Menü auswählbar sind. Darüber hinaus bieten manche Statistiktypen die Möglichkeit, eine Zeitspanne in Form eines **Datumbereichs** einzutragen. +
**Verändern** Sie die **Zeitspanne**, wenn keine Daten angezeigt werden oder nur wenige Daten und klicken Sie auf das Speichern-Symbol, um das Ergebnis zu sehen. Sie können durch Wahl eines **längeren Zeitraums** mehr Informationen anzeigen. Wenn viele Daten vorhanden sind, bewirkt die **Verringerung der Zeitspanne** ggf. eine aussagekräftigere Auswertung.

|**Startdatum ... Enddatum**
|Hier können Sie optional einen bestimmten **Datumsbereich** alternativ zur Option **Zeitspanne** eingrenzen.

|**Datumsgrundlage**
|Wählen Sie welches **Datum**, z.B. das des Auftragseingangs, die **Grundlage** für den **Auswertungszeitraum** bilden soll.

|**Mandant (Shop)**
|Wählen Sie die **Webshops** bzw. **Mandanten**, deren Daten für die Statistik verwendet werden sollen.

|**Artikelherkunft**
|Sie können hier eine oder mehrere **Artikelherkünfte** oder mit der Option **Alle** sämtliche verfügbaren auswählen.

|**Unterkategorien anzeigen für**
|Wählen Sie die Kategorie, für die Sie die **Unterkategorien** auswerten möchten. Die jeweiligen Unterkategorien werden dann in der Statistik anstelle der Hauptkategorien angezeigt.

|**Zahlungsart**
|**Wählen** Sie die **Zahlungsart**, die für die Statistik berücksichtigt werden soll oder aktivieren Sie die Option **Alle**, um alle Zahlungsarten einzubeziehen.

|**Versandprofil**
|**Markieren** Sie die **Versandprofile**, die für die Statistik berücksichtigt werden sollen oder aktivieren Sie die Option **Alle**, um alle Versandprofile einzubeziehen.

|**Nur bezahlte Aufträge**
|Markieren Sie die Option, um **nur bezahlte Aufträge** in die Statistik einzubeziehen.

|**Nur Aufträge mit gebuchtem Warenausgang**
|Markieren Sie die Option, um **nur Aufträge mit gebuchtem Warenausgang** in die Statistik einzubeziehen.

|**Diagramm**
|Wählen Sie die Option für die **Anzeige der Beträge** in der Statistik: **Brutto** oder **Netto**.
|====
