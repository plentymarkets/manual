== Was ist die DSGVO?

Am 25. Mai 2018 trat die link:https://www.bfdi.bund.de/SharedDocs/Publikationen/Infobroschueren/INFO6.pdf%3F__blob%3DpublicationFile%26v%3D24[Datenschutzgrundverordnung^] offiziell und europaweit geltend in Kraft und löste somit nationale Regelungen ab.

Die DSGVO gilt für alle Unternehmen, die in der EU tätig sind und personenbezogene Daten wie Namen, Bankdaten und Adressdaten verarbeiten. Daher müssen Unternehmen ihre Prozesse, bei denen personenbezogene Daten verarbeitet werden, DSGVO-konform gestalten und dokumentieren.

== Vertragsanlage zur Auftragsverarbeitung i.S.d. Art. 28 Abs. 3 Datenschutzgrundverordnung (DSGVO)

Am Donnerstag den 24. Mai 2018 wird ein Update von plentymarkets 7 veröffentlicht, um den Vertragsabschluss zur Auftragsverarbeitung i.S.d. Art. 28 Abs. 3 der DSGVO digital abwickeln zu können.

Der Vertragsabschluss erfolgt digital im Bereich *Verträge*. Den Menüpunkt finden Sie als Admin-Benutzer im Terra-Backend von plentymarkets 7. Unter dem ersten Menüpunkt (rotes plenty P) befindet sich dann der neue Eintrag *Verträge*.
Klicken Sie in diesem Bereich den Vertrag "Vertragsanlage zur Auftragsverarbeitung (DS-GVO) Mai 2018" an. Auf der Folgeseite können Sie das Vertragsdokument herunterladen, indem Sie auf das Icon in der Spalte *Dokument* klicken. Senden Sie darauf bitte das Formular ab, dadurch ist der Vertrag geschlossen. Das Schöne an diesem Verfahren ist, dass der Vertragsschluss auf dem Postweg damit entfällt. Das spart uns und Ihnen Zeit und Kosten.

Sobald der Vertrag geschlossen wurde, können Sie das Vertragsdokument inklusive des Namens des Unterzeichners und des Datums der Vertragsunterzeichnung herunterladen. Dazu öffnen Sie erneut den Bereich *Verträge* und klicken auf *Vertragsdokument signiert herunterladen*.

Bitte legen Sie dieses Dokument aus Gründen der Dokumentation in Ihren Unterlagen ab.

Weitere Informationen hierzu finden Sie im link:https://forum.plentymarkets.com/t/vertragsanlage-zur-auftragsverarbeitung-i-s-d-art-28-abs-3-datenschutz-grundverordnung-dsgvo/483954[im Forum^].

== Änderungshistorie für Kontaktdaten

Im Menü *Daten » Historie* können Sie über die neue Änderungshistorie sehen, welcher Benutzer – beispielsweise ein Mitarbeiter oder der Kunde selbst über den Webshop – wann welche Daten geändert hat. Geändert werden können sind Daten wie Adressdaten, Kontaktoptionen, Adressoptionen, Firmendaten und Bankdaten.

Weitere Informationen finden Sie im link:https://forum.plentymarkets.com/t/dsgvo-aenderungshistorie-fuer-kontaktdaten-integriert/486705[Forum^].

[#700]
== Übertragung von E-Mail-Adresse und Telefonnummer des Kunden an Versanddienstleister

Bitte beachten Sie, dass plentymarkets als Software laut DSGVO den Grundsatz von _Privacy by Default_ befolgen muss und wir daher pauschal die Einstellung setzen, dass diese Daten (E-Mail und Telefonnummer) des Kunden nicht an die Versanddienstleister übertragen werden. Das bedeutet, Sie können selbst wählen, ob die Daten Ihrer Kunden übertragen werden oder ob Sie einen alternativen Wert übertragen möchten. Natürlich können Sie auch unsere voreingestellte Auswahl so belassen.

Weitere Informationen hierzu finden Sie im link:https://forum.plentymarkets.com/t/dsgvo-uebertragung-von-e-mail-adresse-und-telefonnummer-an-versanddienstleister/486536[Forum^].

[#800]
== Alle Daten zu einem Kontakt herunterladen

Im Menü *CRM » Schnellsuche* ist es möglich, alle zu einem Kontakt gespeicherten Daten und Dokumente per Klick auf einen Button als ZIP-Datei herunterzuladen. Die ZIP-Datei enthält eine HTML-Datei, die Sie in Ihrem Browser öffnen und über die Sie auch die gespeicherten Dokumente herunterladen können. Außerdem ist es möglich, die Daten in einem maschinenlesbaren Format zu speichern.

[#900]
== Daten automatisch bereinigen

Im neuen Menü *System » Einstellungen » Daten » Bereinigung* können Sie festlegen, nach welchem Zeitraum bestimmte Datensätze automatisch aus Ihrem plentymarkets System gelöscht werden. Sukzessive werden weitere Bereiche in diesem Menü ergänzt.

Sie können auch Tickets aus Ihrem Ticketsystem automatisch löschen. Diese Einstellung nehmen Sie jedoch nicht im Menü *System » Einstellungen » Daten » Bereinigung* vor, sondern über den Aktionsmanager im Ticketsystem im Menü *System » CRM » Ticketsystem » Aktionen*. Weitere Informationen über das automatische Löschen von Tickets finden Sie auf der Handbuchseite <<crm/ticketsystem-nutzen#3000, Ticketsystem nutzen>>.

[#1000]
== Daten manuell anonymisieren

Im Menü *CRM » Schnellsuche* ist es möglich, mit einem Klick auf den Button *Datensatz anonymisieren* die Daten zu einem Kontakt zu entfernen bzw. diese so unkenntlich zu machen, dass kein Bezug zu der Person hergestellt werden kann.

*_Ausnahme:_* Wenn Adressdaten einem Auftrag zugeordnet sind, werden diese Daten aus steuerrechtlichen Gründen nicht entfernt oder unkenntlich gemacht. Die Verknüpfung zum Kontakt wird jedoch entfernt.

Weitere Informationen finden Sie auf der Handbuchseite <<crm/kontakte-verwalten#1707, Kontakte verwalten>>.
