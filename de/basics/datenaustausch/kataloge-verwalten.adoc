== Was ist ein Katalog?

Das Menü *Daten » Kataloge* dient dazu, Variantendaten in strukturierter Form zusammenzustellen und an ein Ziel zu übertragen. Dabei können Kataloge nur erstellt werden, wenn ein Template verfügbar ist. Das Template gibt dabei vor, welche Daten übertragen werden können. Templates werden als Plugin im plentymarkets Backend bereitgestellt.
Kataloge können also nicht flexibel zusammengestellt werden. Auch ein Export eines Katalogs in ein für Sie lesbares Format ist nicht möglich. Dafür benötigen Sie das Plugin ElasticSync. Stattdessen erstellen Sie Kataloge z.B. für Marktplätze. Da der Marktplatz bestimmte Daten benötigt, wird der Katalog mithilfe eines Templates erstellt. Das Template enthält die Artikeldaten, die auf dem Marktplatz veröffentlich werden können, und gibt auch Pflichtfelder vor.

Ein Katalog ist eine Sammlung von Variantendaten für ein bestimmtes Ziel. Der Katalog besteht aus dem Template, das das Plugin vorgibt, und den plentymarkets Daten, die damit verknüpft werden.

Quelle = plenty Variantendaten
Ziel = Marktplatz


Ermöglicht die strukturierte Übertragung von Artikeldaten an ein Ziel zur Veröffentlichung.



== Welchen Vorteil bieten Kataloge?

* Das Template gibt die Struktur vor, Sie wissen also genau, welche Daten zur Veröffentlichung der Variantendaten auf dem Marktplatz nötig sind.
* Sie bestimmen, welche Variantendaten auf dem Marktplatz veröffentlicht werden. Über die Eigenschaften veröffentlichen Sie auf jedem Marktplatz eigene Namen und Beschreibungen von Varianten.




== Was ist ein Template?

 Über das Template gibt der Anbieter/Marktplatz vor, welche Daten für die Datenübermittlung erforderlich sind und wie diese Daten benannt werden. Das Template stellt ein Gerüst mit den Daten bereit, die an den Marktplatz übertragen werden. Die Templates werden über das Marktplatz-Plugin ins plentymarkets Backend geladen. Jeder Marktplatz kann je nach Produktart mehrere Templates anbieten. Das liegt daran, dass je nach Produktart unterschiedliche Daten erforderlich sind. Für Lebensmittel müssen zum Beispiel Mindesthaltbarkeitsdaten übermittelt werden. In anderen Zusammenhängen können Chargen eine Rolle spielen.


== Warum wird das Menü *Daten » Kataloge* nicht angezeigt?

Das Template wird im Menü *Daten » Kataloge* wählbar, wenn das Plugin aktiviert ist. Deshalb wird das Menü *Daten » Kataloge* erst sichtbar, wenn Sie ein Katalog-Plugin installieren und aktivieren.

== Katalog erstellen

[.instruction]
Katalog erstellen:

. Öffnen Sie das Menü *Daten » Kataloge*. +
→ Die Katalogübersicht wird angezeigt.
. Klicken Sie auf *Katalog erstellen*.
. Geben Sie einen Namen für den Katalog ein.
. *Speichern* Sie die Einstellungen. +
→ Der Katalog wird erstellt.
