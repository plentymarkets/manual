==  Allgemeines zum Menü API-Daten

Die API-Daten werden vom System automatisch hinterlegt und können, mit Ausnahme von SOAP Namespace, nicht bearbeitet werden. Hier finden Sie Ihren System-Host, alle offenen Schnittstellen, über die Daten aus dem System ausgelesen oder in das System importiert werden können sowie interne System- und Kundendaten.

==  Erläuterungen zu den Daten

In der folgenden Tabelle werden die Felder des Menüs API-Daten erläutert.

[cols="a,a"]
|====
| Einstellung | Erläuterung

|**Host**
| Die System-URL

|**SOAP WSDL**
| Dieser Endpunkt der **<<datenaustausch/soap-api#, SOAP API  >>** wird immer entsprechend der aktuellen Version angezeigt, auch wenn noch ältere Versionen unterstützt werden.

|**SOAP WSDL C#**
| Dieser Endpunkt der **<<datenaustausch/soap-api#, SOAP API  >>** wird immer entsprechend der aktuellen Version angezeigt, auch wenn noch ältere Versionen unterstützt werden.

|**SOAP Namespace**
| Pro plentymarkets-System kann ein individueller SOAP-Namespace eingetragen werden. Der Standard-Namespace ist weiterhin die System-URL der API-Version, die oben unter **SOAP-WSDL** hinterlegt ist.

|**PlentyToken**
| Der PlentyToken wird zur Verknüpfung von plentymarkets mit den Tools, plentymarkets Shipping und plentyConnect, verwendet.

|**WebToken**
| Der WebToken wird von externen Dienstleistern verwendet, die mit der alten plentymarkets Schnittstelle verbunden sind.

|**PlentyID**
| Die interne plentymarkets System-ID

|**CustomerID**
| Ihre Kunden-ID bei plentymarkets.
|====

__Tab. 1: Das Menü **API-Daten**__
