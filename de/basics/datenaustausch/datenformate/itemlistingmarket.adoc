= ItemListingMarket
:lang: de
include::{includedir}/_header.adoc[]
:keywords: Datenformat ItemListingMarket
:position: 155

Das Datenformat **ItemListingMarket** wird verwendet, um neue Listings zu erstellen oder bestehende Listings zu bearbeiten. Sie können z.B. einzelne Parameter ändern oder löschen. Sie erstellen und exportieren das Datenformat im Menü **Datenaustausch » Dynamischer Export**. Importieren Sie das Datenformat wieder im Menü **Datenaustausch » Datenimport » Dynamischer Import**. Wie Sie das Datenformat exportieren und importieren, ist auf den Handbuchseiten <<basics/datenaustausch/daten-exportieren#, Daten exportieren>> und <<basics/datenaustausch/daten-importieren#, Daten importieren>> beschrieben.

Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.
