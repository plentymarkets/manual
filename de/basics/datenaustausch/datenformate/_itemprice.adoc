= ItemPrice
:lang: de
include::{includedir}/_header.adoc[]
:keywords: Datenformat
:description: Datenformat ItemPrice
:position: 170

Das Datenformat **ItemPrice** wird verwendet, um Preissets von Artikeln anzulegen, zu bearbeiten oder zu löschen. Sie erstellen und exportieren das Datenformat im Menü **Datenaustausch » Dynamischer Export**. Importieren Sie das Datenformat wieder im Menü **Datenaustausch » Datenimport » Dynamischer Import**. Wie Sie das Datenformat exportieren und importieren, ist auf den Handbuchseiten **<<basics/datenaustausch/daten-exportieren#, Daten exportieren>>** und **<<basics/datenaustausch/daten-importieren#, Daten importieren>>** beschrieben.

Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.

== Übersicht der Pflichtfelder

Die folgenden Pflichtfelder müssen für das Datenformat **ItemPrice** vorhanden sein und ein gültiger Wert muss eingegeben sein:

* **ItemID** zum Anlegen eines Preissets
* **PriceID** zum Bearbeiten eines Preissets
* **PriceID** und **DeleteItemPrice** zum Löschen eines Preissets

== Übersicht der Abgleichfelder

Die in Tabelle 2 aufgelisteten Datenfelder stehen zum <<basics/datenaustausch/daten-importieren#25, **Datenabgleich**>> zur Verfügung. Bei Pflichtabgleichfeldern (P) muss für die **Importaktion** die Option **Abgleich** gewählt werden. Wählen Sie mindestens eines der Datenfelder aus der Tabelle zum Abgleich. Sie können aber auch beide wählen. Wird nur das Datenfeld **ItemID** verwendet, wird nur **Preisset 1** abgeglichen.
