Der FormatDesigner ermöglicht Ihnen Formate für den Export per ElasticExport anzulegen. Alle per FormatDesigner angelegten Formate stehen Ihnen also im Menü Datenaustausch » Elastischer Export zur Verfügung. Sie können mithilfe des FormatDesigners nicht nur CSV-Optionen und ausgewählte Datenfelder bestimmen, sonder auch numerische Datenwerte während des Exports manipulieren.

== Format anlegen

. Öffnen Sie das Menü *Daten » FormatDesigner*.
. Klicken Sie auf *Format hinzufügen*.
. Geben Sie einen eindeutigen Namen ein.
. Wählen Sie das gewünschte Format und Feldtrennzeichen.
. Klicken Sie auf *Speichern*.

== Format bearbeiten

Soblad ein Format angelegt oder geöffnet wurde, wird die Bearbeitungsansicht des Formats angezeigt. Auf der rechten Seite befindet sich die Auswahl der verfügbaren Datenfelder. Durch einen Klick auf ein Feld wird das Feld auf der linken Seite ergänzt. Nachdem ein Datenfeld hinzugefügt wurde, können weitere Einstellungen vorgenommen werden. Nach dem Speichern ist das Datenfeld im Export enthalten.

[[tabelle-einstellungen-datenfelder]]
.Einstellungen der Datenfelder
[cols="1,3"]
|====
|Einstellung |Erläuterung

|Namen
|Namen für das Datenformat eingeben. Der Name erscheint in der Kopfzeile der CSV-Datei.

|Verschieben
|Über den mit Punkten markierten Bereich links vom Datenfeldnamen können mit gedrückter Maustaste die Datenfelder verschoben werden. Durch das Verschieben legen Sie die Reihenfolge der CSV-Spalten fest.

|Zuordnung
|Datenfelder, wie z.B. VariationSalesPrice.price, benötigen eine genauere Zuordnung, da z.B. mehrere Preise für jede Variante vorhanden sein können. Mit der Zuordnung wird ein konkreter Verkaufspreis bestimmt. Um z.B. mehrere Preise zu exportieren, kann das Datenfeld mehrfach zum Exportformat hinzugefügt werden. Bei Datenfeldern die dies nicht benötigen, ist der Button deaktiviert.

|Wert ändern
|Datenfelder mit numerischen Werten können während des Exports durch verschiedene Aktionen verändert werden. Dabei können beliebig viele Aktionen definiert werden. Diese Aktionen werden nacheinander ausgeführt und manipulieren den Wert je nach Einstellung.

|Datenfeld entfernen
|Das Datenfeld wird aus dem Format entfernt.
|====
