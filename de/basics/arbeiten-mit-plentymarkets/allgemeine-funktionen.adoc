[#10]
== Handbuchseiten aufrufen

Klicken Sie in einem Menü rechts oben auf **Handbuchsuche**, um die Handbuchseite aufzurufen.

[#20]
== Buttons für Schnellzugriff verwenden

image::basics/arbeiten-mit-plentymarkets/assets/DE-Allgemeine-Funktionen-01.png[]

__Bild 1: Buttons für Schnellzugriff__

Folgende Buttons (von links nach rechts) ermöglichen den Schnellzugriff auf wichtige Bereiche aus der Kopfzeile des plentymarkets Administrationsbereichs heraus:

* **Webseite** +
Öffnet die <<basics/arbeiten-mit-plentymarkets/dashboard#, **Startseite**>> Ihres Webshops.
* **Kundenkarte** +
Öffnet die <<crm/kundendaten-verwalten#1200, **Kundenkarte**>>, in der Sie nach Kunden suchen können.
* **Statistik** +
Öffnet das **<<basics/statistik#, Statistikmodul>>**, mit dem Sie eine Vielzahl von Auswertungen anzeigen lassen können.
* **Support** +
Öffnet eine Übersichtsseite, auf der sämtliche Support-Medien angezeigt werden.
* **Handbuch** +
Öffnet die link:/[**Startseite**] des plentymarkets Handbuchs, in dem alle Funktionen ausführlich beschrieben sind.
* **Changelog** +
Öffnet die Startseite link:https://www.plentymarkets.eu/software/versionen/[plentymarkets Up2date^]. Hier werden alle wichtigen Neuerungen und Änderungen einer plentymarkets Version angezeigt.
* **plentymarkets Download** +
Installiert den <<basics/automatisierung/plentymarkets-client#, **plentymarkets Client**>> auf Ihrem Rechner.
* **plentymarkets Forum** +
Öffnet das plentymarkets Forum.

[#30]
== Menüansicht anpassen

Sie können sowohl die gesamte Menüansicht als auch die Übersicht innerhalb eines Menüs anpassen. Außerdem können Sie die Positionen der Menüansicht sortieren.

[#40]
=== Menüansicht wechseln

Wählen Sie zwischen der horizontalen Menüansicht und der vertikalen Menüansicht. Wechseln Sie die Ansicht mit einem Klick auf den Pfeil in der oberen linken Ecke Ihrer Startseite.

[#50]
=== Menüansicht einrichten

Sie können die Ansicht innerhalb eines Menüs, d.h. die Spalten der Parameter, die angezeigt werden sollen, individuell pro Menü einrichten. Bild 2 zeigt als Beispiel die Übersicht des Menüs **Artikel » Artikel bearbeiten**.

[.instruction]
Ansicht eines Menüs einrichten:

. Zeigen Sie mit der Maus auf die Titelzeile des Menüs.  +
→ Der Button **Ansicht einrichten** wird angezeigt.
. Klicken Sie auf **Ansicht einrichten**.  +
→ Das Fenster **Ansicht einrichten** wird geöffnet.
. Aktivieren Sie die Parameter, die in der Menüansicht angezeigt werden sollen.  +
→  Wenn ein Parameter nicht mehr angezeigt werden soll, deaktivieren Sie den Parameter.
. **Speichern** Sie die Einstellungen.

image::basics/arbeiten-mit-plentymarkets/assets/DE-Allgemeine-Funktionen-02.png[]

__Bild 2: Fenster **Ansicht einrichten**__

[#60]
=== Positionen der Menüansicht sortieren

Gehen Sie wie im Folgenden beschrieben vor, um die Positionen der Menüansicht anhand einer Spalte alphabetisch bzw. numerisch aufsteigend oder absteigend zu sortieren.

[.instruction]
Positionen der Menüansicht sortieren:

. Zeigen Sie mit der Maus auf die Überschrift der Spalte, nach der Sie sortieren möchten. +
→ Ein kleines Dreieck wird angezeigt.
. Klicken Sie auf das Dreieck, um die Einträge anhand dieser Spalte zu sortieren. +
→ Wenn das Dreieck nach unten zeigt, werden die Spalteneinträge aufsteigend sortiert (der größte Wert steht oben). Zeigt das Dreieck nach oben, wird absteigend sortiert.

[#70]
== Kontextmenü über Rechtsklick aufrufen

Der Rechtsklick in plentymarkets auf ausgewählte Tabs und Buttons ermöglicht es Ihnen, schnell gezielte Aktionen auszuführen.

Mit einem Rechtsklick im Menü **Artikel » Kategorien** auf eine Kategorie z.B. haben Sie die Möglichkeit, die Kategorie zu bearbeiten, zu verschieben, zu kopieren, zu löschen, eine Unterkategorie oder einen neuen Artikel zu erstellen. Im Menü **CMS » Bilder-Galerie** können Sie z.B. mit einem Rechtsklick auf die Bilderordner im Verzeichnisbaum auf der linken Seite komplette Ordner mit Inhalt löschen. Im Menü **System » Einstellungen » Drucker** können Sie per Rechtsklick auf **Drucker** einen neuen Drucker einrichten.

[#80]
== Favoriten hinzufügen und entfernen

Favoriten dienen dazu, häufig gebrauchte Menüpunkte schnell aufzurufen.

[.instruction]
Menü zu Favoriten hinzufügen:

. Öffnen Sie das Menü.
. Klicken Sie auf **Zu Favoriten hinzufügen** rechts oben im Menü.  +
→ Alternativ klicken Sie mit der rechten Maustaste auf den Tab-Titel des Menüs und wählen die Option **Zu Favoriten hinzufügen**.
. Prüfen Sie im Menü **System » Einstellungen » Benutzer » Konten** unter **Mein Konto** im Tab **Persönliche Einstellungen » Favoriten**, ob das Menü hinzugefügt wurde.  +
→ **__Hinweis__**: Favoriten sind im Menü erst sichtbar, wenn Sie mindestens einen Favoriten gespeichert haben.

[.instruction]
Menü aus Favoriten entfernen:

. Öffnen Sie das Menü.
. Klicken Sie auf **Aus Favoriten entfernen** rechts oben im Menü.  +
→ Alternativ klicken Sie mit der rechten Maustaste auf den Tab-Titel des Menüs und wählen die Option **Aus Favoriten entfernen**.
. Prüfen Sie im Menü **System » Einstellungen » Benutzer » Konten** unter **Mein Konto** im Tab **Persönliche Einstellungen » Favoriten**, ob das Menü entfernt wurde.

[#90]
== Mehrere Optionen wählen

Wenn Sie in Auswahlfeldern die Möglichkeit haben, mehrere Optionen gleichzeitig zu wählen, drücken Sie die Strg-Taste (Windows) bzw. die cmd-Taste (Mac) und klicken auf die Optionen. Optionen, die beim Öffnen eines Auswahlfensters ausgegraut sind, wurden bereits gespeichert. Sie können nun per Strg- bzw. cmd-Taste und Mausklick weitere Optionen dazuwählen. Alternativ können Sie per Mausklick die bereits gewählten Optionen deaktivieren. Während der Bearbeitung werden alle gewählten Optionen - auch diejenigen, die vorher ausgegraut waren - blau angezeigt.

== Ungesicherte Änderungen anzeigen

In den Menüs **CMS » Webdesign » Layout** und **Artikel » Kategorien** z.B. werden Tabs durch eine rote Markierung gekennzeichnet, wenn Sie Änderungen vorgenommen, diese jedoch noch nicht gespeichert haben. Diese Markierung wird auch links im Menübaum angezeigt.

image::basics/arbeiten-mit-plentymarkets/assets/DE-Allgemeine-Funktionen-03.png[]

__Bild 3: Kennzeichnung ungesicherter Änderungen__

Wenn Sie eine Kategorie schließen, ohne die Änderungen gespeichert zu haben, erscheint ein Hinweisfenster, das Sie aktiv bearbeiten müssen, um fortzufahren.

* Klicken Sie auf **Schließen**, um die Kategorie zu schließen, ohne die Änderungen zu speichern.
* Klicken Sie auf **Abbrechen**, um die Änderungen zu speichern oder weitere Anpassungen vorzunehmen.

Wenn Sie solche Meldungen nicht mehr angezeigt bekommen möchten, deaktivieren Sie die Funktion.

[.instruction]
Meldungen deaktivieren:

. Öffnen Sie das Menü **System » Einstellungen » Benutzer » Konten**.
. Klicken Sie auf **Mein Konto**.
. Wechseln Sie in das Tab **Persönliche Einstellungen » Allgemein**.
. Deaktivieren Sie die Option **Warnen, wenn Tabs offen sind und das Fenster geschlossen wird**.
. **Speichern** Sie die Einstellungen.

[#120]
== Updates durchführen

Ihr plentymarkets System prüft regelmäßig, ob eine Aktualisierung der Menüs bzw. Funktionen verfügbar ist, wenn z.B. eine Erweiterung oder ein Fix eingespielt wurde. Wenn ein Update verfügbar ist, erscheint ein Dialogfenster mit der Frage, ob die Aktualisierung jetzt oder zu einem späteren Zeitpunkt vorgenommen werden soll.

Wenn Sie die Aktualisierung mit einem Klick auf **Seite neu laden** bestätigen, werden alle geöffneten Menüs bzw. Tabs geschlossen. Klicken Sie auf **Später**, wenn die Aktualisierung zu einem späteren Zeitpunkt ausgeführt werden soll.

**__Hinweis__**: Wenn die Seite neu geladen wird, wird automatisch der Cache erneuert.
