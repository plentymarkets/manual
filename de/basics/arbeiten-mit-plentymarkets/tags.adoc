= Tags verwalten
:lang: de
include::{includedir}/_header.adoc[]
:keywords: Tag, Tags, Tag erstellen
:description: Erfahren Sie auf dieser Handbuchseite, wie Sie Tags in plentymarkets verwalten.
:position: 60

Tags sind Schlagworte, mit denen Datensätze in plentymarkets markiert werden. Tags werden zentral im Menü *System » Einstellungen » Tags* erstellt und verwaltet. Diese Tags können mit Datensätzen in den folgenden Bereichen von plentymarkets verknüpft werden:

* Board
* Kategorie
* Ticket
* Kontakt
* Variante
* Auftrag

[#100]
== Tag erstellen

Gehen Sie wie unten beschrieben vor, um ein Tag zu erstellen.

[.instruction]
Tag erstellen:

. Öffnen Sie das Menü *System » Einstellungen » Tags*.
. Klicken Sie auf *Neues Tag erstellen*. +
→ Rechts wird ein Fenster geöffnet.
. Nehmen Sie die Einstellungen vor. Beachten Sie dazu die Erläuterungen in <<tabelle-tag-einstellungen>>.
. *Speichern* Sie die Einstellungen. +
→ Das Tag wird erstellt und der Liste hinzugefügt.

[[tabelle-tag-einstellungen]]
.Tag erstellen
[cols="1,3"]
|====
|Einstellung |Erläuterung

|*Name DE*
|Namen für den Tag eingeben. +
*_Tipp:_* Tag-Namen können Leerzeichen enthalten, um zum Beispiel ein Tag zu erstellen, das aus zwei Worten besteht.

|*Farbe*
|*_Optional:_* Farbe für das Tag als Hexadezimalwert eingeben. Für die 16 Grundfarben kann statt des Hexadezimalwerts die englische Bezeichnung der Farbe eingegeben werden (black, red, silver, white etc.) +
*_Tipp:_* Auf das Quadrat rechts klicken, um den Farbwähler zu öffnen.

|*Bereich*
|Bereiche wählen, in denen das Tag verfügbar sein soll. +
*Board* = Tag kann mit Aufgabenkarten auf Boards des Menüs *Start » Boards* verknüpft werden. +
*Kategorie* = Tag kann mit Kategorien verknüpft werden. +
*Ticket* = Tag kann mit Tickets des Ticketsystems verknüpft werden. +
*Kontakt* = Tag kann mit Kontakten verknüpft werden. +
*Variante* = Tag kann mit Varianten verknüpft werden. +


|Name in anderen Sprachen
|Tag-Namen in weiteren Sprachen eingeben. Weitere Informationen finden Sie im Kapitel <<basics/arbeiten-mit-plentymarkets/tags#200, Weitere Sprachen zu Tags hinzufügen>>.
|====

[#200]
== Tag-Namen in weiteren Sprachen hinzufügen

Gehen Sie wie unten beschrieben vor, um zu einem bereits erstellten Tag Namen in weiteren Sprachen hinzuzufügen.

[.instruction]
Tag-Namen in weiteren Sprachen hinzufügen:

. Öffnen Sie das Menü *System » Einstellungen » Tags*.
. Klicken Sie auf *Suchen*. +
→ Alle Tags werden angezeigt.
. Kicken Sie auf das Tag. +
→ Rechts wird ein Fenster geöffnet.
. Geben Sie in die Felder der gewünschten Sprachen die Übersetzung ein.
. *Speichern* Sie die Einstellungen.

[#300]
== Tags filtern

Tags können nach *Name*, *Bereich* und *Sprache* gefiltert werden. Gehen Sie wie unten beschreiben vor, um Tags anhand dieser Kriterien zu filtern.

[.instruction]
Tags filtern:

. Öffnen Sie das Menü *System » Einstellungen » Tags*.
. Wählen Sie einen oder mehrere Filter. Beachten Sie dazu die Erläuterungen in <<tabelle-tag-filter>>.
. Klicken Sie auf *Suchen*. +
→ Die Tags werden angezeigt, die den eingestellten Suchkriterien entsprechen.

[[tabelle-tag-filter]]
.Tag-Filter
[cols="1,3"]
|====
|Einstellung |Erläuterung

|*Name*
|Name des Tags eingeben. +
*_Tipp:_* Wenn alle Sprachen gewählt wurden, werden alle Sprachen berücksichtigt. Angezeigt wird jedoch der unter *Name DE* eingegebene Wert.

|*Bereich*
|Einen oder mehrere Bereiche wählen, um nur für diese Bereiche aktivierte Tags anzuzeigen.

|*Sprache*
|Einen oder mehrere Sprachen wählen, um nur Tags mit Namen in diesen Sprachen anzuzeigen.
|====
