= Tags
:lang: de
include::{includedir}/_header.adoc[]
:position: 110


Tags sind Schlagworte, mit denen Texte und Seiten markiert werden können. In plentymarkets werden Tags zentral in einem Menü angelegt und verwaltet und stehen dann in verschiedenen Bereichen, z.B. **Kategorien** und **Ticketsystem**, zur Verfügung. Sie können Tags auch mit Leerzeichen anlegen. Wenn ein Tag z.B. aus zwei Worten besteht, ist ein Leerzeichen dazwischen möglich.

== Tag anlegen

Im Folgenden wird erklärt, wie Sie Tags anlegen.



[.instruction]
Neues Tag anlegen:

. Öffnen Sie das Menü **System » Einstellungen » Tags**.
. Klicken Sie auf das **grüne Plus**, um ein neues Tag hinzuzufügen. +
→ Ein Fenster wird geöffnet.
. Tragen Sie einen **Namen** für das Tag ein.
. Klicken Sie auf **Speichern**, um die Einstellung zu sichern. +
→ Das Tag wird im Verzeichnisbaum links angezeigt.
. Öffnen Sie das Tag.
. Wählen Sie bestimmte **Bereiche**, in denen das Tag verwendet werden soll, oder **Alle**.
. *Speichern* Sie die Einstellungen.

Es besteht die Möglichkeit, Tags in anderen Sprachen anzulegen. Gehen Sie dazu, wie im Folgenden beschrieben vor.

[.instruction]
Tag in weiterer Sprache hinzufügen:

. Öffnen Sie das Menü **System » Einstellungen » Tags**.
. Wählen Sie das Tag aus dem Verzeichnisbaum links.
. Wählen Sie eine **Sprache** aus der Dropdown-Liste im Bereich **Texte**.
. Klicken Sie auf das **grüne Plus** hinter der Sprach-Dropdown-Liste. +
→ Die Sprache wird hinzugefügt.
. Klicken Sie in die Zeile der Sprache und tragen Sie einen **Text** ein.
. Klicken Sie auf **Speichern**, um die Einstellung zu sichern.

Die folgende Tabelle erläutert die Einstellungen eines **Tags**:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|**Name**
|Zeigt den Namen des Tags. Dieser Name ist __**keiner**__ Sprache zugeordnet.

|**Bereich**
|Einen Bereich, wie **Kategorien** oder **Ticketsystem**, oder **Alle** wählen, damit das Tag dort verfügbar ist.

|**Texte**
|Tag in weiteren **Sprachen** anlegen oder Tag für eine Sprache ändern.
|====

== Tags filtern

Es stehen drei Filter zur Verfügung, um Tags zu filtern. Es können der **Bereich**, die **Sprache** sowie der **Name** gefiltert werden.



Die folgende Tabelle erläutert die **Filter** für Tags:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|**Bereich**
|Einen Bereich (Tabelle 1) oder **Alle** wählen. Nach einem Klick in das Feld, öffnet sich eine Dropdown-Liste mit den Bereichsoptionen.

|**Sprache**
|Eine Sprache oder **Alle** wählen. Nach einem Klick in das Feld, öffnet sich eine Dropdown-Liste mit den Sprachoptionen.

|**Name**
|Einen Namen eintragen. Nach Eingabe des ersten Buchstabens, werden passende Ergebnisse angezeigt. Wenn alle Sprachen gewählt wurden, werden hierbei alle Sprachen berücksichtigt. Angezeigt wird jedoch der unter **Name** eingetragene Werte.
|====
