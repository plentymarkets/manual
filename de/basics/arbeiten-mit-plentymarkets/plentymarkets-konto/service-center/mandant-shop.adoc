= Mandant (Shop)
:lang: de
include::{includedir}/_header.adoc[]
:keywords: Mandant (Shop), Multishop
:position: 50

<<omni-channel/online-shop#, **Mandant (Shop)**  >> bezeichnet in plentymarkets weitere Webshops, die über den zugeordneten Hauptshop in Ihrem plentymarkets System konfigurierbar sind. Im Menü **Start » plentymarkets Konto » Service-Center » Tab: Mandant (Shop)** werden Ihre Mandanten (Shops) angezeigt. Bei Bedarf können Sie über dieses Menü auch eine Kündigung der Mandanten (Shops) veranlassen.

== Prüfen: Domain angelegt?

Um einen Mandanten (Shop) zu bestellen, benötigen Sie eine <<basics/arbeiten-mit-plentymarkets/plentymarkets-konto/service-center/domain#, **Domain**  >>, die Sie ggf. vorher im Tab **Domain** für das Hauptsystem anlegen müssen.

== Einen Mandanten (Shop) bestellen

Um einen Mandanten (Shop) zu bestellen, öffnen Sie das Tab **Vertrag** des Service-Centers und klicken auf **Zum Systemkonfigurator**. Nehmen Sie im Systemkonfigurator die gewünschte Änderung Ihres Vertrags vor.

== Mandant (Shop) kündigen

Sie können einen Mandanten (Shop) kündigen, indem Sie im Menü **Start » plentymarkets Konto » Service-Center » Tab: Mandant (Shop)** auf **Mandant (Shop) kündigen** klicken. Bestätigen Sie im angezeigten Fenster Ihre Kündigung. Vertragsbeginn und -ende werden angezeigt.

== Domain-Einstellungen von Mandanten (Shops) bearbeiten

Im Tab **Mandant (Shop)** werden außerdem Informationen zu den Domains angezeigt, die Ihren Mandanten zugeordnet sind. Nehmen Sie Änderungen an diesen Einstellungen wie auf der Handbuchseite **<<basics/arbeiten-mit-plentymarkets/plentymarkets-konto/service-center/domain#, Domain  >>** vor.
