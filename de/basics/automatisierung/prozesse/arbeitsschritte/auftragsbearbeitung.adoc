== Der Prozess Auftragsbearbeitung

Mit einem Arbeitsschritt des Typs **Auftragsbearbeitung** können Sie festlegen, wie einzelne Aufträge weiter bearbeitet werden sollen, nachdem diese z.B. mittels eines Handscanners erfasst wurden. Es handelt sich hierbei um einen sehr flexiblen Arbeitsschritt, den Sie durch die Verwendung der entsprechenden Aktionen und Subaktionen nach Ihren Wünschen konfigurieren.

== Konfiguration

Erstellen Sie zunächst einen Arbeitsschritt.

[.instruction]
Arbeitsschritt erstellen:

. Klicken Sie auf den Button **Neuer Arbeitsschritt**.
. Wählen Sie als Typ **Auftragsbearbeitung** und klicken Sie auf **Speichern**. +
→ Das Konfigurationsfenster wird geöffnet (Bild 1).

Bei einem bereits erstellten Arbeitsschritt klicken Sie auf den **Stift**, um dieses Fenster zu öffnen.

image::basics/automatisierung/prozesse/arbeitsschritte/assets/DE-Einstell-Prozesse-Auftragsbearbeitung-01.png[]

__Bild 1: **Arbeitsschritt** konfigurieren__

Die folgende Tabelle erläutert die __**Einstellungen**__ des Arbeitsschrittes:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|Name
|Name bzw. Bezeichnung für den Arbeitsschritt. Wählen Sie einen Namen, der den Arbeitsschritt möglichst __**eindeutig**__ bezeichnet.

|Folgeschritt
|Wählen Sie, ob ein weiterer Arbeitsschritt __**manuell**__ oder __**automatisch**__ gestartet werden soll. Diese Einstellung ist nur relevant, wenn mindestens zwei Arbeitsschritte eingerichtet sind. +
**Eingabe abwarten** = Der nächste Arbeitsschritt wird __**nicht automatisch**__ gestartet, sondern es wird vom System ein erneuter Klick auf den **Start-Button** in der Ausführung des Prozesses abgewartet. +
**Automatisch starten** = Ein Folgeschritt oder eine Aktion wird __**automatisch**__ gestartet.
|====

__Tab. 1: **Arbeitsschritt konfigurieren**; **Einstellungen**__

__**Nicht vergessen**__: Einstellungen oder Änderungen prüfen und speichern!

Der betreffende Arbeitsschritt wird angezeigt. Wenn Sie dann mit der Maus auf das Element zeigen, werden die Konfigurationselemente sichtbar:

image::basics/automatisierung/prozesse/arbeitsschritte/assets/DE-Einstell-Prozesse-Auftragsbearbeitung-02-SI.png[]

__Bild 2: Arbeitsschritt **Auftragsbearbeitung**, **Konfiguration** öffnen__

[NOTE]
.Weitere Elemente hinzufügen
====
Mit dem **grünen Plus** (Bild 2) fügen Sie eine **Aktion** bzw. **Subaktion** hinzu, indem Sie jeweils das gewünschte Element aus dem Dropdown-Menü wählen.
====
