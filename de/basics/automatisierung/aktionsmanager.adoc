= Aktionsmanager
:lang: de
include::{includedir}/_header.adoc[]
:position: 50

In plentymarkets lassen Sie tägliche Ereignisse automatisch über den *Aktionsmanager* zu bestimmten Uhrzeiten ausführen.

So können Sie den Aktionsmanager beispielsweise nutzen, um einen täglichen Zeitpunkt festzulegen, zu dem nicht zahlende Kunden von eBay oder Ihrem eigenen Webshop eine Zahlungserinnerung oder eine Mahnung erhalten sollen und diese automatisch per E-Mail versenden lassen.

== Neue Aktion einrichten

Die Einrichtung einer Aktion erfolgt in den drei Tabs **Einstellungen**, **Filter** und **Aktion**. Gehen Sie wie im Folgenden beschrieben vor, um die einzelnen Tabs zu konfigurieren.

=== Allgemeine Einstellungen vornehmen

Im ersten Schritt benennen Sie die Aktion, legen einen Zeitpunkt für die Ausführung fest und aktivieren den täglichen Vorgang.

[.instruction]
Neue Aktion einrichten:

. Öffnen Sie das Menü *System » Aufträge » Aktionen*.
. Öffnen Sie das Tab *Neue Aktion*.
. Tragen Sie einen aussagekräftigen Namen für die Aktion ein.
. Wählen Sie einen Zeitpunkt, zu dem die Aktion täglich ausgeführt werden soll.
. *Speichern* Sie die Einstellungen. +
→ Die Aktion wird angelegt.

=== Filtereinstellungen vornehmen

Wählen Sie im nächsten Schritt einen genauen Filter für die Aktion, damit die Aktion nur bei den gewünschten Aufträgen greift. Gehen Sie dazu wie im Folgenden beschrieben vor.

[.instruction]
Filtereinstellungen vornehmen:

. Öffnen Sie das Menü *System » Aufträge » Aktionen*.
. Öffnen Sie das Tab *Filter*.
. Nehmen Sie die gewünschten Einstellungen anhand der Informationen in <<tabelle-aktionsmanager-filter>> vor.
. *Speichern* Sie die Einstellungen.

=== Aktion auswählen

Nach der Auswahl der Filtereinstellungen wählen Sie, welche konkreten *Aktionen* automatisch zum eingestellten Zeitpunkt für die gefilterten Aufträge von Ihrem plentymarkets System ausgeführt werden. Versenden Sie zum Beispiel automatisch eine E-Mail-Vorlage oder veranlassen Sie, dass der Status des Auftrags oder die Mahnstufe automatisch geändert werden. Für eBay-Aufträge ist es möglich eine <<omni-channel/multi-channel/ebay-einrichten#1300, Bewertung automatisch abzugeben>> oder den Disputprozess zu starten. Darüber hinaus schalten Sie einen der aufgeführten <<auftragsabwicklung/payment#400, Inkassodienstleister>> durch Wählen der Option *AKTIV* ein. Betroffene Aufträge werden dann automatisch von uns an den Dienstleister gemeldet. Gehen Sie dazu wie im Folgenden beschrieben vor.

[.instruction]
Aktion auswählen:

.  Öffnen Sie das Menü *System » Aufträge » Aktionen*.
.  Nehmen Sie die gewünschten Einstellungen vor.
.  *Speichern* Sie die Einstellungen. +
→ Die Aktion wird hinzugefügt.
