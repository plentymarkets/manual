== Status der aktuellen Aufträge

Diese Statistik zeigt die aktuellen Aufträge aufgesplittet nach __**Auftragsstatus**__, wobei die jeweilige __**Anzahl der Aufträge**__ und der zugehörige __**Brutto-Umsatz in Euro**__ pro Auftragsstatus angezeigt wird. Die jeweiligen Auftragsstatus sind durch **Start**- und **End**-Status genau eingrenzbar.

== Statistik-Details

Es wird die __**Anzahl der Aufträge**__ und der __**Umsatz in Euro**__ für den jeweiligen __**Auftragsstatus**__ angezeigt. Sind für den jeweiligen Auftragsstatus keine Aufträge vorhanden, wird der Linienpunkt **0** bzw. kein Balken angezeigt. Als Tooltip wird der __**Betrag in Euro**__ sowie die __**Anzahl der Aufträge**__ des jeweiligen Auftragsstatus angezeigt (Bild 1). Sie haben mit dieser Statistik einen Überblick über die Verteilung Ihrer Aufträge hinsichtlich des Auftragsstatus und können z.B. auf einen Blick erkennen, wieviele Aufträge noch nicht bezahlt wurden.

In Bild 1 wird z.B. für den Auftragsstatus __**Bezahlt**__ ein Umsatz von __**8634,51 Euro**__ angezeigt.

[IMPORTANT]
.Status Versandt
====
Für den Status **Versandt** werden nur Aufträge des aktuellen Tages berücksichtigt.
====

[NOTE]
.Details anzeigen (Tooltip)
====
Zeigen Sie mit dem __**Mauszeiger**__ auf ein Element der Statistik, um die jeweiligen __**Details**__ anzuzeigen.
====

== Einstellungen

Legen Sie zunächst wie auf der Handbuchseite **<<basics/statistik#, Statistik  >>** beschrieben eine neue Statistik an. Für die Statistik **Status aktuelle Aufträge** stehen dann die in Tabelle 1 beschriebenen __**Einstellungen**__ zur Verfügung.

image::basics/statistik/allgemeine-statistiken/prozesse/assets/DE-Statistik-StatusAktuelleAuftraege-02-SI.png[]

__Bild 2: Statistik **Status aktuelle Aufträge**; **Einstellungen**__

Die folgende Tabelle erläutert die Einstellungen im Menü **Start » Statistik → Statistik-Typ: Prozesse / Status der aktuellen Aufträge**:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|**Säulen- und Liniendiagramm** = Darstellung der Werte als Säule bzw. Linie +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine __**CSV-Datei**__ exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Es wird ebenfalls eine __**CSV-Datei**__ erzeugt. Die Daten werden allerdings detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Die __**Bezeichnung**__ der Statistik, die beim Erstellen gewählt wurde. Der Name kann hier geändert werden.

|**Start-Status / End-Status**
|Einen **Start**- und **Endstatus** für jeden Auftragsstatus wählen.

|**Mandant (Shop)**
|Die __**Webshops**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Auftragsherkunft**
|Die __**Auftragsherkünfte**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Diagramm**
|Wählen, ob __**Umsätze**__ in der Statistik als __**Brutto**__ oder __**Netto**__ angezeigt werden sollen.

|**Amazon-Konto**
|Das __**Amazon-Konto**__ wählen, das für die Statistik berücksichtigt werden soll.

|**eBay-Konto**
|Das __**eBay-Konto**__ wählen, das für die Statistik berücksichtigt werden soll.
|====

__Tab. 1: **Statistik bearbeiten**; **Einstellungen**__
