== Webbesucher-Statistik

plentymarkets liefert Ihnen in diesem Menü mit dem Webalizer ein eigenes Tool zur Auswertung der Webbesucher-Statistik Ihres Webshops. Sie erhalten damit einen Überblick der Seitenaufrufe Ihres Webshops.

image::omni-channel/online-shop/extras/assets/DE-Mandant-GlobEinstell-Statistik-01-SI.png[]

__Bild 1: **Webbesucher-Statistik** in plentymarkets__

== Menü öffnen

Um die **Webbesucher-Statistik** Ihres plentymarkets-Systems zu öffnen, wählen Sie das entsprechende Menü (Menüpfad siehe oben).

image::omni-channel/online-shop/extras/assets/DE-Mandant-GlobEinstell-Statistik-02-SI.png[]

__Bild 2: Anzeige der **Login-Daten**__

Die für das Menü nötigen **Login-Daten** werden angezeigt, diese müssen Sie im nächsten Fenster eingeben.

[NOTE]
.
====
Klicken Sie zunächst auf den Link **Webalizer-Statistik öffnen** (siehe Bild 2).
====

image::omni-channel/online-shop/extras/assets/DE-Mandant-GlobEinstell-Statistik-03-SI.png[]

__Bild 3: Bei Webalizer **anmelden**__

Tragen Sie die **Login-Daten** nun in die Textfelder **Nutzername** und **Passwort** ein und klicken Sie dann auf **Anmelden** (siehe Bild 3). +
→ Das Menü wird angezeigt.

== Webalizer-Auswertungen

In den verschiedenen Bereichen des Webalizers erhalten Sie vielfältige Informationen zu den Webbesuchen Ihres Webshops, wie z.B. die Anzahl der Seitenaufrufe mit prozentualer Angabe oder die Anzahl der Aufrufe einer Kategorie.

image::omni-channel/online-shop/extras/assets/DE-Mandant-GlobEinstell-Statistik-04-SI.png[]

__Bild 4: **Zeitraum** im Webalizer auswählen__

[NOTE]
.
====
Klicken Sie nun auf den gewünschten **Monat**, um eine Auswertung des betreffenden Monats anzuschauen (siehe Bild 4).
====

image::omni-channel/online-shop/extras/assets/DE-Mandant-GlobEinstell-Statistik-05-SI.png[]

__Bild 5: **Monatsstatistik**__

[TIP]
.Tipp
====
Zu den Diagrammen sehen Sie jeweils eine Legende am Rand der Auswertung (siehe Bild 5, Pfeile).
====
