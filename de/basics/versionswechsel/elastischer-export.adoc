= Elastischer Export
:lang: de
include::{includedir}/_header.adoc[]
:position: 30

Im Menü **Datenaustausch » Export » Elastischer Export** erstellen Sie Exportformate für u.a. Preissuchmaschinen, Preisportale, Marktplätze und Affiliate-Dienstleister. Auf dieser Handbuchseite erfahren Sie, welche Exportformate in plentymarkets zur Verfügung stehen und wie Sie diese erstellen, bearbeiten und filtern.

== Bei der Preissuchmaschine registrieren

Bevor Sie Ihr Exportformat an die Preissuchmaschine übertragen können, müssen Sie sich auf der Webseite der Preissuchmaschine registrieren und dort den Hinweisen folgen. Die Links zu den Webseiten der Preissuchmaschinen, zu denen es eine Schnittstelle in plentymarkets gibt, finden Sie in Tabelle 1.

== Exportformate erstellen

Dieses Kapitel beschreibt, wie Sie neue Exportformate erstellen und welche Einstellungen Sie vornehmen können.

[.instruction]
Neues Exportformat erstellen:

. Öffnen Sie das Menü **Datenaustausch » Export » Elastischer Export**.
. Klicken Sie auf **Neuer Export**.
. Nehmen Sie die Einstellungen vor. Beachten Sie dazu die Erläuterungen in Tabelle 2.
. **Speichern** Sie die Einstellungen. +
→ Eine ID für das Exportformat wird vergeben und das Exportformat erscheint in der Übersicht **Exporte**.

[cols="a,a"]
|====
|Einstellung |Erläuterung

2+|Einstellungen

|**Name**
|Namen eingeben. Unter diesem Namen erscheint das Exportformat in der Übersicht im Tab **Exporte**.

|**Typ**
|Typ aus dem Dropdown-Menü wählen. Aktuell ist nur die Auswahl **Artikel** verfügbar. Weitere Auswahlmöglichkeiten folgen.

|**Format**
|Format für das Preisportal aus dem Dropdown-Menü wählen.

|**Limit**
|Zahl eingeben. Wenn mehr als 10.000 Datensätze an die Preissuchmaschine übertragen werden sollen, wird die Ausgabedatei für 24 Stunden nicht erneut generiert, um Ressourcen zu sparen.

|**Bereitstellung**
|**URL**, **FTP** oder **Admin-Bereich** wählen. +
Je nach gewählter Option, erscheinen unterschiedliche Einstellungsmöglichkeiten.

|**Token, +
URL**
|Wenn unter **Bereitstellung** die Option **URL** gewählt wurde, auf **Token generieren** klicken. Der Token wird dann automatisch eingetragen. Die URL wird automatisch eingetragen, wenn unter **Token** der Token generiert wurde.

|**Server, +
Port, +
Benutzername, +
Passwort, +
Pfad**
|Wenn unter **Bereitstellung** die Option **FTP** gewählt wurde, Parameter eingeben. Der Standard-FTP-Port ist **21**.

|**Übertragung**
|Wählen, wie oft die Übertragung stattfinden soll.

2+|Artikelfilter

|**Artikelfilter hinzufügen**
|Artikelfilter aus dem Dropdown-Menü wählen und auf **Hinzufügen** klicken. Es ist möglich, alle Artikelfilter aus dem Dropdown-Menü nacheinander hinzuzufügen. +
**Varianten** = **Alle übertragen** oder **Nur Hauptvarianten übertragen** wählen. +
**Märkte** = Einen, mehrere oder **ALLE** Märkte wählen. +
**Währung** = Währung wählen. +
**Kategorie** = Aktivieren, damit der Artikel mit Kategorieverknüpfung übertragen wird. +
**Bild** = Aktivieren, damit der Artikel mit Bild übertragen wird. +
**Herkunft** = Herkunft wählen. Die Herkunft ist relevant für die Verkaufspreisermittlung. +
**Bestand** = Wählen, welche Bestände exportiert werden sollen. +
**Markierung 1 - 2** = Markierung wählen. +
**Hersteller** = Einen, mehrere oder **ALLE** Hersteller wählen.

2+|Formateinstellungen

|**Produkt-URL**
|Aktivieren, damit die Produkt-URL an das Preisportal übertragen wird.

|**Auftragsherkunft**
|Auftragsherkunft aus dem Dropdown-Menü wählen. Die Produkt-URL wird um die gewählte Auftragsherkunft erweitert, damit die Verkäufe später analysiert werden können.

|**URL-Parameter**
|Suffix für die Produkt-URL eingeben, wenn dies für den Export erforderlich ist. Die Produkt-URL wird dann um die eingegebene Zeichenkette erweitert, wenn weiter oben die Option **übertragen** für die Produkt-URL aktiviert wurde.

|**Artikelname**
|**Name 1**, **Name 2** oder **Name 3** wählen. Die Namen sind im Tab **Texte** eines Artikels gespeichert. +
Im Feld **Maximale Zeichenlänge (def. Text)** optional eine Zahl eingeben, wenn die Preissuchmaschine eine Begrenzung der Länge des Artikelnamens beim Export vorgibt.

|**Vorschautext**
|Wählen, ob und welcher Text als Vorschautext übertragen werden soll. +
Im Feld **Maximale Zeichenlänge (def. Text)** optional eine Zahl eingeben, wenn die Preissuchmaschine eine Begrenzung der Länge des Vorschautexts beim Export vorgibt. +
Option **HTML-Tags entfernen** aktivieren, damit die HTML-Tags beim Export entfernt werden. +
Im Feld **Erlaubte HTML-Tags, kommagetrennt (def. Text)** optional die HTML-Tags eingeben, die beim Export erlaubt sind. Wenn mehrere Tags eingegeben werden, mit Komma trennen.

|**Beschreibung**
|Wählen, welcher Text als Beschreibungstext übertragen werden soll. +
Im Feld **Maximale Zeichenlänge (def. Text)** optional eine Zahl eingeben, wenn die Preissuchmaschine eine Begrenzung der Länge der Beschreibung beim Export vorgibt. +
Option **HTML-Tags entfernen** aktivieren, damit die HTML-Tags beim Export entfernt werden. +
Im Feld **Erlaubte HTML-Tags, kommagetrennt (def. Text)** optional die HTML-Tags eingeben, die beim Export erlaubt sind. Wenn mehrere Tags eingegeben werden, mit Komma trennen.

|**Barcode**
|ASIN, ISBN oder eine EAN aus dem Dropdown-Menü wählen.

|**Bild**
|**Position 0** oder **Erstes Bild** wählen, um dieses Bild zu exportieren. +
**Position 0** = Ein Bild mit der Position 0 wird übertragen. +
**Erstes Bild** = Das erste Bild wird übertragen.

|**Angebotspreis**
|Aktivieren, um den Angebotspreis zu übertragen.

|**UVP**
|Aktivieren, um den UVP zu übertragen.

|**Versandkosten**
|Aktivieren, damit die Versandkosten aus der Konfiguration übernommen werden. Wenn die Option aktiviert ist, stehen in den beiden Dropdown-Menüs Optionen für die Konfiguration und die Zahlungsart zur Verfügung. +
Option **Pauschale Versandkosten übertragen** aktivieren, damit die pauschalen Versandkosten übertragen werden. Wenn diese Option aktiviert ist, im Feld darunter einen Betrag eingeben.

|**MwSt.-Hinweis**
|Betrag eingeben. Dieser Betrag enthält die Mehrwertsteuer zuzüglich Versand.

|**Artikelverfügbarkeit**
|Option **überschreiben** aktivieren und in die Felder **1** bis **10** Artikelverfügbarkeiten eintragen. Somit werden die Artikelverfügbarkeiten, die im Menü **Einstellungen » Artikel » Artikelverfügbarkeit** eingestellt wurden, überschrieben.

== Exportformate bearbeiten

In der Exportübersicht werden alle erstellten Exportformate angezeigt. Über die Funktion **Ansicht einrichten** wählen Sie, welche Spalten angezeigt werden. Zeigen Sie mit der Maus auf die Titelleiste und klicken Sie auf **Ansicht einrichten**. Das gleichnamige Fenster wird geöffnet und Sie aktivieren oder deaktivieren die Spaltennamen. Die folgenden Spalten stehen zur Verfügung:

* ID
* Name
* Download
* Typ
* Format
* Limit
* Letzte Export-Änderung
* Export erstellt
* Löschen

Öffnen Sie ein Exportformat, um Einstellungen zu prüfen oder zu bearbeiten.

[.instruction]
Exportformat bearbeiten:

. Öffnen Sie das Menü **Datenaustausch » Export » Elastischer Export**.
. Öffnen Sie das Exportformat.
. Ändern Sie die Einstellungen. Beachten Sie dazu die Erläuterungen in den Tabellen 2 und 3.
. **Speichern** Sie die Einstellungen.

== Exportformate filtern

Sie können die von Ihnen erstellten Exportformate nach bestimmten Kriterien filtern, um nur diese Exportformate in der Übersicht **Exporte** anzuzeigen.

[.instruction]
Exportformate filtern:

. Öffnen Sie das Menü **Datenaustausch » Export » Elastischer Export**.
. Nehmen Sie die Einstellungen im Bereich **Filter** vor. Beachten Sie dazu die Erläuterungen in Tabelle 4.
. Klicken Sie auf **Suchen**.

[cols="a,a"]
|====
|Filter |Erläuterung

|**ID**
|Zahl eingeben.

|**Name**
|Namen bzw. einen Teil des Namens eingeben.

|**Typ**
|Typ **Artikel** oder **Alle** aus dem Dropdown-Menü wählen.

|**Format**
|Format aus dem Dropdown-Menü wählen, damit alle Exportformate, die zu dem gewählten Exportformat erstellt wurden, angezeigt werden.
|====

__Tab. 4: Filtereinstellungen im Menü **Datenaustausch » Export » Elastischer Export**__

[NOTE]
.Suchkriterien löschen
====
Mit einem Klick auf **Reset** löschen Sie die Suchkriterien. Klicken Sie anschließend erneut auf **Suchen**, damit alle bereits erstellten Exportformate wieder in der Übersicht angezeigt werden und Sie einen neuen Filter einstellen können.
