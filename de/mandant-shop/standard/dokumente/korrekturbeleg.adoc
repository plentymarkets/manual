Ein Korrekturbeleg wird verwendet, um eine Erstattung, z.B. von einer Storno-Rechnung zu belegen. plentymarkets bietet Ihnen vorkonfigurierte Vorlagen, die Sie beliebig anpassen können. Das Dokument **Korrekturbeleg** konfigurieren und verwalten Sie sprachbezogen im Ordner **Dokumente** eines Standorts unter **plentymarkets » Einstellungen » Mandant (Shop) » Shop wählen » Standorte » Standort wählen**.

== Korrekturbelege einrichten

Die Konfiguration umfasst die Einrichtung der Vorlage im Menü **Dokumente** eines Standorts, die Übernahme oder Übertragung von Eigenschaften, das optionale Hinterlegen einer PDF-Vorlage und das Einrichten eines Nummernkreises.

=== Standard-Einstellungen übernehmen

Im Tab **Transfer** besteht die Möglichkeit, Einstellungen aus einer Standard-Vorlage in eine Vorlage zu übernehmen oder Einstellungen der Vorlage auf ein anderes Dokument, z.B. den Lieferschein, zu übertragen. Dies ist z.B. sinnvoll, wenn die Vorlage noch keine Einstellungen enthält. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftraege/einstellungen/dokumente#140, **Einstellungstransfer** >>.

=== Vorlage einrichten

Damit ein Korrekturbeleg erzeugt werden kann, muss zuerst eine Vorlage eingerichtet werden. Details zur Konfiguration finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftraege/einstellungen/dokumente#40, **Dokumentenvorlage konfigurieren** >>.

[IMPORTANT]
.Positionierung durch Koordinaten
====
Die Inhalte der Vorlage werden durch die Eingabe der **Koordinaten X** und **Y** positioniert. Dabei wird als Ausgangspunkt die obere linke Ecke einer Seite angenommen.
====

=== Einstellungen übertragen

Ebenfalls im Tab **Transfer** können Sie die Einstellungen des Tabs **Vorlage** eines Korrekturbelegs auf ein anderes Dokument, z.B. den Lieferschein, übertragen. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftraege/einstellungen/dokumente#140, **Einstellungstransfer** >>.

=== PDF-Vorlagen hinterlegen

Wenn Sie eine Rechnungsvorlage erstellt haben, können Sie im Tab **PDF-Vorlage** eine PDF-Datei z.B. mit den Daten Ihres Shops inklusive Logo hinterlegen. Diese PDF-Datei müssen Sie zunächst auf Ihrem Rechner speichern. Beachten Sie dazu die Handbuchseite **Dokumente**, Kapitel <<auftraege/einstellungen/dokumente#170, **PDF-Vorlage** >>.

=== Nummernkreise erstellen

Für die Gestaltung von Nummernkreisen bietet die Dokumentenkonfiguration verschiedene Möglichkeiten im Tab **Nummernkreis**. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftraege/einstellungen/dokumente#200, **Nummernkreise** >>.

== Korrekturbelege generieren

Korrekturbelege erstellen Sie manuell oder lassen diese automatisch über eine Ereignisaktion erzeugen.

=== Korrekturbeleg manuell erstellen

Einen Korrekturbeleg erstellen Sie in einer Gutschrift im Tab **Belege**.

image::mandant-shop/standard/dokumente/assets/DE-Standort-Dokumente-Korrekturbeleg-01.png[]

__Bild 1: Korrekturbeleg manuell erstellen__

[.instruction]
Korrekturbeleg manuell erstellen:

. Öffnen Sie die Gutschrift im Menü **Aufträge » Aufträge bearbeiten**.
. Wechseln Sie in das Tab **Belege**.
. Wählen Sie im Dropdown-Menü **Beleg erstellen** die Option **Korrekturbeleg**. +
→ Ein Fenster zur Bearbeitung der Belegdaten wird geöffnet.
. Nehmen Sie bei Bedarf Änderungen vor oder tragen Sie einen Kommentar ein.
. **Speichern** Sie die Einstellungen. +
→ Der Korrekturbeleg wird als PDF geöffnet und in der Auflistung (Bild 1) mit Datum angezeigt.

=== Korrekturbelege automatisch erzeugen

Richten Sie eine **<<auftraege/einstellungen/ereignis-aktionen#, Ereignisaktion  >>** ein, um Korrekturbelege automatisch zu erzeugen. Die Gutschriften können Sie z.B. durch eine Markierung kennzeichnen, die Sie dann in der Ereignisaktion als **Filter** einstellen.

image::mandant-shop/standard/dokumente/assets/DE-Standort-Dokumente-Korrekturbeleg-02.png[]

__Bild 2: Korrekturbelege per Ereignisaktion erzeugen__

[.collapseBox]
.Ereignisaktion einrichten
--
. Öffnen Sie das Menü **Einstellungen » Aufträge » Ereignisaktionen**.
. Klicken Sie auf **Ereignisaktion hinzufügen**. +
→ Das Fenster **Neue Ereignisaktion erstellen** wird geöffnet.
. Tragen Sie einen Namen ein.
. Wählen Sie das **Ereignis** gemäß Tabelle 1.
. **Speichern** Sie die Einstellungen.
. Nehmen Sie die Einstellungen gemäß Tabelle 1 vor.
. Setzen Sie ein Häkchen bei **Aktiv**.
. **Speichern** Sie die Einstellungen.
--
[cols="a,a,a"]
|====
|Einstellung |Option |Auswahl

|**Ereignis**
|**Auftragsanlage &gt; Neue Gutschrift**
|

|**Filter 1**
|**Auftrag &gt; Auftragstyp**
|**Gutschrift**

|**Filter 2**
|**Auftrag &gt; Markierung**
|Markierung wählen

|**Aktion 1**
|**Dokumente &gt; Korrekturbeleg erzeugen** +
**__Optional:__** Hinweis in Textfeld eintragen (Bild 2).
|

|**Aktion 2**
|**Auftrag &gt; Markierung ändern**
|Neue Markierung wählen
|====
