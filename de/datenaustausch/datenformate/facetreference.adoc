= FacetReference
:lang: de
include::{includedir}/_header.adoc[]
:position: 105

Das Datenformat **FacetReference** wird für die Bearbeitung der Verknüpfungen der Facettensuche verwendet. Sie erstellen und exportieren das Datenformat im Menü **Datenaustausch » Export » Dynamischer Export**. Importieren Sie das Datenformat wieder im Menü **Datenaustausch » Import » Dynamischer Import**. Wie Sie das Datenformat exportieren und importieren, ist auf den Handbuchseiten **<<datenaustausch/daten-exportieren#, Daten exportieren  >>** und **<<datenaustausch/daten-importieren#, Daten importieren  >>** beschrieben. Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.

[IMPORTANT]
.Modul aktiv?
====
Das Datenformat ist nur sichtbar, wenn das Modul Facettensuche gebucht wurde und aktiv ist.
====
