= Customer
:lang: de
include::{includedir}/_header.adoc[]
:keywords: Datenformat Customer, Kundendstammdaten
:description: Mit dem Datenformat Customer exportieren, bearbeiten und importieren Sie Kundenstammdaten.
:position: 50

Das Datenformat **Customer** wird verwendet, um Kundenstammdaten zu exportieren, zu bearbeiten und zu importieren. Sie erstellen und exportieren das Datenformat im Menü **Datenaustausch » Export » Dynamischer Export**. Importieren Sie das Datenformat wieder im Menü **Datenaustausch » Import » Dynamischer Import**. Wie Sie das Datenformat exportieren und importieren, ist auf den Handbuchseiten **<<datenaustausch/daten-exportieren#, Daten exportieren  >>** und **<<datenaustausch/daten-importieren#, Daten importieren  >>** beschrieben.

Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.

== Übersicht der Pflichtfelder

Die folgenden Pflichtfelder müssen für das Datenformat **Customer** vorhanden sein und ein gültiger Wert muss eingegeben sein:

* **City**
* **FirstName**
* **HouseNo**
* **LastName**
* **Street**
* **ZIP**

== Übersicht der Abgleichfelder

Das folgende Datenfeld steht zum <<datenaustausch/daten-importieren#25., **Datenabgleich** >> zur Verfügung. Bei diesem Pflichtabgleichfeld muss für die **Importaktion** die Option **Abgleich** gewählt werden.

* **CustomerID** = Kunden-ID

== Übersicht der Aktionen

Im Tab **Aktionen** wählen Sie, was beim Datenexport ausgeführt werden soll. Aktivieren Sie die Aktion **LastUpdate** und wählen Sie die Option **setze aktuelles Datum**. Den Kundendaten wird dann beim Import das Datum und die Uhrzeit der letzten Aktualisierung hinzugefügt.
