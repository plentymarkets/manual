= Datenformate
:lang: de
include::{includedir}/_header.adoc[]
:keywords: Datenformate, dynamischer Export, dynamischer Import
:description: Erfahren Sie, welche Datenformate Ihnen für den dynamischen Import und Export in plentymarkets zur Verfügung stehen.
:position: 16

Auf dieser Handbuchseite finden Sie alle in plentymarkets verfügbaren Datenformate sowie eine kurze Erläuterung. Die meisten dieser Datenformate können sowohl für den Export als auch für den Import verwendet werden. Erstellen Sie die Formate zunächst im Menü **Datenaustausch » Export » Dynamischer Export**. Exportieren Sie die entsprechenden Daten, um sie zu bearbeiten und importieren Sie diese anschließend wieder im Menü **Datenaustausch » Import » Dynamischer Import**.

Mithilfe der Exporte können Sie Daten zu Artikeln, Kunden, Aufträgen, Umsätzen etc. exportieren. Für den Export von Umsatzdaten geben Sie zunächst alle Buchhaltungsdaten im Menü <<mandant-shop/standard/buchhaltung#, **Einstellungen » Mandant (Shop) » Standard » Standorte » Standort wählen » Buchhaltung**  >> ein, damit plentymarkets die **<<mandant-shop/standard/buchhaltung#, Buchungsschlüssel  >>** erzeugen kann.

Erfahren Sie mehr über die Datenformate im folgenden Videotutorial:

video::138182973[vimeo]

Tabelle 1 listet alle in plentymarkets verfügbaren Datenformate auf. Wenn nicht anders angegeben, können Sie die Datenformate sowohl für den Import als auch für den Export verwenden.

Auf den Unterseiten der Datenformate finden Sie eine Auflistung der Datenfelder, deren Erläuterung und mögliche Filteroptionen und Aktionen. Außerdem stehen Angaben zu Pflichtfeldern und Pflichtabgleichfeldern zur Verfügung.

[IMPORTANT]
.Auf exakte Schreibweise und Pflichtfelder achten
====
Achten Sie bei allen Datenfeldern auf die exakte Schreibweise, da die Daten sonst nicht verarbeitet werden können. +
Pflichtfelder müssen vorhanden sein und ein gültiger Wert muss eingetragen sein. Pflichtabgleichfelder müssen für den Datenabgleich eingestellt werden. Bei einigen Abgleichfeldern stehen alternative Abgleichfelder zur Verfügung.
====
