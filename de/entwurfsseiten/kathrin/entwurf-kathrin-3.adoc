== Monatsschlussbeleg erstellen

Um rechtssicher gemäß RKSV zu arbeiten, muss am Ende jedes Monats ein Monatsschlussbeleg erstellt werden. Der Monatsbeleg des Monats Dezember dient gleichzeitig als Jahresabschlussbeleg.

=== Sammelbeleg erstellen

Ein Sammelbeleg muss erstellt werden, wenn die Signatureinrichtung ausgefallen ist.

==== Nullbeleg erstellen

Sie müssen auf Verlangen von Mitarbeitenden der österreichischen Finanzverwaltung jederzeit einen Nullbeleg vorlegen können.

Wenn Sie eine Kasse dauerhaft außer Betrieb nehmen, schreibt das RKSV vor, dass ein Schlussbeleg erstellt werden muss. Nachdem Sie diesen Schlussbeleg erstellt haben, kann die Kasse nicht mehr verwendet werden.
