[WARNING]
.Telefonnummer des Empfängers angeben
====
Geben Sie in den Absenderdaten die Telefonnummer des Empfängers generell auf Sendungen ins Ausland an.

Beachten Sie, dass die Schnittstelle seitens UPS auch bei Sendungen innerhalb Deutschlands die Angabe der Telefonnummer des Empfängers fordert. Damit die Sendung angemeldet wird, übermittelt plentymarkets für Sie daher im Hintergrund einen Dummy-Wert an UPS, wenn keine nationale Telefonnummer angegeben ist.
====

== Hinweistext für die UI

Hinweis: Die Schnittstelle seitens UPS fordert auch bei Sendungen innerhalb Deutschlands die Angabe der Empfängertelefonnummer. Damit die Sendung angemeldet wird, übermittelt plentymarkets für Sie im Hintergrund einen Dummy-Wert an UPS, wenn keine nationale Telefonnummer angegeben ist.

Merkmale an DHL Fulfillment übermitteln[#5300]

Erstellen Sie im Menü **Einstellungen » Artikel » Merkmale** Eigenschaften wie z.B. "Zerbrechlich" oder "Originalkarton", die Sie anschließend an DHL Fulfillment übermitteln, damit die Artikel logistisch entsprechend vorbereitet werden können. Wie Sie ein Merkmal erstellen, ist auf der Handbuchseite <<artikel/artikel-verwalten#, **Artikel verwalten**  >> im Kapitel <<artikel/artikel-verwalten#, **Merkmale erstellen**  >> beschrieben. Die folgenden Merkmale können an DHL Fulfillment übermittelt werden:

* **Mindesthaltbarkeitsdatum**: Der Artikel wird als MHD-pflichtiger Artikel im Lager bei DHL Fulfillment geführt. Beim Wareneingang erfasst DHL Fulfillment für Artikel mit diesem Merkmal ein Mindesthaltbarkeitsdatum.
* **Originalkarton**: Der Artikel wird von DHL Fulfillment einzeln im Originalkarton versendet.
* **Zerbrechlich**: Der Artikel wird von DHL Fulfillment mit zusätzlichem Füllmaterial am Packplatz verpackt.
* **Seriennummer**: Die am Packplatz erfassten Seriennummern werden mit der Versandmeldung (Auftragsrückmeldung) an plentymarkets von DHL Fulfillment zurückgespielt, damit die Seriennummer z.B. auf die Rechnung gedruckt und vom Kunden für weitere Verwendungen (z.B. Dokumentation, Gutscheinfreigabe) verwendet werden kann.

[IMPORTANT]
.Korrekten Merkmaltyp für die Merkmale wählen
====
Erstellen Sie im Menü **Einstellungen » Artikel » Merkmale** die folgenden drei Merkmale mit dem Merkmaltyp **Kein**:

* **Mindesthaltbarkeitsdatum**
* **Originalkarton**
* **Zerbrechlich**

Erstellen Sie das Merkmal **Seriennummer** mit dem Merkmaltyp **Auswahl** und speichern Sie die folgenden drei Werte:

* **SN** (Erfassung der Seriennummer eines Artikels)
* **SNK** (Erfassung von Seriennummernbereichen pro Artikel)
* **SNK2** (Erfassung von Seriennummernbereichen pro Auftragsposition)

Weitere Informationen zu Merkmalen finden Sie auf der Handbuchseite <<artikel/artikel-verwalten#, **Artikel verwalten**  >> im Kapitel <<artikel/artikel-verwalten#480, **Merkmale verwalten** >>.
====

Nachdem Sie die Merkmale erstellt haben, <<artikel/artikel-verwalten#, **aktivieren**  >> Sie die Merkmale am Artikel im Tab **Merkmale**. Abschließend ordnen Sie die Merkmale in der Schnittstellenkonfiguration im Menü **Einstellungen » Aufträge » Versand » Fulfillment-Dienstleister » DHL Fulfillment** zu.

Vor der Übertragung des Auftrags an DHL Fulfillment findet eine Prüfung der Empfängeradresse auf Richtigkeit der Form und Struktur statt, d.h. plentymarkets prüft z.B., ob alle Pflichtfelder wie beispielsweise der Name ausgefüllt wurden. +
Sollten fehlerhafte HTML-Zeichen bzw. für die API ungültige Zeichen in der Adresse enthalten sein, werden diese korrigiert. Wenn dennoch ein nicht korrigierbarer Fehler in der Adresse enthalten ist, wird der Auftrag nicht übermittelt. Dies verhindert eine Störung im Prozess, da DHL Fulfillment keine ungültigen bzw. fehlerhaften XML-Dateien von dem FTP-Server abholen kann.
