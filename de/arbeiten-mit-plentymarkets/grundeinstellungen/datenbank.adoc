In diesem Menü finden Sie zwei Aktionen mit denen Sie große Datenmengen in der Datenbank ändern oder löschen. Führen Sie diese Aktionen nur nach sorgfältiger Prüfung durch, da Ihre Daten unwiederbringlich gelöscht werden.

toc::[]
== Löschen ungenutzter Artikelkategorien

Mit dieser Funktion können Sie **ungenutzte Artikelkategorien** löschen, dabei werden keine Artikeldaten berührt.

__Bild 1: **Löschen ungenutzter Artikelkategorien**__

== Löschen sämtlicher Warenbestände
