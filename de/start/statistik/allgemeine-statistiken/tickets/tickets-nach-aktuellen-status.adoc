== Tickets nach aktuellen Status

Diese Statistik liefert eine Auswertung der **Tickets nach aktuellen Status** gemäß der gewählten **Ticket-Typen** und der gewählten **Ticket-Status** zum Zeitpunkt der Statistikerstellung.

== Statistik-Details

Es werden die **Ticket-Typen****statusbezogen** und die **Anzahl der Tickets** in diesem Status in verschiedenen Farben angezeigt. Als **Tooltip** wird die Ticketanzahl auch prozentual im Verhältnis zu allen angezeigten Tickets wiedergegeben. Wenn zu einem Status keine Tickets vorhanden sind, wird der jeweilige Status nicht angezeigt. Sie können mit dieser Statistik z.B. erkennen, wie hoch die **aktuelle Anzahl der Tickets eines Typs** im jeweiligen Status ist.

[NOTE]
.**Details anzeigen (Tooltip)**
====
Zeigen Sie mit dem **Mauszeiger** auf ein Element der Statistik, um die jeweiligen **Details** anzuzeigen.
====

[TIP]
.Tipp
====
In Bild 1 sehen Sie z.B. verschiedene Ticket-Typen, ihren derzeitigen Status und die Anzahl der Tickets, die sich aktuell in diesem Status befinden.
====

== Einstellungen

Die nachfolgend beschriebenen **Einstellungen** stehen für diese Statistik zur Verfügung.

image::start/statistik/allgemeine-statistiken/tickets/assets/DE-Statistik-Tickets-aktuelle-Status-02-SI.png[]

__Bild 2: Statistik **Tickets nach aktuellen Status**; **Einstellungen**__

Beachten Sie die Erläuterungen in der folgenden Tabelle:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|In diesem Dropdown-Menü wählen Sie die Art der Darstellung aus. +
**2D-Kreisdiagramm** = Darstellung der Werte als flächige Kreissektoren +
**3D-Kreisdiagramm** = Darstellung der Werte als dreidimensionale Kreissektoren +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine **CSV-Datei** exportiert. Dabei werden die Parameter wie angezeigt verwendet, sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|
|Einstellungen

|**Name**
|Hier wird der **Name** bzw. die **Bezeichnung** der Statistik angezeigt, die beim Erstellen einer neuen Statistik gewählt wurde. Sie können diesen Namen hier ändern.

|**Status**
|Hier stehen Ihnen die von Ihnen angelegten **Ticket-Typen**, mit den dazugehörigen **Status** für die Statistik zur Auswahl. Standardmäßig sind alle Typen und Status ausgewählt.

|**Kunden-ID**
|Für eine Statistik der Ticketdaten eines **einzelnen Kunden** geben Sie hier dessen **ID** ein.

|**Kundenklasse**
|Wählen Sie die **Kundenklasse(n)**, deren Daten für die Statistik verwendet werden sollen.

|**Priorität**
|Wählen Sie die **Ticket-Priorität(en)**, die in der Statistik berücksichtigt werden sollen.

|**Eigner**
|Filtern Sie hier die auszuwertenden Tickets nach ihrem Eigner.

|**Rolle der Eigner**
|Filtern Sie hier die auszuwertenden Tickets nach der Rolle der Eigner.
|====
