== Wareneingang nach Lieferant

Auswertung des __**Wareneingangs nach Lieferant**__ Ihrer __**Artikel**__ gemäß dem/den gewählten __**Lager/n**__ sowie dem/den __**Lieferanten**__ innerhalb einer gewählten Zeitspanne. Standardeinstellung der Zeitspanne: **Die letzten 24 Stunden**.

== Statistik-Details

Es wird die __**Anzahl der Artikel**__ und der __**Warenwert in Euro**__ des Wareneingangs nach gewähltem/n __**Lieferanten**__ angezeigt. Als Tooltip wird der genaue __**Warenwert**__ und der jeweils zugehörige __**Lieferant**__ angezeigt, außerdem in einer anderen Farbe die genaue __**Artikelmenge**__, mit der der jeweilige Warenwert erzielt wird (Bild 1). Sie können mit dieser Statistik z.B. erkennen, wie hoch der Warenwert und die Artikelmenge für jede Ihrer Lieferanten ist.

In Bild 1 wird z.B. für den __**Lieferant unidentified supplier**__ ein __**Netto-Warenwert**__ von __**106,92 Euro**__ angezeigt, der mit __**12 Artikeln**__ generiert wurde.

[NOTE]

Zeigen Sie mit dem __**Mauszeiger**__ auf ein Element der Statistik, um die jeweiligen __**Details**__ anzuzeigen.

== Einstellungen

Legen Sie zunächst wie auf der Handbuchseite **<<start/statistik#, Statistik  >>** beschrieben eine neue Statistik an. Für die Statistik **Wareneingang nach Lieferant** stehen dann die in Tabelle 1 beschriebenen __**Einstellungen**__ zur Verfügung.

image::start/statistik/allgemeine-statistiken/lager/assets/DE-Statistik-Wareneingang-Lieferant02-SI.png[]

__Bild 2: Statistik **Wareneingang nach Lieferant**; **Einstellungen**__

Die folgende Tabelle erläutert die Einstellungen im Menü **Start » Statistik → Statistik-Typ: Lager / Wareneingang nach Lieferant**:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|**Doppel-Säulendiagramm** = Darstellung der Werte als Säulen +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine __**CSV-Datei**__ exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Es wird ebenfalls eine __**CSV-Datei**__ erzeugt. Die Daten werden allerdings detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Die __**Bezeichnung**__ der Statistik, die beim Erstellen gewählt wurde. Der Name kann hier geändert werden.

|**Zeitspanne**
|Eine Zeitspanne für die Statistik aus dem Dropdown-Menü wählen. Standardwert: **Die letzten 24 Stunden**. +
Die Zeitspanne ändern, wenn keine oder nur wenige Daten angezeigt werden. Dann auf **Speichern** klicken, um das Ergebnis anzuzeigen. +
Durch Wahl eines __**längeren Zeitraums**__ werden mehr Informationen angezeigt. Bei vielen Daten ermöglicht die __**Verringerung der Zeitspanne**__ ggf. eine aussagekräftigere Auswertung.

|**Lager**
|Die __**Lager**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Lieferant**
|Die __**Lieferanten**__ wählen, deren Daten für die Statistik verwendet werden sollen.
|====

__Tab. 1: **Statistik bearbeiten**; **Einstellungen**__

== Daten-Export

Wie in Tabelle 1 erläutert können Sie zwischen einem einfachen und einem detaillierten Exportergebnis wählen. Nachfolgend finden Sie entsprechende Beispiele.

=== Export der Diagrammdaten

[IMPORTANT]
.Datenfelder
====
**SupplierID** = Lieferanten-ID, **QuantityOfItems** = Artikelmenge, **NetItemValue** = Netto-Warenwert, **SupplierName** = Name des Lieferanten
====

=== Export der Rohdaten

[IMPORTANT]
.Datenfelder
====
**SupplierID** = Lieferanten-ID, **PriceSetID** = ID des Preissets des Artikels, **ItemID** = Artikel-ID, **AttributeValueSetID** = ID der Artikelvariante, **SupplierName** = Name des Lieferanten, **Quantity** = Anzahl der Wareneingänge, **PurchasePrice** = Einkaufspreis
====
