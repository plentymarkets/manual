== Versandkosten nach Versanddienstleister

Die __**Versandkosten**__ werden gesplittet nach __**Warenausgangsdatum**__ angezeigt. Je nach Zeitspanne werden diese tage- oder monatsweise in __**Prozent**__ pro Versanddienstleister angezeigt.

== Statistik-Details

Entsprechend der gewählten Zeitspanne werden die __**Versandkosten**__ und das __**Datum des Warenausgangs**__ datumsbezogen angezeigt. Sind keine Aufträge vorhanden, wird kein Balken angezeigt. Als Tooltip wird die __**Versandkosten pro Versanddienstleister**__ und der __**prozentuale Anteil des jeweiligen Versanddienstleisters**__ an den jeweiligen Gesamtkosten zum jeweiligen __**Datum**__ angezeigt (Bild 1). Sie können mit dieser Statistik z.B. die Verteilung Ihrer Versandkosten auf die Versanddienstleister erkennen.

In Bild 1 wird z.B. für __**Oktober 2012**__ angezeigt, dass über den Versanddienstleister __**Deutsche Post**__ Versandkosten von __**48,00 Euro**__ bezahlt wurden, über den Versanddienstleister __**DHL**__ ein Betrag von __**3,98 Euro**__ und über __**Selbstabholung**__ sind Versandkosten in Höhe von __**432,00 Euro**__ bezahlt worden.

== Einstellungen

Legen Sie zunächst wie auf der Handbuchseite **<<start/statistik#, Statistik  >>** beschrieben eine neue Statistik an. Für die Statistik **Versandkosten nach Versanddienstleister** stehen dann die in Tabelle 1 beschriebenen __**Einstellungen**__ zur Verfügung.

image::start/statistik/allgemeine-statistiken/umsatz/assets/DE-Statistik-Versandkosten-02-SI.png[]

__Bild 2: Statistik **Versandkosten nach Versanddienstleister**; **Einstellungen**__

Die folgende Tabelle erläutert die Einstellungen im Menü **Start » Statistik → Statistik-Typ: Umsatz / Umsatz Versandkosten nach Versanddienstleister**:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|**Gestapeltes-Säulendiagramm mit 100%** = Darstellung der Werte als Säulen. Die Anteile der Versandkosten sind in verschiedenen Farben gestapelt dargestellt. +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine __**CSV-Datei**__ exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Es wird ebenfalls eine __**CSV-Datei**__ erzeugt. Die Daten werden allerdings detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Die __**Bezeichnung**__ der Statistik, die beim Erstellen gewählt wurde. Der Name kann hier geändert werden.

|**Zeitspanne**
|Eine Zeitspanne für die Statistik aus dem Dropdown-Menü wählen. Standardwert: **Die letzten 6 Monate**. +
Die Zeitspanne ändern, wenn keine oder nur wenige Daten angezeigt werden. Dann auf **Speichern** klicken, um das Ergebnis anzuzeigen. +
Durch Wahl eines __**längeren Zeitraums**__ werden mehr Informationen angezeigt. Bei vielen Daten ermöglicht die __**Verringerung der Zeitspanne**__ ggf. eine aussagekräftigere Auswertung.

|**Startdatum / Enddatum**
|Optional einen bestimmten __**Datumsbereich**__ alternativ zur Option **Zeitspanne** eingrenzen.

|**Mandant (Shop)**
|Die __**Webshops**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Auftragsherkunft**
|Die __**Auftragsherkünfte**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Versandregion**
|Die __**Versandregion**__ wählen, die für die Statistik berücksichtigt werden soll. **Alle** wählen, um alle Versandregionen einzubeziehen.

|**Versandkosten-Quelle**
|Wählen, ob die __**Quelle für die Versandkosten**__ der __**Hauptauftrag**__ oder ein __**Lieferauftrag**__ sein soll.

|**Amazon-Konto**
|Das __**Amazon-Konto**__ wählen, das für die Statistik berücksichtigt werden soll.

|**eBay-Konto**
|Das __**eBay-Konto**__ wählen, das für die Statistik berücksichtigt werden soll.
|====

.Datenfelder
====
**ShippingProviderID** = ID des Versanddienstleisters, **ShippingProviderName** = Name des Versanddienstleisters, **ShippingCosts** = Betrag in Euro, **Percentage** = Prozentualer Anteil, **Date** = Datum
