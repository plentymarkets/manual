== Warenkorbentwicklung

Diese Statistik zeigt die durchschnittliche __**Warenkorbwert**__ und die durchschnittliche __**Versandkosten**__ in einer gewählten Zeitspanne. Standardeinstellung der __**Zeitspanne**__: Warenkorbwerte der **letzten 30 Tage**.

== Statistik-Details

Entsprechend der gewählten Zeitspanne wird der __**Warenkorbwert**__ und die __**Versandkosten**__ datumsbezogen angezeigt. Sind keine Warenkorbwerte vorhanden, wird kein Balken angezeigt. Als Tooltip wird der __**Warenkorbwert**__ und das zugehörige __**Datum**__ angezeigt, außerdem in einer anderen Farbe die __**Versandkosten**__, der mit dem jeweiligen Warenkorbwert erzielt wurden (Bild 1). Sie können mit dieser Statistik z.B. erkennen, ob der durchschnittliche Warenkorbwert in einem Monat oder zu einem bestimmten Zeitpunkt, wie der Vorweihnachtszeit oder während eines Saisonschlussverkaufs, höher bzw. niedriger war, als zu einem anderen Zeitpunkt.

In Bild 1 wird z.B. für __**August 2012**__ ein __**durchschnittlicher Warenkorbwert**__ von __**171,97 Euro**__ angezeigt, mit durchschnittlichen __**Versandkosten**__ von __**2,88 Euro**__.

[NOTE]

====
Zeigen Sie mit dem __**Mauszeiger**__ auf ein Element der Statistik, um die jeweiligen __**Details**__ anzuzeigen.
====

== Einstellungen

Legen Sie zunächst wie auf der Handbuchseite **<<start/statistik#, Statistik  >>** beschrieben eine neue Statistik an. Für die Statistik **Warenkorbentwicklung** stehen dann die in Tabelle 1 beschriebenen __**Einstellungen**__ zur Verfügung.

image::start/statistik/allgemeine-statistiken/umsatz/assets/DE-Statistik-Warenkorbentwicklung-02-SI.png[]

__Bild 2: Statistik **Warenkorbentwicklung**; **Einstellungen**__

Die folgende Tabelle erläutert die Einstellungen im Menü **Start » Statistik → Statistik-Typ: Umsatz / Warenkorbentwicklung**:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagrammtyp**
|**Gestapeltes-Säulendiagramm** = Darstellung der Werte als Säulen. Die Warenkorbwerte (blau) sind unten und die Versandkosten (rot) sind darauf gestapelt. +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine __**CSV-Datei**__ exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Es wird ebenfalls eine __**CSV-Datei**__ erzeugt. Die Daten werden allerdings detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Die __**Bezeichnung**__ der Statistik, die beim Erstellen gewählt wurde. Der Name kann hier geändert werden.

|**Zeitspanne**
|Eine Zeitspanne für die Statistik aus dem Dropdown-Menü wählen. Standardwert: **Die letzten 30 Tage**. +
Die Zeitspanne ändern, wenn keine oder nur wenige Daten angezeigt werden. Dann auf **Speichern** klicken, um das Ergebnis anzuzeigen. +
Durch Wahl eines __**längeren Zeitraums**__ werden mehr Informationen angezeigt. Bei vielen Daten ermöglicht die __**Verringerung der Zeitspanne**__ ggf. eine aussagekräftigere Auswertung.

|**Startdatum / Enddatum**
|Optional einen bestimmten __**Datumsbereich**__ alternativ zur Option **Zeitspanne** eingrenzen.

|**Mandant (Shop)**
|Die __**Webshops**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Auftragsherkunft**
|Die __**Zahlungsart**__ wählen, die für die Statistik berücksichtigt werden soll. **Alle** wählen, um alle Zahlungsarten einzubeziehen.

|**Zahlungsart**
|Die __**Zahlungsart**__ wählen, die für die Statistik berücksichtigt werden soll. **Alle** wählen, um alle Zahlungsarten einzubeziehen.

|**Versandprofil**
|Das __**Versandprofil**__ wählen, das für die Statistik berücksichtigt werden soll. **Alle** wählen, um alle Versandprofile einzubeziehen.

|**Auftragseigner**
|Der __**Auftragseigner**__ wählen, der für die Statistik berücksichtigt werden soll. **Alle** wählen, um alle Auftragseigner einzubeziehen.

|**Diagramm**
|Wählen, ob __**Umsätze**__ in der Statistik als __**Brutto**__ oder __**Netto**__ angezeigt werden sollen.

|**Amazon-Konto**
|Das __**Amazon-Konto**__ wählen, das für die Statistik berücksichtigt werden soll.

|**eBay-Konto**
|Das __**eBay-Konto**__ wählen, das für die Statistik berücksichtigt werden soll.
|====

__Tab. 1: **Statistik bearbeiten**; **Einstellungen**__

== Daten-Export

Wie in Tabelle 1 erläutert können Sie zwischen einem einfachen und einem detaillierten Exportergebnis wählen. Nachfolgend finden Sie entsprechende Beispiele.

=== Export der Diagrammdaten

[IMPORTANT]
.Datenfelder
====
**Date** = Datum, **QuantityOfOrders** = Anzahl der Aufträge, **ShippingCosts** = Versandkosten, **AverageShipping** = durchschnittliche Versandkosten, **AverageSales** = durchschnittliche Umsätze, **Sales** = Umsatz
====
