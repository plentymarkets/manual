==  Allgemeines zu Artikelmengen / Retouren

In dieser Statistik werden für eine bestimmte Zeitspanne die Artikel mit der __**Anzahl der Verkäufe**__, der __**Anzahl der Retouren**__ sowie deren __**Verhältnis**__ in Prozent angezeigt. Außerdem werden die __**Verkaufs- und Retourenumsätze in Euro**__ angegeben. Standardeinstellungen der Zeitspanne: **Die letzten 6 Monate**.

==  Statistik-Details

Entsprechend der gewählten Zeitspanne werden die __**verkaufte Artikelmenge**__ und die __**Anzahl der Retouren**__ sowie die entsprechenden Angaben zum __**Umsatz in Euro**__ angezeigt. Anhand der __**Retouren-Rate**__können Sie das Verhältnis zwischen Verkaufsmenge und Retoure eines Artikels einsehen. Es werden ungefiltert alle Artikel angezeigt, das heißt auch solche, für die noch keine Retouren vermerkt wurden.

[TIP]
.Tipp
====
Als Tooltip wird in den Spaltenüberschriften der Daten-Tabelle ein **Zahnrad** angezeigt. Klicken Sie darauf, um zu wählen, welche Daten für Ihre Statistik relevant sind und angezeigt werden sollen. Entfernen Sie den Haken bei den Daten, Sie nicht benötigen.
====

==  Einstellungen

Legen Sie zunächst wie auf der Handbuchseite **<<start/statistik#, Statistik  >>** beschrieben eine neue Statistik an. Für die Statistik **Artikelmengen / Retouren** stehen dann die in Tabelle 1 beschriebenen __**Einstellungen**__ zur Verfügung.

image::start/statistik/allgemeine-statistiken/artikel/assets/Statistik-Artikel-ArtikelmengenRetouren02-SI.png[]

__Bild 2: Statistik **Artikelmengen / Retouren**; **Einstellungen**__

Die folgende Tabelle erläutert die Einstellungen im Menü **Start » Statistik → Statistik-Typ: Artikel / Artikelmengen / Retouren**:

[cols="a,a"]
|====
| Einstellung | Erläuterung

|
| Diagramm-Anzeige

|**Diagrammtyp**
|**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
| Die Daten des Diagramms werden in eine __**CSV-Datei**__ exportiert. Dabei werden die Parameter wie angezeigt verwendet. Sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
| Es wird ebenfalls eine __**CSV-Datei**__ erzeugt. Die Daten werden allerdings detaillierter exportiert. Jeder Artikel wird einzeln aufgeführt.

|
| Einstellungen

|**Name**
| Die __**Bezeichnung**__ der Statistik, die beim Erstellen gewählt wurde. Der Name kann hier geändert werden.

|**Zeitspanne**
| Eine Zeitspanne für die Statistik aus dem Dropdown-Menü wählen. Standardwert**: Die letzten 6 Monate**. +
Die Zeitspanne ändern, wenn keine oder nur wenige Daten angezeigt werden. Dann auf **Speichern** klicken, um das Ergebnis anzuzeigen. +
Durch Wahl eines __**längeren Zeitraums**__ werden mehr Informationen angezeigt. Bei vielen Daten ermöglicht die __**Verringerung der Zeitspanne**__ ggf. eine aussagekräftigere Auswertung.

|**Startdatum / Enddatum**
| Optional einen bestimmten __**Datumsbereich**__ alternativ zur Option **Zeitspanne** eingrenzen.

|**Datumsgrundlage**
| Wählen, anhand welchen Ereignisses die Daten ausgewertet werden sollen: **Auftragseingang**, **Zahlungseingang** oder **Warenausgang**.

|**Von Status / Bis Status**
| Je nach Statistiktyp stehen zur Eingrenzung des **Status** verschiedene Optionen zur Verfügung. Es sind jeweils __**Standardwerte**__ voreingestellt. +
Für die Auswertung der Statistik werden __**alle Auftragsarten**__ berücksichtigt außer __**Stornierungen**__. Somit werden die __**Status 8**__ und __**8.1**__ nicht in die Auswertung einbezogen. __**Retouren**__und __**Gutschriften**__werden nur berücksichtigt, wenn sie per Einstellung eingebunden wurden, siehe Option **Einbindung Retouren/Gutschriften**.

|**Mandant (Shop)**
| Die __**Webshops**__ wählen, deren Daten für die Statistik verwendet werden sollen.

|**Artikelherkunft**
| Eine oder mehrere **Auftragsherkünfte** oder mit der Option **Alle** sämtliche verfügbaren wählen.

|**Unterkategorien anzeigen für**
| Bei Bedarf wählen, für welche Artikelkategorie auch die Unterkategorien angezeigt werden sollen.

|**Markierung 1 und 2**
| Einstellen, ob **alle Artikel**, nur Artikel mit einer **bestimmten Markierung** oder nur Artikel **ohne Markierung** in der Statistik berücksichtigt werden sollen. Standardmäßig sind **alle Artikel** eingestellt. Es können höchstens zwei Markierungen auf einmal gewählt werden.

|**Artikelpaket anzeigen**
| Häkchen setzen, um zusätzlich zu den Einzelartikeln auch die Artikelpakete anzuzeigen.
|====

==  Daten-Export

Wie in Tabelle 1 erläutert, können Sie zwischen einem einfachen und einem detaillierten Export wählen. Nachfolgend finden Sie entsprechende Beispiele.

===  Export der Diagrammdaten

[IMPORTANT]
.Datenfelder
====
**ItemID** = Artikel-ID, **PriceID** = Preis-ID,**VariantID** = ID der Artikelvariante, **VariantName** = Name der Artikelvariante,**ItemNo** = Artikelnummer, **ItemName** = Artikelname, **ItemWeight** = Artikelgewicht, **SoldItems** = Anzahl verkaufter Artikel, **Returned Items** = Anzahl retournierter Artikel, **Sales** = Umsatz, **SalesNet** = Netto-Umsatz, **Returns** = Retouren, **ReturnsNet** = Netto-Retouren, **AdjustedItems** = bereinigte Artikel, **AdjustedSales** = Umsatz bereinigt, **Adjusted SalesNet** = Netto-Umsatz bereinigt, **ReturnRate** = Retouren-Rate
====
