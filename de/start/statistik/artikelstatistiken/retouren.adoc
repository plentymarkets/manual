== Retouren

Diese Statistik zeigt die Beträge der **Retouren** an, in Abhängigkeit **der Zeitspanne** und nach **Retourengrund**. Alternativ zu einer Zeitspanne können Sie den Zeitraum per **Start**- und **Enddatum** eingrenzen. Standardeinstellung der **Zeitspanne**: die **letzten 10 Tage**.

== Statistik-Details

Entsprechend der gewählten **Zeitspanne** wird die jeweilige **Auftragsanzahl** und der jeweilige **Umsatz in Euro** angezeigt, außerdem sind die Retouren nach **Retouregründen** sortiert. Sind keine Buchungen für eine Auftragsart vorhanden, wird die jeweilige Auftragsart nicht angezeigt. Als **Tooltip** wird zu einem **Retouregrund** der zugehörige **Betrag****in Euro** angezeigt (siehe Bild 1). Sie können mit dieser Statistik z.B. erkennen, wieviele **Retouren** vorhanden sind und wieviel **Umsatz** im jeweiligen Zeitraum durch Retouren betroffen ist. Außerdem können Sie ablesen, wie **Anzahl** und **Umsatz** sich auf die **Retourengründe** verteilen. Das dient dem Controlling Ihrer Rückläufer.

[WARNING]
.**Artikelpakete**
====
**Retouren-Artikel aus Artikelpaketen** werden nach **Menge**, jedoch pass:q[<u>nicht nach Umsatz</u>] ausgewertet. Bei **Artikelpaketen** steht im Tab **Statistik** keine Auswertung zur Verfügung.
====

[NOTE]
.**Details anzeigen (Tooltip)**
====
Zeigen Sie mit dem **Mauszeiger** auf ein Element der Statistik, um die jeweiligen **Details** anzuzeigen.
====

[TIP]
.Tipp
====
In Bild 1 wird z.B. zum Retourengrund **defekt** ein **Betrag** vo**n****393,49 Euro** für **1** Retoure angezeigt.
====

Beachten Sie die Erläuterungen in der folgenden Tabelle:

[cols="a,a"]
|====
|Einstellung |Erläuterung

|
|Diagramm-Anzeige

|**Diagramm-Typ**
|In diesem Dropdown-Menü wählen Sie die Art der Darstellung aus. +
**2D-Kreisdiagramm** = Darstellung der Werte als flächige Kreissektoren +
**3D-Kreisdiagramm** = Darstellung der Werte als dreidimensionale Kreissektoren +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
|Die Daten des Diagramms werden in eine **CSV-Datei** exportiert. Dabei werden die Parameter wie angezeigt verwendet, sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Beim Rohdaten-Export wird ebenfalls eine **CSV-Datei** erzeugt. Dabei werden die Daten aber detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
|Einstellungen

|**Name**
|Hier wird der **Name** bzw. die **Bezeichnung** der Statistik angezeigt, die beim Erstelle einer neuen Statistik gewählt wurde. Sie können diesen Namen hier ändern.

|**Zeitspanne**
|Die **Zeitspanne** ist für die Statistiktypen jeweils voreingestellt, die Standardwerte können Sie Tabelle 1 der Handbuchseite <<start/statistik#, **Statistik**  >> entnehmen. Es stehen feste Werte zur Verfügung, die per Dropdown-Menü auswählbar sind. Darüber hinaus bieten manche Statistiktypen, die Möglichkeit eine Zeitspanne in Form eines **Datumbereichs** einzutragen. +
**Verändern** Sie die **Zeitspanne**, wenn keine Daten angezeigt werden oder nur wenige Daten und klicken Sie auf das Speichern-Symbol, um das Ergebnis zu sehen. Sie können durch Wahl eines **längeren Zeitraums** mehr Informationen anzeigen. Wenn viele Daten vorhanden sind, bewirkt die **Verringerung der Zeitspanne** ggf. eine aussagekräftigere Auswertung.

|**Startdatum ... Enddatum**
|Hier können Sie optional einen bestimmten **Datumsbereich** alternativ zur Option **Zeitspanne** eingrenzen.

|**Status-Optionen**
|Je nach Statistiktyp stehen zur Eingrenzung des **Status** verschiedene Optionen zur Verfügung. Es sind jeweils **Standardwerte** voreingestellt.

|**Mandant (Shop)**
|Wählen Sie die **Webshops** bzw. **Mandanten**, deren Daten für die Statistik verwendet werden sollen.

|**Auftragsherkunft**
|Wählen Sie eine oder mehrere **Auftragsherkünfte** oder mit der Option **Alle** sämtliche verfügbaren aus. +
Wenn Sie die **Retouren** verschiedener Marktplätze **vergleichen** möchten, legen Sie jeweils Statistiken dazu an, in denen Sie immer nur einen Marktplatz auswählen. Geben Sie diesen Marktplatz dann jeweils im Namen der Statistiken mit an, so können Sie diese dann einfach vergleichen.

|
|Beachten Sie dazu auch die Handbuchseite **Start / Statistik**.
|====
