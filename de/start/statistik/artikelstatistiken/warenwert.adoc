== Warenwert

Auswertung des **Warenwertes** Ihrer **Artikel** gemäß dem/den gewählten **Lager**/n, dem **Preis** sowie dem/den **Lieferanten** innerhalb einer gewählten Zeitspanne. Standardeinstellung der Zeitspanne: **Heute**.

== Statistik-Details

Es wird die **Anzahl der Artikel** und der **Warenwert in Euro** nach gewähltem/n **Lager**/n angezeigt. Als **Tooltip** wird der genaue **Warenwert** und das jeweils zugehörige **Lager** angezeigt, außerdem in einer anderen Farbe die genaue **Artikelmenge**, mit der der jeweilige Warenwert erzielt wird (siehe Bild 1). Sie können mit dieser Statistik z.B. erkennen, wie hoch der Warenwert und die Artikelmenge in jedem Ihrer Lager ist.

[NOTE]
.**Details anzeigen (Tooltip)**
====
Zeigen Sie mit dem **Mauszeiger** auf ein Element der Statistik, um die jeweiligen **Details** anzuzeigen.
====

[TIP]
.Tipp
====
In Bild 1 wird z.B. für das **Lager Kassel** ein **Warenwert** von **26.479,20 Euro** angezeigt, der mit **408 Artikeln** generiert wurde.Die genaue **Artikelmenge** wird per **Tooltip** über den blauen Balken angezeigt.
====

[IMPORTANT]
.
====
Bezüglich **Menge** und **Warenwert** werden nur **positive physische Bestände** ausgewertet.
====

[cols="a,a"]
|====

|
|Diagramm-Anzeige

|**Diagrammtyp**
|In diesem Dropdown-Menü wählen Sie die Art der Darstellung aus. +
**Doppel-Säulendiagramm** = Darstellung der Werte als Säulen +
**Daten-Tabelle** = Datenübersicht in Tabellenform +
**Netto** = Beträge abzüglich UsSt**. +
Umsatz bereinigt** = Umsatz abzüglich Gutschriften +
**Deckungsbeitrag I** = Umsatz abzüglich EK, Steuern und Versandkosten

|**Export**
|Die Daten des Diagramms werden in eine **CSV-Datei** exportiert. Dabei werden die Parameter wie angezeigt verwendet, sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
|Beim Rohdaten-Export wird ebenfalls eine **CSV-Datei** erzeugt. Dabei werden die Daten aber detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|**Anzeige der Werte**
|Klicken Sie links unten auf die Werte, um die **Anzeige** jeweils im Diagramm zu **deaktivieren** bzw. zu **aktivieren**. So können Sie nur jeweils die Daten separat anzeigen, die Sie sehen möchten und andere ausblenden. Dies kann bei komplizierten Daten eine Hilfe sein, die einzelnen Werte besser zu erkennen.

|
|Einstellungen

|**Name**
|Hier wird der **Name** bzw. die **Bezeichnung** der Statistik angezeigt, die beim Erstellen einer neuen Statistik gewählt wurde. Sie können diesen Namen hier ändern.

|**Lager**
|Wählen Sie die **Lager**, deren Daten für die Statistik verwendet werden sollen.Mit der Option **Alle** sämtliche verfügbaren auswählen.

|**Preisauswahl**
|Wählen Sie, welche **Preise** ausgewertet werden sollen. +
**Durchschnittlicher EK** = Der Durchschnitt aller Einkaufspreise für jeden Artikel +
**Aktueller EK** = Der zuletzt für diesen Artikel hinterlegte Einkaufspreis

|**Lieferant**
|Wählen Sie die **Zahlungsart**, die für die Statistik berücksichtigt werden soll oder aktivieren Sie die Option **Alle**, um alle Zahlungsarten einzubeziehen.
|====

== Daten-Export

Wie in Tabelle 1 erläutert können Sie zwischen einem einfachen und einem detaillierten Exportergebnis wählen. Nachfolgend entsprechende Beispiele.

=== Export der Diagrammdaten

[IMPORTANT]
.**Datenfelder**
