==  Wareneingang nach Datum

Auswertung des **Wareneingang nach Datum** Ihrer **Artikel** gemäß dem/den gewählten **Lager**/n sowie dem/den **Lieferanten** innerhalb einer gewählten Zeitspanne. Standardeinstellung der Zeitspanne: **Die letzten 24 Stunden**.

==  Statistik-Details

Es wird die **Anzahl der Artikel** und der **Warenwert in Euro** des Wareneingangs nach **Datum** angezeigt. Als **Tooltip** wird der genaue **Warenwert** und das jeweils zugehörige **Lieferant** angezeigt, außerdem in einer anderen Farbe die genaue **Artikelmenge**, mit der der jeweilige Warenwert erzielt wird (siehe Bild 1). Sie können mit dieser Statistik z.B. erkennen, wie hoch der Warenwert und die Artikelmenge für jede Ihrer Lieferanten ist.

[NOTE]
.**Details anzeigen (Tooltip)**
====
Zeigen Sie mit dem **Mauszeiger** auf ein Element der Statistik, um die jeweiligen **Details** anzuzeigen.
====

[TIP]
.Tipp
====
In Bild 1 wird z.B. für den **Lieferant Der Lieferprofi** ein **Warenwert** von **3.130,00 Euro** in Dezember, 2012 angezeigt, der mit **120,00 Artikeln** generiert wurde.
====

==  Einstellungen

Die nachfolgend beschriebenen **Einstellungen** stehen für diese Statistik zur Verfügung.

image::start/statistik/artikelstatistiken/assets/DE-ArtikelBearbeiten-Statistik-Wareneingang-Datum02-SI.png[]

__Bild 2: Statistik **Wareneingang nach Datum**; **Einstellungen**__

Beachten Sie die Erläuterungen in der folgenden Tabelle:

[cols="a,a"]
|====
| Einstellung | Erläuterung

|
| Diagramm-Anzeige

|**Diagrammtyp**
| In diesem Dropdown-Menü wählen Sie die Art der Darstellung aus. +
**Gestapeltes Doppel-Säulendiagramm** = Darstellung der Werte als Säulen. Die Lieferanten sind in verschiedenen Farben gestapelt dargestellt. +
**Daten-Tabelle** = Datenübersicht in Tabellenform

|**Export**
| Die Daten des Diagramms werden in eine **CSV-Datei** exportiert. Dabei werden die Parameter wie angezeigt verwendet, sie entsprechen denen des Diagrammtyps **Daten-Tabelle**.

|**Rohdaten-Export**
| Beim Rohdaten-Export wird ebenfalls eine **CSV-Datei** erzeugt. Dabei werden die Daten aber detaillierter exportiert. So wird anstelle der Anzahl der Aufträge für jeden Auftrag die relevanten Datenfelder, wie **Auftrags-ID**, **Preis** etc. in die CSV-Datei eingefügt.

|
| Einstellungen

|**Name**
| Hier wird der **Name** bzw. die **Bezeichnung** der Statistik angezeigt, die beim Erstellen einer neuen Statistik gewählt wurde. Sie können diesen Namen hier ändern.

|**Zeitspanne**
| Die **Zeitspanne** ist für die Statistiktypen jeweils voreingestellt, die Standardwerte können Sie Tabelle 1 der Handbuchseite <<start/statistik#, **Statistik**  >> entnehmen. Es stehen feste Werte zur Verfügung, die per Dropdown-Menü auswählbar sind. Darüber hinaus bieten manche Statistiktypen die Möglichkeit, eine Zeitspanne in Form eines **Datumbereichs** einzutragen. +
**Verändern** Sie die **Zeitspanne**, wenn keine Daten angezeigt werden oder nur wenige Daten und klicken Sie auf das Speichern-Symbol, um das Ergebnis zu sehen. Sie können durch Wahl eines **längeren Zeitraums** mehr Informationen anzeigen. Wenn viele Daten vorhanden sind, bewirkt die **Verringerung der Zeitspanne** ggf. eine aussagekräftigere Auswertung.

|**Lager**
| Wählen Sie die **Lager**, deren Daten für die Statistik verwendet werden sollen.Mit der Option **Alle** sämtliche verfügbaren auswählen.

|**Lieferant**
| Wählen Sie die **Lieferanten**, die für die Statistik berücksichtigt werden soll oder aktivieren Sie die Option **Alle**, um alle Lieferanten einzubeziehen.
|====

==  Daten-Export

Wie in Tabelle 1 erläutert können Sie zwischen einem einfachen und einem detaillierten Exportergebnis wählen. Nachfolgend entsprechende Beispiele.

=== Export der Diagrammdaten

[IMPORTANT]
.**Datenfelder**
====
**Date** = Datum, **SupplierID** = Lieferanten-ID, **SupplierName** = Lieferanten-Name, **QuantityOfItems** = Artikelmenge, **NetItemValue** = Netto-Warenwert
====

=== Export der Rohdaten

[IMPORTANT]
.**Datenfelder**
====
**Date** = Datum, **SupplierID** = Lieferanten-ID, **PriceSetID** = ID des Preissets des Artikels, **ItemID** = Artikel-ID, **AttributeValueSetID** = ID der Artikelvariante, **SupplierName** = Name des Lieferanten, **Quantity** = Anzahl der Wareneingänge, **PurchasePrice** = Einkaufspreis
