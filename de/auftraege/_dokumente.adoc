= Dokumente
:lang: de
include::{includedir}/_header.adoc[]
:position: 50

Im Menü **Dokumente** eines Standorts unter **System » Mandant » Mandant wählen » Standorte** konfigurieren und verwalten Sie alle Einstellungen für auftragsbezogene Dokumente, z.B. **Rechnungen**, **Lieferscheine** und **Auftragsbestätigungen**.

== Standortbezogene Verwaltung

Die Dokumente werden pro Standort angelegt und befinden sich jeweils im Unterordner **Dokumente** eines Standorts. Sie erfahren in den Erläuterungen auf der Handbuchseite <<auftragsabwicklung/auftraege/auftragsdokumente#, **Dokumente**>>, wie Sie die Standardeinstellungen der Dokumente anpassen oder die Einstellungen eines Dokuments auf eine andere Vorlage übertragen. Des Weiteren wird erläutert, wie Sie Geschäftspapier als PDF-Vorlagen erstellen und mit den Dokumenten verknüpfen. Auch das Anlegen und Verwalten von Nummernkreisen wird erklärt. Schließlich folgen Informationen mit steuerrechtlicher Relevanz für Dokumente.

== Benutzerrechte

Den Zugriff auf die Dokumente regeln Sie unter <<basics/admin-aufgaben/benutzer-verwalten#, **System » Einstellungen » Benutzer**>> pro Benutzer im Tab **Berechtigungen**, Tab <<basics/admin-aufgaben/benutzer-verwalten#90, **Menü**>>.

== Weitere Dokumente

Die nicht standortbezogenen Dokumente, z.B. Adressetiketten und Picklisten, richten Sie unter <<auftragsabwicklung/auftraege/auftragsdokumente#, **System » Aufträge » Dokumente**>> ein.

== Rechnung

Eine Rechnung enthält unter anderem Artikeldaten der Bestellung, die Empfängeranschrift sowie den zu zahlenden Betrag. plentymarkets bietet Ihnen Vorlagen, in denen Sie Parameter beliebig anpassen können, z.B. die Angabe eines Zahlungsziels oder von Skonto-Optionen. Diese Parameter werden tlw. direkt in der Vorlage eingetragen oder auch automatisch aus den Kundendaten eingefügt. Das Dokument **Rechnung** erstellen und verwalten Sie standort- und sprachbezogen unter **System » Mandant » Mandant wählen » Standorte » Standort wählen » Dokumente**.

=== Automatisierte Rechnungsinformationen

Es gibt 2 Angaben, die beim Generieren von Rechnungen automatisch eingetragen werden. Zum einen wird bei Lastschriftzahlungen automatisch ein Abbuchungshinweis eingetragen und bei Sendungen, die außerhalb der EU geliefert werden, wird die Zolltarifnummer beim Artikelnamen eingefügt.

[IMPORTANT]
.Anzeige der Bankdaten bei Lastschrift
====
Bei der Zahlungsart **Lastschrift** wird automatisch ein Abbuchungshinweis mit den Kontodaten (IBAN und BIC) des Kunden in das Dokument eingefügt. Die IBAN wird dabei aus Sicherheitsgründen ab dem 12. Zeichen bis zu den letzten 3 Zeichen durch "xxx" ersetzt. +
Wenn für den Kunden keine Bankdaten gespeichert sind, werden **IBAN** und **BIC** nicht auf der Rechnung ausgegeben. +
Der Hinweis ist im System gespeichert und kann nicht geändert werden.
====

[IMPORTANT]
.Zolltarifnummer wird bei Lieferungen in Länder außerhalb der EU automatisch eingefügt
====
Die Zolltarifnummer wird immer automatisch eingefügt, wenn in ein Land außerhalb der EU geliefert wird. Nur bei Lieferungen in Länder innerhalb der EU wird die Einstellung zur Zolltarifnummer für die Artikelbezeichnung berücksichtigt.
====

=== Rechnungen konfigurieren

Die Konfiguration umfasst das Einrichten der Vorlage im Menü **Dokumente** eines Standorts, die Übernahme oder Übertragung von Eigenschaften, das optionale Hinterlegen einer PDF-Vorlage und das Einrichten eines Nummernkreises.

==== Standard-Einstellungen übernehmen

Im Tab **Transfer** besteht die Möglichkeit, Einstellungen aus einer Standard-Vorlage in eine Vorlage zu übernehmen. Dies ist z.B. sinnvoll, wenn die Vorlage noch keine Einstellungen enthält. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#140, **Einstellungstransfer**>>.

==== Vorlagen einrichten

Um eine Rechnung zu erstellen, muss zuerst eine Vorlage erstellt werden. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#40, **Dokumentenvorlage konfigurieren**>>.

[IMPORTANT]
.Positionierung durch Koordinaten
====
Die Inhalte der Vorlage werden durch die Eingabe der **Koordinaten X** und **Y** positioniert. Dabei wird als Ausgangspunkt die obere linke Ecke einer Seite angenommen.
====

==== Einstellungen übertragen

Ebenfalls im Tab **Transfer** können Sie die Einstellungen des Tabs **Vorlage** einer Rechnung auf ein anderes Dokument, z.B. den Lieferschein, übertragen. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#140, **Einstellungstransfer**>>.

==== PDF-Vorlagen hinterlegen

Wenn Sie eine Rechnungsvorlage erstellt haben, können Sie im Tab **PDF-Vorlage** eine PDF-Datei z.B. mit den Daten Ihres Shops inklusive Logo hinterlegen. Diese PDF-Datei müssen Sie zunächst auf Ihrem Rechner speichern. Beachten Sie dazu die Handbuchseite **Dokumente**, Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#170, **PDF-Vorlage**>>.

==== Nummernkreise erstellen

Für die Gestaltung von Nummernkreisen bietet die Dokumentenkonfiguration verschiedene Möglichkeiten im Tab **Nummernkreis**. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#200, **Nummernkreise**>>.

== Gelangensbestätigung

Bei Sendungen ins EU-Ausland müssen Händler gegenüber den Finanzbehörden belegen, dass eine Ware tatsächlich von Deutschland in ein anderes EU-Mitgliedsland gelangt ist. Dafür hat der deutsche Gesetzgeber die sogenannte Gelangensbestätigung eingeführt. Der ausländische Käufer muss dem deutschen Unternehmen nach Abschluss der Lieferung den Erhalt der Ware bestätigen.

Das Dokument **Gelangensbestätigung** konfigurieren und verwalten Sie sprachbezogen im Ordner **Dokumente** eines Standorts unter **System » Mandant » Mandant wählen » Standorte » Standort wählen**.

=== Angaben in der Gelangensbestätigung

Wenn ein Händler Artikel an einen Kunden mit gültiger Umsatzsteuer-ID im europäischen Ausland versendet, entfällt die Umsatzsteuer. Der Auftrag wird also netto ausgeführt. Liegt die Lieferadresse des Kunden jedoch innerhalb Deutschlands, wird für den Auftrag Umsatzsteuer berechnet. Per Gesetz ist es jedoch zulässig, bei Kunden mit gültiger Umsatzsteuer-ID im europäischen Ausland auch bei einer deutschen Lieferadresse den Auftrag netto auszuführen, wenn dem Auftrag eine Gelangensbestätigung beigelegt wird. Daher gibt es im Menü **<<crm/kundendaten-verwalten#1900, Lieferanschriften>>** die Option **Gelangensbestätigung**. Wird die Option **Gelangensbestätigung** aktiviert (Häkchen gesetzt), werden Lieferungen ins EU-Ausland mit deutscher Lieferanschrift wie eine EU-Auslandssendung behandelt. Ob die Rechnung brutto oder netto ausgestellt wird, ist dann von den Einstellungen **Umsatzsteuer-ID** und **Ausfuhrlieferung** im Menü <<auftragsabwicklung/auftraege/buchhaltung#, **Buchhaltung**>> abhängig. Ist die Option **Gelangensbestätigung** nicht aktiviert, werden diese Aufträge brutto berechnet.

Neben Pflichtelementen bietet die Vorlage optionale Elemente, z.B. die Angabe eines Zahlungsziels oder Skonto-Optionen. Diese Parameter werden direkt in der Vorlage konfiguriert oder per Einstellung automatisch über Platzhalter aus den Kundendaten eingefügt. Welche Angaben eine Gelangensbestätigung enthalten muss, erfahren Sie in der link:http://www.gesetze-im-internet.de/ustdv_1980/__17a.html[**Umsatzsteuer-Durchführungsverordnung**^]. Fragen Sie alternativ Ihren Steuerberater.

=== Gelangensbestätigungen konfigurieren

Die Konfiguration umfasst die Einrichtung der Vorlage, die Übernahme oder Übertragung von Eigenschaften, das optionale Hinterlegen einer PDF-Vorlage und das Einrichten eines Nummernkreises.

==== Standard-Einstellungen übernehmen

Im Tab **Transfer** besteht die Möglichkeit, Einstellungen aus einer Standard-Vorlage in eine Vorlage zu übernehmen oder Einstellungen der Vorlage auf ein anderes Dokument, z.B. den Lieferschein, zu übertragen. Dies ist z.B. sinnvoll, wenn die Vorlage noch keine Einstellungen enthält. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#140, **Einstellungs-Transfer**>>.

==== Vorlage einrichten

Damit eine Gelangensbestätigung erzeugt werden kann, muss zuerst eine Vorlage eingerichtet werden. Details zur Konfiguration finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#40, **Dokumentenvorlage konfigurieren**>>.

[IMPORTANT]
.Positionierung durch Koordinaten
====
Die Inhalte der Vorlage werden durch die Eingabe der **Koordinaten X** und **Y** positioniert. Dabei wird als Ausgangspunkt die obere linke Ecke einer Seite angenommen.
====

==== Einstellungen übertragen

Ebenfalls im Tab **Transfer** können Sie die Einstellungen des Tabs **Vorlage** einer Gelangensbestätigung auf ein anderes Dokument, z.B. den Lieferschein, übertragen. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#140, **Einstellungstransfer**>>.

==== PDF-Vorlagen hinterlegen

Wenn Sie eine Rechnungsvorlage erstellt haben, können Sie im Tab **PDF-Vorlage** eine PDF-Datei z.B. mit den Daten Ihres Shops inklusive Logo hinterlegen. Diese PDF-Datei müssen Sie zunächst auf Ihrem Rechner speichern. Beachten Sie dazu die Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#170, **PDF-Vorlage**>>.

==== Nummernkreise erstellen

Für die Gestaltung von Nummernkreisen bietet die Dokumentenkonfiguration verschiedene Möglichkeiten im Tab **Nummernkreis**. Details dazu finden Sie auf der Handbuchseite **Dokumente** im Kapitel <<auftragsabwicklung/auftraege/auftragsdokumente#200, **Nummernkreise**>>.

=== Gelangensbestätigung erstellen

Eine Gelangensbestätigung erstellen Sie im Tab **Belege** eines Auftrags. Gehen Sie dazu wie unten beschrieben vor.

image::omni-channel/online-shop/assets/DE-Auftraege-Dokumente-Gelangensbestaetigung-03.png[]

__Bild 1: Gelangensbestätigung erstellen__

[.instruction]
Gelangensbestätigung erstellen:

. Öffnen Sie das Menü **Aufträge » Aufträge bearbeiten**.
. Öffnen Sie den gewünschten Auftrag.
. Wechseln Sie in das Tab **Belege**.
. Wählen Sie in der Dropdown-Liste **Beleg erstellen** die Option **Gelangensbestätigung**. +
→ Ein Fenster zur Bearbeitung der Gelangensbestätigung wird geöffnet.
. Nehmen Sie, wenn gewünscht, Änderungen vor.
. Klicken Sie auf **Speichern**. +
→ Die Gelangensbestätigung wird als PDF geöffnet und in der Auflistung mit Datum angezeigt.
