== Allgemeines zur Lager-Pickliste

Mit plentymarkets können Sie Lager-Picklisten als PDF-Dateien generieren. Im Gegensatz zur Pickliste ist bei der Lager-Pickliste eine Zuordnung zum Auftrag bzw. zur Rechnung möglich. So können die Artikel bei der Kommissionierung direkt zugeordnet und anschließend zum Verpacken gegeben werden.

== Artikeldaten darstellen

Im Menü **Lager-Pickliste** bestimmen Sie, welche Artikeldaten Ihre Lager-Picklisten enthalten, z.B. **Artikel-ID**, **EAN**, **Lagerort** usw. und wie diese Informationen dargestellt werden sollen.

image::auftraege/einstellungen/dokumente/assets/DE-Einstellungen-Auftraege-Dokumente-Lager-Pickliste-01.png[]

__Bild 1: Optionen im Menü **Lager-Pickliste**__

[IMPORTANT]
.Felder anzeigen bzw. verbergen
====
Damit ein Feld angezeigt wird, muss für das jeweilige Feld eine Positionsnummer ausgewählt sein. Wenn Sie für ein Feld keine Anzeige auf der Lager-Pickliste wünschen, wählen Sie für die Positionsnummer die Einstellung **Pos**.
====

[.instruction]
Artikeldaten darstellen:

. Öffnen Sie das Menü **Einstellungen » Aufträge » Dokumente » Lager-Pickliste**.
. Entscheiden Sie, welche Felder auf der Lager-Pickliste angezeigt werden sollen. Bestimmen Sie die Reihenfolge dieser Felder, indem Sie für jedes zu verwendende Feld eine **Positionsnummer** wählen.
. Tragen Sie unter **Name** eine eigene Bezeichnung für die verwendeten Felder ein.
. Bestimmen Sie die Breite für die Felder sowie die Schriftgröße und den Schriftschnitt für die Texte.
. Nehmen Sie weitere Einstellungen gemäß Tabelle 1 vor.
. Klicken Sie auf **Speichern**, um die Einstellungen zu sichern.

[cols="a,a"]
|====
|Einstellung |Erläuterung

|**Seitenrand oben**
|Einen Wert (in mm) für den oberen Seitenrand eintragen. Den Wert bei Bedarf anpassen, wenn z.B. eine PDF-Vorlage verwendet wird, die oben im "Briefpapier" die Firmenangaben und ein Logo enthält und dieser "Header" oberhalb der Lager-Pickliste positioniert werden soll (siehe Kapitel 3).

|**Lagerort**
|Bei Verwendung dieses Feldes bestimmen, wie der **Lagerortname** auf der Lager-Pickliste angezeigt werden soll. Es kann z.B. nur der Lagerortname angezeigt werden oder eine Kombination aus Lager-Informationen, d.h. **Regalboden/Lagerortname**.

|**Freitext**
|Bei Verwendung dieses Feldes das entsprechende **Freitextfeld 1-20** aus dem Dropdown-Menü wählen. +
Freitextfelder zuerst im Menü **<<artikel/artikel-verwalten#70., Einstellungen » Artikel » Freitextfelder >>** anlegen. Danach die angelegten Freitextfelder mit den entsprechenden Artikeln verknüpfen und die Werte für die Felder eintragen. Dies erfolgt im Menü <<artikel/artikel-verwalten#75., **Artikel » Artikel bearbeiten » Tab: Felder** >>.

|**Artikelbeschreibung**
|Wählen durch Setzen des Häkchens, welche Informationen die Artikelbeschreibung in der Lager-Pickliste enthalten soll. Der **Artikelname** wird immer mit diesem Feld angezeigt. Zusätzlich können noch weitere Angaben ausgewählt werden, z.B. **Artikel-ID**, **Externe Artikel-ID**, **Artikelnummer** und **EAN**. Sollen Parameter in der Lager-Pickliste nicht angezeigt werden, Häkchen jeweils entfernen. Die Auswahl gilt auch für Varianten.
|====

== PDF-Vorlage hochladen

Eine Vorlage wird für das "Briefpapier" Ihres Shops benötigt, also eine Seite, die Ihre Firmendaten, Layoutelemente gemäß dem Corporate Design Ihres Unternehmens sowie Ihr Logo enthält. Diese Vorlage wird für das System im PDF-Format benötigt. Mit plentymarkets können Sie eine Dokumenten-Vorlage im PDF-Format für die Lager-Pickliste verwenden. Gehen Sie wie im Folgenden beschrieben vor, um eine PDF-Vorlage hochzuladen.

image::auftraege/einstellungen/dokumente/assets/DE-Einstellungen-Auftraege-Dokumente-Lager-Pickliste-03.png[]

__Bild 3: PDF-Vorlage hochladen__

[.instruction]
PDF-Vorlage hochladen:

. Öffnen Sie das Menü **Einstellungen » Aufträge » Dokumente » Lager-Pickliste**.
. Klicken Sie auf das Tab **PDF-Vorlage**.
. Klicken Sie auf das Tab **Neue Vorlage**.
. Klicken Sie auf **Durchsuchen**.
. Wählen Sie die PDF-Datei der Vorlage aus.
. Klicken Sie auf **Speichern**, um die Einstellungen zu sichern. +
→ Die PDF-Vorlage erscheint nun im Tab **Vorlagen**. Sie können dort eine Vorschau sehen oder die PDF-Vorlage wieder löschen.

[TIP]
.Tipp
====
Wenn Sie z.B. eine PDF-Vorlage mit "Header" verwenden und Sie diesen Header oberhalb der Lager-Pickliste positionieren möchten, müssen Sie ggf. den hinterlegten Wert im Feld **Seitenrand oben** noch anpassen (siehe Tabelle 1).
====

Weitere Details zu Dokumenten-Vorlagen finden Sie auf der Handbuchseite **<<auftraege/einstellungen/dokumente#, Dokumente  >>**.
