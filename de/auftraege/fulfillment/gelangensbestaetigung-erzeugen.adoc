== Gelangensbestätigung erzeugen

In diesem Menü werden Gelangensbesätigungen mit Hilfe einer **Gruppenfunktion** erzeugt. So können beispielsweise über eine Suchanfrage mit der Option **Auftragsstatus [5] Freigabe Versand** per Gruppenfunktion für alle Aufträge Gelangensbestätigungen generiert werden, die aktuell bereit zum Versand sind.

== Optionen

Einstellungen zum Layout der Gelangensbestätigung können Sie im Menü **Einstellungen &gt;****Aufträge &gt;****Dokumente** vornehmen.

Im Menü **Einstellungen &gt;****Aufträge &gt;****Ereignisaktionen** definieren Sie, welche **Aktion(en)** bei der Generierung der Gelangensbestätigung zusätzlich ausgeführt werden soll(en).

== Gelangensbestätigung erzeugen

[NOTE]
.Gelangensbestätigung erzeugen:
====
Öffnen Sie das Menü **Gelangensbestätigung erzeugen** (Pfad siehe oben).

Nehmen Sie die **Einstellungen** für den **Suchfilter** gemäß Tabelle 1 auf dieser Seite vor.

Wählen Sie unter **Aktion** einen neuen **Auftragsstatus**, der den gewählten Aufträgen nach erfolgreicher Erstellung der Gelangensbestätigung zugewiesen werden soll und geben Sie, wenn gewünscht, einen **Kommentar** ein.

Wenn Sie alle Einstellungen vorgenommen haben, klicken Sie auf das **Zahnrad**. +
→ Sie erhalten eine **pdf-Datei**, welche die zugehörigen Gelangensbestätigungen aller von Ihnen durch die Filteroptionen bestimmten Aufträge beinhaltet.
====

__Bild 1: Menü: **Gelangensbestätigung erzeugen**__

[cols="a,a"]
|====
|Einstellung |Erläuterung

|**Auftragsstatus**
|Wählen Sie den **Status**, in dem sich die Aufträge befinden, für die Sie Gelangensbestätigungen erzeugen möchten.

|Eigner
|Wählen Sie **einen Eigner** der zu suchenden Aufträge oder die Option **ALLE**.

|Mandant (Shop)
|Wählen Sie einen **Shop** oder die Option **ALLE**.

|**Lager**
|Begrenzen Sie die Suche auf ein **Lager** oder führen Sie die Suche für **ALLE** Lager aus.

|**Zahlungsart**
|Wählen Sie eine der **Zahlungsarten** oder die Option **ALLE**.

|**Sortierung**
|Bestimmen Sie, nach welcher Option die erzeugten Gelangensbestätigungen sortiert werden sollen. Optionen: **Auftrags-ID**, **Kunden-ID**, **Artikel-ID**. Bei der Sortierungsoption **Artikel-ID** werden die Gelangensbestätigungen automatisch auch nach der **Auftrags-ID** sortiert.

|**Limit**
|Sie müssen die Erstellung der Gelangensbestätigungen auf einen der Werte im Drop-down-Menü **begrenzen**. Das Maximum sind **200** Gelangensbestätigungen.

|**Ändere Auftragsstatus**
|Wählen Sie einen neuen **Auftragsstatus** aus, der den gewählten Aufträgen nach erfolgreicher Erzeugung der Gelangensbestätigungen zugewiesen werden soll oder wählen Sie die Option **Keine Änderung**.

|**Kommentar**
|Sie können hier einen **Text** verfassen, der später auf allen Gelangensbestätigungen **unterhalb der Artikelpositionen** erscheint. Die Eingabe ist optional.

|**Gelangensbestätigungs-Layout ändern**
|Klicken Sie auf diesen Link um den Bereich **Einstellungen &gt;****Aufträge &gt;****Dokumente** in einem neuen Fenster zu öffnen. Dort haben Sie die Möglichkeit, detaillierte Einstellungen für die Gelangensbestätigungs-Vorlage vorzunehmen.

|**Ereignisaktion**
|Klicken Sie auf diesen Link um den Bereich **Einstellungen &gt;****Aufträge &gt;****Ereignisaktionen** in einem neuen Fenster zu öffnen. Dort können Sie neue Ereignisaktionen erstellen und bestehende bearbeiten.
|====
