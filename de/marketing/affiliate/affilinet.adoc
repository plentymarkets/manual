== affilinet

**link:http://www.affili.net/de/Startseite.aspx[affilinet, window="_blank"]** ist Deutschlands führende Affiliate-Plattform mit einem attraktiven Angebot sowie einem umfangreichen virtuellen Vertriebsnetzwerk.

== Bei affilinet registrieren

Der erste Schritt dazu ist eine **link:https://www.affili.net/de/Advertiser/Voranmeldung.aspx[Voranmeldung, window="_blank"]** bei affilinet. Sie können sich erst bei affilinet registrieren, wenn Ihre Voranmeldung akzeptiert wurde. Bevor Sie auf dieser Plattform aktiv werden können, müssen Sie von affilinet als **Advertiser** akzeptiert sein, Sie erhalten dann eine **Advertiser-ID**, die Sie zur Einrichtung in plentymarkets benötigen.

[WARNING]
.**URL-Parameter**
====
Für die Registrierung muss die **Tracking-Kennung** in der Form **ref=XYZ** mit der URL gesendet werden.
====

**Beispiel**: **www.IHR-WEBSHOP.de/?****ref****=XYZ (XYZ = Parameter von Affilinet)**

Weiterhin müssen Sie in den Einstellungen Ihres Accounts bei Affilinet die Funktion **Parameter-Forwarding** aktivieren. Dadurch wird automatisch der **ref**-Parameter an die URL Ihres Webshops angehängt.

== affilinet in plentymarkets einrichten

Wir stellen Ihnen mit der folgenden Konfiguration die Provisionsvariante **Pay-Per-Sale** zur Verfügung, d.h. Sie zahlen dabei eine **Provision** nur bei einem Verkauf. Die Provision wird **prozentual** berechnet. Der Parameter in plentymarkets für die Abrechnung der Provision ist die **Tarif-ID**, die Sie Ihrem Provisionsmodell bei affilinet entnehmen.

Wenn Sie sich bei affilinet angemeldet und die **Advertiser-ID** sowie die **Tarif-ID** erhalten haben, können Sie die Schnittstelle in plentymarkets einrichten.

[IMPORTANT]
.**Provision**
====
Die Konditionen für die Provision legen Sie bei affilinet fest. Diese Schnittstelle ist auf das **Pay-Per-Sale** Affiliate Programm von affilinet ausgerichtet.
====
