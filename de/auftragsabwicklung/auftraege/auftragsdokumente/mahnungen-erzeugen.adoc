[#100]
== Allgemeines zu Mahnungen

In plentymarkets können Sie Mahnungen als PDF-Dateien generieren, um diese z.B. zu drucken oder Ihren Kunden als E-Mail Anhang zukommen zu lassen. +
Eine Mahnung können Sie nach Ablauf der Zahlungsfrist verwenden, um Ihre Kunden an die ausstehende Zahlung zu erinnern.

[#200]
== Dokumentenvorlage für Mahnungen einrichten

Bevor Sie Mahnungen generieren können, muss eine Dokumentenvorlage eingerichtet werden. Dies geschieht im Menü **System » Mandant » Mandant wählen » Standorte » Standort wählen » Dokumente**. +
Weitere Informationen und Anleitungen zur Einrichtung der Dokumentenvorlage finden Sie auf der Handbuchseite <<auftragsabwicklung/auftraege/auftragsdokumente#, Auftragsdokumente>>.

[#300]
== Mahnungen erzeugen

Nachdem Sie eine Dokumentenvorlage für Mahnungen eingerichtet haben, können Sie Mahnungen als PDF-Dateien erzeugen. Dabei können Sie über einen Suchfilter festlegen, welche Aufträge berücksichtigt werden und ob beispielsweise eine Statusänderung erfolgen soll, nachdem die Mahnungen erzeugt wurden.

[.instruction]
Mahnungen erzeugen:

. Öffnen Sie das Menü **Aufträge » Fulfillment » Mahnungen erzeugen**.
. Nehmen Sie die Einstellungen anhand <<tabelle-einstellungen-fulfillment-mahnungen>> vor.
. Klicken Sie auf das **Zahnrad**, um die Mahnungen zu erzeugen. +
→ Es öffnet sich eine **PDF-Datei**, welche die zugehörigen Mahnungen aller von Ihnen durch die Suchfilter bestimmten Aufträge beinhaltet.

[[tabelle-einstellungen-fulfillment-mahnungen]]
.Optionen im Menü *Mahnungen erzeugen*
[cols="1,3"]
|====
|Einstellung |Erläuterung

|**Auftragsdatum**
|Begrenzen Sie die zu mahnenden Aufträge durch Auswahl eines Anfangs- und Enddatums aus den Kalendern. Es werden die Aufträge gefunden, deren Auftragsdatum im gewählten Zeitraum liegt.

|**Auftragsstatus**
|Aus der Liste den Status der Aufträge wählen, die für die Mahnungen berücksichtigt werden sollen.

|**Land**
|Wählen Sie ein Land aus, auf das die Suche beschränkt werden soll oder die Option **ALLE**.

|**Eigner**
|Wählen Sie einen Eigner aus, für dessen Aufträge die Mahnungen erzeugt werden sollen oder die Option **ALLE**, wenn für alle Eigner die Mahnungen erzeugt werden sollen.

|**Standort**
|Wählen Sie einen Standort aus, für den die Mahnungen erzeugt werden sollen.

|**Lager**
|Wählen Sie ein Lager oder **ALLE** aus.

|**Zahlungsart**
|Wählen Sie eine der Zahlungsarten oder *ALLE* aus.

|**Sortierung**
|Legen Sie fest, wie die erzeugten Mahnungen sortiert werden sollen. Die Optionen **Auftrags-ID** und **Kunden-ID** geben verschiedene Sortierungen aus.

|**Ändere Auftragsstatus**
|Wählen Sie einen Status, in den die Aufträge nach Erzeugung der Mahnungen wechseln sollen. +
Der Statuswechsel ist wichtig, damit für bereits bearbeitete Aufträge ein Prozess nicht noch einmal durchgeführt wird.

|**Varianten-ID**
|Eine **Mahngebühr** müssen Sie als eigenständigen Artikel mit einem Preis anlegen. Dieser Artikel erscheint dann als separater Posten auf der Mahnung und gibt die Mahngebühr aus. Wenn Sie den Artiekl angelegt haben, können Sie die entsprechende ID der Hauptvariante hier eintragen. Diese wird dann automatisch allen generierten Mahnungen hinzugefügt. +
Mehr Informationen und Anleitungen zum Erstellen von Artikeln finden Sie auf der Handbuchseite <<artikel/artikel-verwalten#20, Artikel verwalten>>.

|**Kommentar**
|Sie können hier einen Text verfassen, der später auf allen Mahnungen unterhalb der Artikelpositionen erscheint. Die Eingabe ist optional.
|====
