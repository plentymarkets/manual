== Lieferscheine erzeugen

In diesem Menü werden Lieferscheine mit Hilfe einer **Gruppenfunktion** erzeugt. So können beispielsweise über eine Suchanfrage mit der Option **Auftragsstatus [5] Freigabe Versand** per Gruppenfunktion für alle Aufträge Lieferscheine generiert werden, die aktuell für den Versand freigegeben wurden.

== Optionen

Einstellungen zum Layout der Lieferscheine können Sie im Menü **Einstellungen &gt;****Aufträge &gt;****Dokumente** vornehmen.

Im Menü **Einstellungen &gt;****Aufträge &gt;****Ereignisaktionen** definieren Sie, welche **Aktion(en)** bei der Lieferscheingenerierung zusätzlich ausgeführt werden soll(en).

== Lieferscheine erzeugen

[NOTE]
.Lieferscheine erzeugen:
====
Öffnen Sie das Menü **Lieferscheine erzeugen** (Pfad siehe oben).

Nehmen Sie die **Einstellungen** für den **Suchfilter** gemäß Tabelle 1 auf dieser Seite vor.

Wählen Sie unter **Aktion** einen neuen **Auftragsstatus**, der den gewählten Aufträgen nach erfolgreicher Lieferscheinerstellung zugewiesen werden soll und geben Sie, wenn gewünscht, einen **Kommentar** ein.

Klicken Sie nach Auswahl aller Optionen auf das **Zahnrad-Symbol**. +
→ Sie erhalten eine **pdf-Datei**, welche die zugehörigen Lieferscheine aller von Ihnen durch die Filteroptionen bestimmten Aufträge beinhaltet.
====

__Bild 1: Menü: **Lieferscheine erzeugen**__

[cols="1,3"]
|====
|Einstellung |Erläuterung

|**Auftragsstatus**
|Wählen Sie den **Status**, in dem sich die Aufträge befinden, für die Sie Lieferscheine erzeugen möchten.

|**Ausdruck**
|Entscheiden Sie, ob Sie möchten, dass nur Lieferscheine für Aufträge generiert werden, **für die eine Lieferanschrift vorliegt**, oder wählen Sie die Option **ALLE**.

|**Warenausgangsdatum**
|Wählen Sie ein **Datum aus dem Kalender** aus, das als Warenausgangsdatum auf den Lieferscheinen erscheinen soll.

|Eigner
|Wählen Sie **einen Eigner** der zu suchenden Aufträge oder die Option **ALLE**.

|Mandant (Shop)
|Wählen Sie einen **Shop** oder die Option **ALLE**.

|**Lager**
|Begrenzen Sie die Suche auf ein **Lager** oder führen Sie die Suche für **ALLE** Lager aus.

|**Zahlungsart**
|Wählen Sie eine der **Zahlungsarten** oder die Option **ALLE**.

|**Sortierung**
|Bestimmen Sie, nach welcher Option die erzeugten Lieferscheine sortiert werden sollen. Optionen: **Rechnungsnummer**, **Auftrags-ID**, **Kunden-ID**, **Artikel-ID**. Bei der Sortierungsoption **Artikel-ID** werden die Lieferscheine automatisch auch nach der **Auftrags-ID** sortiert.

|**Limit**
|Sie müssen die Lieferscheinerstellung auf einen der Werte im Drop-down-Menü **begrenzen**. Das Maximum sind **200** Lieferscheine.

|**Ändere Auftragsstatus**
|Wählen Sie einen neuen **Auftragsstatus** aus, der den gewählten Aufträgen nach erfolgreichem Lieferscheinabruf zugewiesen werden soll oder wählen Sie die Option **Keine Änderung**.

|**Kommentar**
|Sie können hier einen **Text** verfassen, der später auf allen Lieferscheinen **unterhalb der Artikelpositionen** erscheint. Die Eingabe ist optional.

|**Lieferscheinlayout ändern**
|Klicken Sie auf diesen Link um den Bereich **Einstellungen &gt;****Aufträge &gt;****Dokumente** in einem neuen Fenster zu öffnen. Dort haben Sie die Möglichkeit, detaillierte Einstellungen für die Lieferschein-Vorlage vorzunehmen.

|**Ereignisaktion**
|Klicken Sie auf diesen Link um den Bereich **Einstellungen &gt;****Aufträge &gt;****Ereignisaktionen** in einem neuen Fenster zu öffnen. Dort können Sie neue Ereignisaktionen erstellen und bestehende bearbeiten.
|====
