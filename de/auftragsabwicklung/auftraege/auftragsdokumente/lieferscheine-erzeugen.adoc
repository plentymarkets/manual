== Allgemeines zu Lieferscheinen

Mit plentymarkets können Sie Lieferscheine als PDF-Dateien generieren, um diese z.B. zu drucken oder in andere Programme zu importieren. Lieferscheine sind für Ihre Kunden ein wichtiges Dokument zur Prüfung des Wareneingangs.
Stellen Sie deshalb sicher, dass wichtige Angaben wie das Bestelldatum, die Artikelpositionen und die Mengenangaben richtig angezeigt werden.

== Dokumentenvorlage für Lieferscheine einrichten

Bevor Sie Lieferscheine generieren können, muss eine Dokumentenvorlage eingerichtet werden. Dies geschieht im Menü **System » Mandant » Mandant wählen » Standorte » Standort wählen » Dokumente**. +
Weitere Informationen und Anleitungen zur Einrichtung der Dokumentenvorlage finden Sie auf der Handbuchseite <<auftragsabwicklung/auftraege/auftragsdokumente#, Auftragsdokumente>>.

== Lieferscheine erzeugen

Nachdem Sie eine Dokumentenvorlage für Lieferscheine eingerichtet haben, können Sie Lieferscheine als PDF-Dateien erzeugen. Dabei können Sie über einen Suchfilter festlegen, welche Aufträge berücksichtigt werden und ob beispielsweise eine Statusänderung erfolgen soll, nachdem die Lieferscheine erzeugt wurden.

[.instruction]
Lieferscheine erzeugen:

. Öffnen Sie das Menü **Aufträge » Fulfillment » Lieferscheine erzeugen**.
. Nehmen Sie die Einstellungen anhand <<tabelle-einstellungen-fulfillment-lieferscheine>> vor.
. Klicken Sie auf das **Zahnrad**, um die Lieferscheine zu erzeugen. +
→ Es öffnet sich eine **PDF-Datei**, welche die zugehörigen Lieferscheine aller von Ihnen durch die Suchfilter bestimmten Aufträge beinhaltet.

[[tabelle-einstellungen-fulfillment-lieferscheine]]
.Optionen im Untermenü *Lieferscheine erzeugen*
[cols="1,3"]
|====
|Einstellung |Erläuterung

|**Auftragsstatus**
|Aus der Liste den Status der Aufträge wählen, die für die Lieferscheine berücksichtigt werden sollen.

|**Ausdruck**
|Legen Sie fest, ob nur Lieferscheine für Aufträge generiert werden, für die eine Lieferanschrift vorhanden ist, oder für **ALLE** Aufträge.

|**Warenausgangsdatum**
|Wählen Sie ein Datum aus dem Kalender aus, das als Warenausgangsdatum auf den Lieferscheinen erscheinen soll.

|**Eigner**
|Wählen Sie einen Eigner aus, für dessen Aufträge die Lieferscheine erzeugt werden sollen oder die Option **ALLE**, wenn für alle Eigner die Lieferscheine erzeugt werden sollen.

|**Standort**
|Wählen Sie einen Standort aus, für den die Lieferscheine erzeugt werden sollen.

|**Lager**
|Wählen Sie ein Lager oder **ALLE** aus.

|**Zahlungsart**
|Wählen Sie eine der Zahlungsarten oder *ALLE* aus.

|**Sortierung**
|Legen Sie fest, wie die erzeugten Lieferscheine sortiert werden sollen. Die Optionen **Rechnungsnummer**, **Auftrags-ID**, **Kunden-ID** und **Artikel-ID** geben verschiedene Sortierungen aus.

|**Limit**
|Legen Sie fest, wie viele Lieferscheine maximal pro Durchgang erzeugt werden. Das Maximum liegt bei **200** Lieferscheinen.

|**Lieferscheindatum**
|Wählen Sie ein Datum aus dem Kalender aus, das als Lieferscheindatum auf den Lieferscheinen erscheinen soll.

|**Ändere Auftragsstatus**
|Wählen Sie einen Status, in den die Aufträge nach Erzeugung der Lieferscheine wechseln sollen. +
Der Statuswechsel ist wichtig, damit für bereits bearbeitete Aufträge ein Prozess nicht noch einmal durchgeführt wird.

|**Kommentar**
|Hier können Sie einen Text verfassen, der später auf allen Lieferscheinen unterhalb der Artikelpositionen erscheint. Die Eingabe ist optional.
|====
