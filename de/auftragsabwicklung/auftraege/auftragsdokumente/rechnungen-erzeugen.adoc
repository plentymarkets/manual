== Rechnungen erzeugen

In diesem Menü werden Rechnungen mit Hilfe einer **Gruppenfunktion** erzeugt. So können beispielsweise über eine Suchanfrage mit der Option **Auftragsstatus [5] Freigabe Versand** per Gruppenfunktion alle Rechnungen generiert werden, die aktuell für den Versand freigegeben wurden.

== Optionen

Einstellungen zum **Layout** der Rechnungen können Sie im Menü **System » Mandant » Mandant öffnen » Standorte » Standort öffnen » Dokumente** vornehmen.

Im Menü **Einstellungen &gt; Aufträge &gt; Ereignisaktionen** definieren Sie, welche **Aktion(en)** bei der Rechnungsgenerierung zusätzlich ausgeführt werden soll(en). Beispielsweise eine **Ereignisaktion** zum automatischen Versand der Rechnung per E-Mail-Vorlage.

[TIP]
.Tipp
====
Sie finden in diesem Menü auch eine direkte Verlinkung zum Menü **Ereignisaktionen** über den Link **Ereignisaktion**.
====

== Rechnungen erzeugen

[NOTE]
.Rechnungen erzeugen:
====
Öffnen Sie das Menü **Rechnungen erzeugen** (Pfad siehe oben).

Nehmen Sie die **Einstellungen** für den **Suchfilter** gemäß Tabelle 1 auf dieser Seite vor.

Wählen Sie unter **Aktion** ein **Rechnungsdatum** und einen neuen **Auftragsstatus**, der den gewählten Aufträgen nach erfolgreicher Rechnungserstellung zugewiesen werden soll.

Klicken Sie nach Auswahl aller Optionen auf das **Zahnrad-Symbol**. +
→ Sie erhalten eine **pdf-Datei**, welche die zugehörigen Rechnungen aller von Ihnen durch die Filteroptionen bestimmten Aufträge enthält.
====

|**Auftragsstatus**
|Wählen Sie den **Status**, in dem sich die Aufträge befinden, für die Sie Rechnungen erzeugen möchten.

|Eigner
|Wählen Sie **einen Eigner** der zu suchenden Aufträge oder die Option **ALLE**.

|Mandant (Shop)
|Wählen Sie einen **Shop** oder die Option **ALLE**.

|**Lager**
|Begrenzen Sie die Suche auf ein **Lager** oder führen Sie die Suche für **ALLE** Lager aus.

|**Zahlungsart**
|Wählen Sie eine der **Zahlungsarten** oder die Option **ALLE**.

|**Sortierung**
|Bestimmen Sie, nach welchem Erkennungsmerkmal die erzeugten Rechnungen sortiert werden sollen oder wählen Sie die Option **KEINE**, wenn Sie keine spezielle Sortierung wünschen. Optionen: **Auftrags-ID**, **Kunden-ID**, **Artikel-ID**.

|**Limit**
|Sie müssen die Rechnungserstellung auf einen der Werte im Drop-down-Menü begrenzen. Das Maximum sind **200** Rechnungen.

|**Rechnungsdatum**
|Wählen Sie ein **Datum aus dem Kalender** aus, das als Rechnungsdatum auf den Rechnungen erscheinen soll.

|**Ändere Auftragsstatus**
|Wählen Sie einen neuen **Auftragsstatus** aus, der den gewählten Aufträgen nach erfolgreichem Rechnungsabruf zugewiesen werden soll oder wählen Sie die Option **Keine Änderung**.
