== Artikeldaten darstellen

Im Menü **Pickliste** bestimmen Sie welche Artikeldaten Ihre Picklisten enthalten sollen, z.B. Artikel-ID, EAN, Lagerort, usw. und wie diese Informationen dargestellt werden sollen.

image::auftraege/einstellungen/dokumente/assets/DE-Einstellungen-Auftraege-Dokumente-Pickliste-01.png[]

__Bild 1: Einstellungen im Menü **Pickliste**__

[IMPORTANT]
.Felder anzeigen bzw. verbergen
====
Damit ein Feld angezeigt wird, muss für das jeweilige Feld eine **Positionsnummer** ausgewählt sein. Wenn Sie für ein Feld keine Anzeige auf der Pickliste wünschen, wählen Sie unter **Position** die Anzeige: **----**.
====

[.instruction]
Artikeldaten darstellen:

. Öffnen Sie das Menü **Einstellungen » Aufträge » Dokumente » Pickliste**.
. Entscheiden Sie, welche Felder auf der Pickliste angezeigt werden sollen. Bestimmen Sie die Reihenfolge dieser Felder, indem Sie für jedes zu verwendende Feld eine **Positionsnummer** wählen.
. Tragen Sie unter **Name** ggf. eine eigene Bezeichnung für die verwendeten Felder ein.
. Bestimmen Sie die **Breite** für die Felder sowie die **Schriftgröße** und den **Schriftschnitt** für die jeweiligen Texte.
. Nehmen Sie ggf. weitere Einstellungen gemäß der nachfolgenden Tabelle vor.
. Klicken Sie auf **Speichern**, um die Einstellungen zu sichern.

[cols="a,a"]
|====
|Einstellung |Erläuterung

|**Seitenrand oben**
|Einen Wert (in mm) für den **oberen Seitenrand** eintragen. Den Wert ggf. anpassen, wenn eine PDF-Vorlage verwendet wird, die oben im "Briefpapier" z.B. die Firmenangaben und ein Logo enthält und dieser "Header" oberhalb der Pickliste positioniert werden soll (siehe dazu Kapitel 3 unten).

|**Lagerort**
|Bei Verwendung dieses Feldes bestimmen, wie der **Lagerortname** auf der Pickliste angezeigt werden soll. Es kann z.B. nur der Lagerortname angezeigt werden oder verschiede Kombinationen aus Lager-Informationen, wie Regal / Regalboden / Lagerortname. Standardmäßig werden auch Angaben zur Menge am Lagerort angezeigt. Wenn dies nicht gewünscht ist, ein Häkchen bei **Menge am Lagerort ausblenden** setzen.

|**Freitext**
|Name für das Freitextfeld wählen, d.h. die Bezeichnung des Feldes auf der Pickliste sowie das entsprechende **Freitextfeld 1-20** aus dem Dropdown-Menü auswählen. +
Freitextfelder zuerst im Menü **Einstellungen » Artikel » Freitextfelder** anlegen. Danach die angelegten Freitextfelder mit den entsprechenden Artikeln verknüpfen und die Werte für die Felder eintragen. Dies erfolgt im Menü **Artikel » Artikel bearbeiten » Tab: Felder**. +
Siehe dazu die Handbuchseite **<<artikel/artikel-verwalten#70, Artikel verwalten >>**.

|**Artikelbeschreibung**
|Hier entscheiden, welche Informationen die Artikelbeschreibung enthalten soll. Der **Artikelname** wird immer mit diesem Feld angezeigt. Zusätzlich können noch weitere Angaben ausgewählt werden, z.B. **Lagerort** (siehe Option oben), **Artikel-ID**, **Externe Artikel-ID**, **Artikelnummer**, **EAN** und **Bestellmerkmale**.
|====

== PDF-Vorlage hochladen

Mit **Dokumenten-Vorlage** bezeichnen wir an dieser Stelle das "Briefpapier" Ihres Shops, also eine Seite, die Ihre Firmendaten, Layoutelemente gemäß dem Corporate Design Ihres Unternehmens sowie Ihr Logo enthält. Diese Vorlage wird für das System im PDF-Format benötigt. Mit plentymarkets können Sie eine Dokumenten-Vorlage für die Pickliste verwenden. Gehen Sie wie nachfolgend beschrieben vor, um eine PDF-Vorlage hochzuladen.

image::auftragsabwicklung/auftraege/einstellungen/dokumente/assets/DE-Einstellungen-Auftraege-Dokumente-Pickliste-03.png[]

__Bild 3: PDF-Vorlage hochladen__

[.instruction]
PDF-Vorlage hochladen:

. Öffnen Sie das Menü **Einstellungen » Aufträge » Dokumente » Pickliste**.
. Klicken Sie auf das Tab **PDF-Vorlage**.
. Klicken Sie auf das Tab **Neue Vorlage**.
. Klicken Sie auf **Durchsuchen** und wählen Sie die PDF-Dokumenten-Vorlage aus.
. Klicken Sie auf **Speichern**, um die Einstellungen zu sichern. +
→ Die PDF-Vorlage erscheint nun im Tab **Vorlagen**. Sie können dort eine Vorschau sehen oder die PDF-Vorlage wieder löschen.

[TIP]
.Tipp für Header
====
Wenn Sie z.B. eine PDF-Vorlage mit "Header" verwenden und Sie diesen Header oberhalb der Pickliste positionieren möchten, müssen Sie ggf. den hinterlegten Wert im Feld **Seitenrand oben** noch anpassen (siehe dazu die Erläuterung in Tabelle 1).
====

Weitere Details zu Dokumenten-Vorlagen finden Sie im Kapitel **Dokumenten-Vorlage erstellen** auf der Handbuchseite **<<auftragsabwicklung/auftraege/einstellungen/dokumente#, Dokumente  >>**.
