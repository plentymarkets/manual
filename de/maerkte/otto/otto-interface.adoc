== OTTO

link:https://www.otto.de/[OTTO^]{nbsp}icon:external-link[] ist Teil der Otto Group und eines der erfolgreichsten Online-Versandhandelsunternehmen Deutschlands. +

Bei OTTO handelt es sich um einen Online-Shop, das heißt, du verkaufst im Namen von OTTO. Auch die Kommunikation mit Kund:innen wird von OTTO übernommen. +

Aus Datenschutzgründen ist der Inhalt der Handbuchseite OTTO nur für Kund:innen zugänglich, die einen OTTO-Vertrag haben. Die OTTO-Handbuchseite steht dir als OTTO-Partner im plentymarkets link:https://forum.plentymarkets.com/[Forum^]{nbsp}icon:external-link[] innerhalb der OTTO-Kategorie zur Verfügung.

[IMPORTANT]
.OTTO verfügt über drei Schnittstellen
====
OTTO verfügt über drei Schnittstellen: *Direktversand*, *Cooperation* und *Integration*. Alle Schnittstellen werden von plentymarkets unterstützt. +
Der Unterschied zwischen den Schnittstellen besteht im Auftragsimport. Bei den Schnittstellen *Direktversand* und *Cooperation* übermittelt OTTO einzelne Aufträge in dein plentymarkets System, die du dann direkt an die Endkund:innen schickst. +
Wenn du die Schnittstelle *Integration* nutzt, übermittelt dein plentymarkets System einmal pro Tag einen Sammelauftrag an OTTO. In diesem Fall schickt OTTO dann die bestellten Artikel an die Endkund:innen.
====
