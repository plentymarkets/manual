.{marketplace}-Zuordnungen
[cols="1,2"]
|====
|Marktplatz-Datenfeld |plentymarkets Datenfeld

| Produktbezeichnung
a| Ordne für dieses Marktplatz-Datenfeld eines oder mehrere der folgenden plentymarkets Datenfelder zu:

* Name 1

_oder_

* Name 2

_oder_

* Name 3

_oder_

* Eigenschaft

| Hersteller-Teilenummer
a| Ordne für dieses Marktplatz-Datenfeld das folgende plentymarkets Datenfeld zu:

* Modell (Manufacturer's Part Number, MPN)

| Inhalt Einheit
a| Erstelle für dieses Marktplatz-Datenfeld eine Eigenschaft. Für alle Einheiten, die du für deine Produkte verwendest, erstelle einen Eigenschaftswert:

* Blatt = unece.unit.ST
* Centimeter = unece.unit.CMT
* Gramm = unece.unit.GRM
* Kilogramm = unece.unit.KGM
* Liter = unece.unit.LTR
* Meter = unece.unit.MTR
* Milliliter = unece.unit.MLT
* Quadratmeter = unece.unit.MTK
* Paar = unece.unit.PR
* Teile = unece.unit.NPT
* Stück = unece.unit.C62
* Bogen = unece.unit.QR
* Set = unece.unit.SET

*_Tipp:_* Wenn alle deine Varianten für diesen Katalog dieselbe Einheit für den Inhalt haben, kannst du auch einen eigenen Wert übergeben. Wenn du zum Beispiel nur die Einheit "Stück" angeben möchtest, gibst du `unece.unit.C62` in das Feld *Eigener Wert* ein.

| Verkaufsmengeneinheit
a| Erstelle für dieses Marktplatz-Datenfeld eine Eigenschaft. Für alle Einheiten, die du für deine Produkte verwendest, erstelle einen Eigenschaftswert:

* Stück = ST
* Rolle = ROL
* Palette = PAL
* Pack = PAK
* Paar = PAA
* Meter = M
* Karton = KAR
* LE = LE
* Kubikmeter = M3
* Quadratmeter = M2
* Kilogramm = KG
* Gramm = G

*_Tipp:_* Wenn alle deine Varianten für diesen Katalog dieselbe Verkaufsmengeneinheit haben, kannst du auch einen eigenen Wert übergeben. Wenn du zum Beispiel nur die Einheit "Stück" angeben möchtest, gibst du `ST` in das Feld *Eigener Wert* ein.

| Gewicht Einheit
a| Erstelle für dieses Marktplatz-Datenfeld eine Eigenschaft. Für alle Einheiten für das Gewicht, die du für deine Produkte verwendest, erstelle einen Eigenschaftswert:

* Gramm = unece.unit.GRM
* Kg/m = unece.unit.KL
* g/m = unece.unit.GM1
* g/m² = unece.unit.gqm
* KG = unece.unit.KGM

*_Tipp:_* Wenn alle deine Varianten für diesen Katalog dieselbe Einheit für das Gewicht haben, kannst du auch einen eigenen Wert übergeben. Wenn du zum Beispiel nur die Einheit "Gramm" angeben möchtest, gibst du `unece.unit.GRM` in das Feld *Eigener Wert* ein.


| Variantengruppe
| Das Datenfeld *Variantengruppe* dient als Gruppierungsmerkmal, also dazu, dem Marktplatz zu signalisieren, welche Gemeinsamkeiten Produkte haben. Deshalb solltest du dem Feld das plentymarkets Datenfeld *Artikel-ID* zuordnen.

| Variante
| Das Datenfeld *Variante* dient als Differenzierungsmerkmal, also um dem Marktplatz mitzuteilen, wie sich die Varianten eines Artikels voneinander unterscheiden. Deshalb ordnest du diesem Feld plentymarkets *Attributwertnamen* zu.
|====
