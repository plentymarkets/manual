[#plugins-herunterladen]
== Plugins Mirakl Connector und {market} herunterladen

Die Schnittstelle zu {market} wird über ein Plugin integriert. Um das {market}-Plugin nutzen zu können, lade das Plugin im link:{marketplace-url}[plentyMarketplace^]{nbsp}icon:external-link[] herunter.

Um das {market}-Plugin nutzen zu können, benötigst du außerdem das Plugin *Mirakl Connector*. Du findest Mirakl ebenfalls auf link:https://marketplace.plentymarkets.com/mirakl_6917[plentyMarketplace^]{nbsp}icon:external-link[].

:plugin: Mirakl Connector
