[#ereignisaktionen]
== Ereignisaktionen einrichten

Richte eine <<automatisierung/ereignisaktionen#, Ereignisaktion>> ein, um den Auftragsstatus bei {marketplace} automatisch zu aktualisieren, wenn bestimmte Ereignisse eintreten. Folgende Ereignisaktionen können eingerichtet werden:

// Welche Ereignisaktionen es gibt hängt vom Marktplatz ab. Hier die gängigsten. Bei Bedarf anpassen!

ifeval::["{marketplace}" == "Conrad"]
include::./conrad-spezial.adoc[tag=automatisierung-zusammenfassung]
endif::[]

ifeval::["{marketplace}" == "Mediamarkt"]
include::./mediamarkt-spezial.adoc[tag=automatisierung-zusammenfassung]
endif::[]

ifeval::["{marketplace}" == "Voelkner"]
include::./voelkner-spezial.adoc[tag=automatisierung-zusammenfassung]
endif::[]

In den folgenden Kapiteln wird beschrieben, wie du die Ereignisaktionen in plentymarkets einrichtest.

[IMPORTANT]
.Weitere Aktionen vorhanden
====
Zusätzlich zu den oben genannten Ereignisaktionen werden durch das {marketplace}-Plugin noch weitere Ereignisaktionen zu Verfügung gestellt. Es gibt diese Aktionen, da sich die Auftragsverarbeitung in plentymarkets und Mirakl voneinander unterscheiden.

Wir empfehlen allerdings, nur die Aktionen zu verwenden, die hier im Detail beschrieben sind.
====

ifeval::["{marketplace}" == "Conrad"]
include::./conrad-spezial.adoc[tag=automatisierung-details]
endif::[]

ifeval::["{marketplace}" == "Mediamarkt"]
include::./mediamarkt-spezial.adoc[tag=automatisierung-details]
endif::[]

ifeval::["{marketplace}" == "Voelkner"]
include::./voelkner-spezial.adoc[tag=automatisierung-details]
endif::[]
