== Lagerauswahl

Nachdem Sie sich in das plentymarkets Mobile Warehouse eingeloggt haben, erscheint zunächst ein Menü zur **Lagerauswahl**.

image::tools/plentymarkets-mobile-warehouse/assets/DE-warehouse-01-lagerauswahl-SI.png[]

__Bild 1: Lagerauswahl__

[WARNING]
.**Benutzereinstellungen prüfen**
====
Für **variable Benutzer** müssen entsprechende Einstellungen im jeweiligen plentymarkets-Benutzerkonto im Tab **Berechtigungen** bezüglich dem Zugriff auf die Läger vorgenommen werden.
====

[NOTE]
.Lager auswählen:
====
Markieren Sie das gewünschte **Lager**.

Klicken Sie dann auf den Button **Weiter**.
====

[IMPORTANT]
.**Variable Benutzer**
====
Diese Option steht für variable Benutzer nicht zur Verfügung, da diese nur Zugriff auf ein bestimmtes Lager haben, das dann fest eingestellt ist.
====

== Ein Menü auswählen

Hier wählen Sie das gewünschte **Menü**, um dort Artikel zu suchen und **einzubuchen**, **umzubuchen** oder eine **Inventur** durchzuführen. Die Funktion der Menüs ist im jeweiligen Abschnitt erläutert.

image::tools/plentymarkets-mobile-warehouse/assets/DE-warehouse-02-menueauswahl-SI.png[]

__Bild 2: Menüauswahl__

Der Button unten in der Mitte zeigt das gewählte **Lager** an. Per Klick auf diesen Button wechseln Sie wieder zur **Lagerauswahl** und können das gewählte Lager ändern. Diese Option steht auch in den einzelnen Menüs und Untermenüs zur Verfügung.

== Artikel suchen

Mit der Artikelsuche suchen Sie den Artikel, dessen Warenbestand Sie bearbeiten möchten. Je nach Konfiguration werden mehrere Artikel angezeigt.

Die verschiedenen Möglichkeiten, **Artikel in Ihrem Warenbestand zu suchen**, zeigt die nachfolgende Tabelle.

image::tools/plentymarkets-mobile-warehouse/assets/DE-warehouse-03-artikelsuche-SI.png[]

__Bild 3: Artikelsuche__

== Einbuchen

In diesem Menü buchen Sie Artikel zu Ihrem Warenbestand hinzu. Die Buchung erfolgt durch Zuordnung eines aktuellen oder eines neuen **Lagerortes** oder durch eintragen oder einscannen des **Barcodes** des gewünschten Lagerortes.

[IMPORTANT]
.
====
Sie können den Barcode im **Menü** eingeben oder mit einem **Barcode-Scanner** einlesen, wenn Sie entsprechende **Barcode-Labels** bei Ihren Lagerorten angebracht haben.
====

[TIP]
.Tipp: **Barcodes generieren**
====
Die Barcodes generieren Sie im Menü **Warenbestand &gt; Lager &gt; Lager-Verwaltung &gt; Jeweiliges Lager &gt; Tab: Label**. Tragen Sie dort die **Größe** des Barcodes ein. Die Barcodes werden dann für die Lagerorte automatisch erzeugt. Bei Klick auf das **Drucker-Symbol** wird eine **PDF-Datei** mit den Barcodes generiert und Sie können diese **ausdrucken** und im Lager bei den Lagerorten anbringen.
====

image::tools/plentymarkets-mobile-warehouse/assets/DE-warehouse-04-einbuchung-SI.png[]

[cols="a,a"]

|**Offene Bestellmenge**
|Die **offene Bestellmenge** gibt an, welche Menge des Artikels im Warenbestand fehlt. Mit dieser Information können Sie die endgültige Bestellmenge dann genau kalkulieren und unter **Menge** eintragen.

|Führt zum Menü **Lager auswählen**. Nach Wahl eines anderen Lagers wählen Sie erneut das gewünschte Menü.

|**Logout**
|Wenn Sie das MobileWarehouse verlassen oder den Benutzer wechseln möchten, klicken Sie auf **Logout**.

Die Funktion **Umbuchen** ermöglicht Ihnen, eine bestimmte Artikelmenge von einem Lagerort auf einen anderen Lagerort umzubuchen. Sie können den Lagerort, wie auch den Barcode im **Menü** eingeben oder den Barcode mit einem **Barcode-Scanner** einlesen, wenn Sie entsprechende **Barcode-Labels** bei Ihren Lagerorten angebracht haben.

== Inventur

Die Funktion **Inventur** ermöglicht Ihnen, **Bestandskorrekturen** Ihres Warenbestands vorzunehmen.

[TIP]
.Tipp: **Standard-Lagerorte**
====
Vor der Inventur sollten Sie alle Bestände, die sich noch auf **Standard-Lagerorten** befinden, auf bestehende Lagerorte umbuchen.
====

[WARNING]
.**Keine Bestandsbuchungen im plentymarkets-System während der Inventur**
====
Um Fehlkonfigurationen zu vermeiden, sollten Sie in Ihrem plentymarkets-System für die Dauer der Inventur keine Buchungen vornehmen, die den Warenbestand beeinflussen.
====

[NOTE]
.
====

====

image::tools/plentymarkets-mobile-warehouse/assets/DE-warehouse-10-inventur-4-SI.png[]

__Bild 10: Inventur__;__Artikelmenge eingeben__

Sie können jetzt für den gefundenen Artikel für diesen Lagerort eine bestimmte **Menge** hinterlegen oder keine Menge eingeben. Wenn Sie keine Menge eingeben, wird der Artikel mit der Menge **0** dort hinterlegt. Die Menge können Sie dann später in der Übersicht (siehe Bild 11) noch anpassen.

[NOTE]
.Inventur durchführen
====
Tragen Sie unter **Menge** die tatsächliche Lagermenge des Artikels ein (siehe Bild 10, Pos. 1). +
⇒ Diese Menge wird dann für den betreffenden Lagerort hinterlegt.

Klicken Sie auf den Button **Nächster Lagerort**, um den nächsten Lagerort zu bearbeiten (siehe Bild 10, Pos. 2).

Wiederholen Sie diese Schritte gemäß Bild 9 und 10 für jeden weiteren Lagerort.
====

Wenn Sie alle Lagerorte durchlaufen haben, erscheint eine **Übersicht** der Lagerorte des betreffenden Regals.

image::tools/plentymarkets-mobile-warehouse/assets/DE-warehouse-11-inventur-5-SI.png[]

__Bild 11: Inventur__;__Übersicht__

Nun erhalten Sie in der **Übersicht** die Details zum Bestand dieses Regals. Sie können diese Werte noch ändern.

[NOTE]
.
====
Klicken Sie auf das **Stift-Symbol**, um die Werte der Lagerorte zu bearbeiten.
====

Wenn alle Werte korrekt sind und den Gegebenheiten Ihres Regals entsprechen, können Sie die Inventur buchen. Alternativ klicken Sie auf **Abbrechen**, in diesem Fall wird keine der bisherigen Eingaben gebucht.

[NOTE]
.
====
Um die Inventur zu buchen, klicken Sie auf den Button **Inventur buchen**.
====

[IMPORTANT]
.
====
Die Werte der Inventurbuchung werden in Ihr plentymarkets-System übernommen.
====
