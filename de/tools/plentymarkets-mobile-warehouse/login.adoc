== Zugriff auf das plentymarkets Mobile Warehouse

Ab der plentymarkets-Version 4.306 erfolgt der Zugriff über den Link: +
**http(s)://IhreDomain/plenty/w/**

== Systemvoraussetzungen

Anforderungen an das mobile Endgerät:

- Betriebssystem: Windows Mobile +
- Zugang zum Internet +
- Webbrowser mit Javascript Unterstützung

[NOTE]
.Login in das plentymarkets Mobile Warehouse:
====
Geben Sie den **Link** ein, wie in Abschnitt 1 erläutert.

Geben Sie Ihren **Benutzernamen** und Ihr **Passwort** ein.

Wähle Sie die **Sprache**.

Klicken Sie auf den Button **Login**.
====

[WARNING]
.**Wichtig bei parallelem Login**
====
Beachten Sie, dass bei gleichzeitigem Zugriff mit einem mobilen Endgerät und per Rechner auf Ihr plentymarkets-System Buchungsvorgänge nur für das jeweils andere Gerät aktualisiert werden, wenn bei Änderungen ein erneutes Login erfolgt. Wir empfehlen daher, die Buchungsvorgänge mit dem mobilen Endgerät komplett durchzuführen, bevor Sie sich per Rechner einloggen.
====
