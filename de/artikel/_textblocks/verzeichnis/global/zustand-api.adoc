|Zulässige Importwerte in CSV-Datei |Optionen in der Dropdown-Liste im Backend

|[0] Neu

|`1`
|[1] Gebraucht wie neu

|`2`
|[2] Gebraucht sehr gut

|`3`
|[3] Gebraucht gut

|`4`
|[4] Gebraucht annehmbar

|`5`
|[5] B-Ware
|====
