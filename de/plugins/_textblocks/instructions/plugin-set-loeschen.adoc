[.instruction]
Plugin-Sets löschen:

. Öffne das Menü *Plugins » Plugin-Übersicht*.
. Klicke bei einem Plugin-Set auf *Löschen* (icon:minus-square[role=red]). +
→ Das Fenster *Plugin-Set löschen* wird geöffnet.
. Bestätige das Löschen. +
→ Das Plugin-Set wird gelöscht.
